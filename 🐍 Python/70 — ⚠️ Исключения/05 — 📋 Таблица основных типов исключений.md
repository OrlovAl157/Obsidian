# 📋 Таблица основных типов исключений Python

## 📌 Коротко

> Справочник наиболее часто используемых встроенных исключений в Python с примерами возникновения и описанием каждого.

---

## 💡 Полезные советы

- **Запомните частые исключения** — это поможет в отладке
- **Проверьте иерархию** — родительское исключение поймает все дочерние
- **Ищите конкретные исключения** — не используйте `except:` для всех
- **Комбинируйте исключения** — `except (TypeError, ValueError):`

---

## 🚀 Быстрая справка - Все исключения одним взглядом

| Исключение | Группа | Когда возникает | Пример |
|---|---|---|---|
| **ValueError** | Значения | Неправильное значение, но правильный тип | `int('abc')` |
| **TypeError** | Типы | Неправильный тип данных | `'2' + 2` |
| **IndexError** | LookupError | Индекс вне диапазона | `[1,2][10]` |
| **KeyError** | LookupError | Ключа нет в словаре | `{}['key']` |
| **ZeroDivisionError** | ArithmeticError | Деление на ноль | `10 / 0` |
| **NameError** | Имена | Переменная не определена | `print(x)` |
| **AttributeError** | Атрибуты | Атрибут объекта не существует | `''.bad_method()` |
| **FileNotFoundError** | OSError | Файл не найден | `open('xxx.txt')` |
| **PermissionError** | OSError | Нет прав доступа | `open('/root/file.txt')` |
| **ImportError** | Импорт | Ошибка при импорте модуля | `import bad_module` |
| **AssertionError** | Проверки | Оператор assert вернул False | `assert False` |
| **StopIteration** | Итерация | Конец итератора | `next(empty_iter)` |
| **RecursionError** | RuntimeError | Слишком глубокая рекурсия | Бесконечная рекурсия |
| **NotImplementedError** | RuntimeError | Метод не реализован | Абстрактный метод |
