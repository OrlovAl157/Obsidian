# 📋 Таблица основных типов исключений Python

## 📌 Коротко

> Справочник наиболее часто используемых встроенных исключений в Python с описанием каждого. Эта таблица поможет быстро найти нужное исключение и понять, когда оно возникает.

---

## 💡 Полезные советы

- **Запомните частые исключения** — это поможет в отладке
- **Проверьте иерархию** — родительское исключение поймает все дочерние
- **Ищите конкретные исключения** — не используйте `except:` для всех
- **Комбинируйте исключения** — `except (TypeError, ValueError):`
- **Exception — базовый класс** — используйте его для своих исключений, а не BaseException

---

## 🚀 Быстрая справка — Все исключения одним взглядом

| Тип исключения | Описание | Когда возникает |
|---|---|---|
| **BaseException** | Класс самого верхнего уровня и базовый для всех прочих классов исключений | Корневой класс всех исключений |
| **Exception** | Базовый класс для большинства встроенных в Python исключений. Именно его, а не BaseException, необходимо наследовать при создании пользовательского класса исключения | Для создания собственных исключений |
| **AssertionError** | Возбуждается инструкцией assert | `assert False` |
| **AttributeError** | Возбуждается при попытке обращения к несуществующему атрибуту объекта | `'text'.bad_method()` |
| **EOFError** | Возбуждается функцией input() при достижении конца файла с вводными данными | `input()` при EOF |
| **ImportError** | Возбуждается, если невозможно импортировать модуль или пакет | `import bad_module` |
| **IndexError** | Возбуждается при попытке получить элемент коллекции по несуществующему индексу | `[1, 2, 3][10]` |
| **KeyError** | Возбуждается при попытке получить элемент коллекции по несуществующему ключу | `{}['missing_key']` |
| **MemoryError** | Возбуждается, если интерпретатору существенно не хватает оперативной памяти | Недостаточно памяти |
| **NameError** | Возбуждается при попытке обратиться к необъявленной переменной | `print(undefined_var)` |
| **TypeError** | Возбуждается при несоответствии типов | `'2' + 2` |
| **ValueError** | Возбуждается при передаче в функцию аргумента с неподдерживаемым значением | `int('abc')` |
| **ZeroDivisionError** | Возбуждается при попытке выполнить деление на ноль | `10 / 0` |
| **NotImplementedError** | Возбуждается, если наследник класса не переопределил метод, который должен был переопределить | Абстрактный метод |

## 🎯 Примеры использования

### AssertionError

```python
age = -5
assert age > 0, "Возраст не может быть отрицательным"
# AssertionError: Возраст не может быть отрицательным
```

### AttributeError

```python
text = "hello"
text.non_existent_method()
# AttributeError: 'str' object has no attribute 'non_existent_method'
```

### ImportError

```python
import non_existent_module
# ImportError: No module named 'non_existent_module'
```

### IndexError

```python
numbers = [1, 2, 3]
print(numbers[10])
# IndexError: list index out of range
```

### KeyError

```python
data = {'name': 'John'}
print(data['age'])
# KeyError: 'age'
```

### NameError

```python
print(undefined_variable)
# NameError: name 'undefined_variable' is not defined
```

### TypeError

```python
result = '5' + 5
# TypeError: can only concatenate str (not "int") to str
```

### ValueError

```python
number = int('not a number')
# ValueError: invalid literal for int() with base 10: 'not a number'
```

### ZeroDivisionError

```python
result = 10 / 0
# ZeroDivisionError: division by zero
```
