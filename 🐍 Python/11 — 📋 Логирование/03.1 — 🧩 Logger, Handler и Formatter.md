# 📘 Logger, Handler, Formatter — памятка

## Справка

| Элемент | Назначение |
|---|---|
| Logger | Принимает сообщения |
| LogRecord | Хранит данные о событии |
| Handler | Определяет, куда отправить сообщение |
| Formatter | Форматирует сообщение |
| extra | Добавляет пользовательские поля |
| propagate | Передаёт запись родительскому логгеру |

---

## Общая схема

```text
logger.info(...)
      │
      ▼
Создаётся LogRecord
      │
      ▼
    Logger
      │
      ├────────► Handler 1 ─────► Formatter ─────► Файл
      │
      └────────► Handler 2 ─────► Formatter ─────► Консоль
```

## Что происходит

1. Вызывается `logger.info()`.
2. Создаётся `LogRecord`.
3. Logger проверяет свой уровень.
4. Logger передаёт запись всем своим Handler.
5. Каждый Handler проверяет свой уровень и фильтры.
6. Formatter превращает `LogRecord` в строку.
7. Handler записывает строку в файл, консоль и т.д.

---

## Если у Logger нет Handler

```text
logger.info(...)
      │
      ▼
Logger
      │
      ▼
Есть Handler?
      │
     Нет
      │
      ▼
propagate=True ?
      │
     Да
      ▼
Родительский Logger
      │
      ▼
...
      │
      ▼
Root Logger
      │
      ├── Есть Handler → сообщение обработано
      └── Нет → сообщение потеряно
```

> По умолчанию `logger.propagate == True`, поэтому именованный логгер пытается передать запись родителю.

## LogRecord

Создаётся автоматически при вызове:

```python
logger.info("Успешно", extra={"date_str": "21.07.2026"})
```

Содержит:

- levelname
- message
- filename
- lineno
- funcName
- created
- поля из `extra`

---

## extra

```python
logger.info(
    "OK",
    extra={"date_str": "21.07.2026"}
)
```

Теперь можно использовать:

```python
"%(date_str)s"
```

Если поля нет, Formatter выдаст ошибку.

---

## Кто за что отвечает

- **Logger** — принимает сообщения.
- **LogRecord** — хранит информацию.
- **Handler** — отправляет сообщение.
- **Formatter** — оформляет сообщение.
- **Filter** — решает, пропускать запись или нет.

---

## Запомнить

> **Logger принимает → LogRecord хранит → Handler отправляет → Formatter оформляет.**

> Если у Logger нет Handler, запись поднимается к родителю (`propagate=True`) и в конечном итоге попадает к Root Logger.
