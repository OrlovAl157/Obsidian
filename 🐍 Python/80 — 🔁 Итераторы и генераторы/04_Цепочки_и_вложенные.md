# 🔀 Цепочки и вложенные выражения

## 📌 Коротко

> Вложенные генераторные выражения и функции-генераторы с несколькими циклами. Порядок циклов в выражениях совпадает с порядком в обычных циклах. Практические примеры: декартово произведение, флаттинг списков, обработка матриц.

---

## 💡 Полезные советы

- **Порядок циклов совпадает** — в выражении и в функции результат одинаковый
- **Внешний цикл в начале** — первый `for` — это внешний, последний `for` — это внутренний
- **`yield from` для делегирования** — прямая передача в вложенный генератор
- **Внешний цикл медленнее** — работает реже, но задаёт общую структуру
- **Внутренний цикл быстрее** — работает чаще, но внутри каждой итерации внешнего
- **Плоские структуры** — `flatten()` для вложенных списков
- **Рекурсия в генераторах** — используйте `yield from` для вложенных структур

---

## 🚀 Быстрая справка

| Задача | Синтаксис | Пример |
|---|---|---|
| Декартово произведение | `((x, y) for x in ... for y in ...)` | `((x, y) for x in [1,2] for y in ['a','b'])` |
| Плоский список | `(item for sublist in ... for item in sublist)` | `(x for lst in [[1,2], [3,4]] for x in lst)` |
| С условием | `(...for x in ... for y in ... if condition)` | `(x*y for x in range(3) for y in range(3) if x!=y)` |
| С функцией | `(f(x, y) for x in ... for y in ...)` | `(x+y for x in [1,2] for y in [10,20])` |
| Вложенные функции | `(yield from sub_gen() for ...)` | `def gen(): yield from (x for x in ...)` |

## 📖 Теория: Порядок циклов

### Правило: Порядок совпадает с функцией

**Генераторное выражение:**
```python
result = ((x, y) for x in range(3) for y in range(3))
```

**Функция-генератор (эквивалентно):**
```python
def gen():
    for x in range(3):
        for y in range(3):
            yield (x, y)

result = gen()
```

**Результат одинаковый!**

```python
# Оба дают:
# [(0, 0), (0, 1), (0, 2), (1, 0), (1, 1), (1, 2), (2, 0), (2, 1), (2, 2)]
```

### Визуализация вложения

```
for x in range(3):        # Внешний цикл (медленный)
    for y in range(3):    # Внутренний цикл (быстрый)
        yield (x, y)

Итерация 1: x=0
    Итерация 1.1: y=0 → (0, 0)
    Итерация 1.2: y=1 → (0, 1)
    Итерация 1.3: y=2 → (0, 2)
Итерация 2: x=1
    Итерация 2.1: y=0 → (1, 0)
    Итерация 2.2: y=1 → (1, 1)
    Итерация 2.3: y=2 → (1, 2)
Итерация 3: x=2
    Итерация 3.1: y=0 → (2, 0)
    Итерация 3.2: y=1 → (2, 1)
    Итерация 3.3: y=2 → (2, 2)
```

### Множественные условия

```python
# Условие для всех циклов
result = (
    (x, y) 
    for x in range(5) 
    for y in range(5) 
    if x != y and (x + y) % 2 == 0
)

list(result)
# [(0, 2), (0, 4), (1, 1), (1, 3), (2, 0), (2, 4), (3, 1), (3, 3), (4, 0), (4, 2), (4, 4)]
```

## 💻 Примеры: Декартово произведение

### Генераторное выражение

```python
# Декартово произведение двух списков
colors = ['red', 'green', 'blue']
sizes = ['S', 'M', 'L']

products = ((color, size) for color in colors for size in sizes)

print(list(products))
# [('red', 'S'), ('red', 'M'), ('red', 'L'),
#  ('green', 'S'), ('green', 'M'), ('green', 'L'),
#  ('blue', 'S'), ('blue', 'M'), ('blue', 'L')]
```

### Функция-генератор

```python
def product(colors, sizes):
    for color in colors:
        for size in sizes:
            yield (color, size)

result = product(['red', 'green'], ['S', 'M', 'L'])
print(list(result))
# [('red', 'S'), ('red', 'M'), ('red', 'L'),
#  ('green', 'S'), ('green', 'M'), ('green', 'L')]
```

### Встроенный itertools

```python
from itertools import product

result = product(['red', 'green'], ['S', 'M', 'L'])
print(list(result))
# [('red', 'S'), ('red', 'M'), ('red', 'L'),
#  ('green', 'S'), ('green', 'M'), ('green', 'L')]
```

## 💻 Примеры: Флаттинг (плоский список)

### Вложенные списки → плоский список

**Генераторное выражение:**
```python
nested = [[1, 2], [3, 4], [5, 6]]

# Порядок циклов: внешний для списков, внутренний для элементов
flat = (item for sublist in nested for item in sublist)

print(list(flat))  # [1, 2, 3, 4, 5, 6]
```

**Функция-генератор:**
```python
def flatten(nested):
    for sublist in nested:
        for item in sublist:
            yield item

result = flatten([[1, 2], [3, 4], [5, 6]])
print(list(result))  # [1, 2, 3, 4, 5, 6]
```

### Рекурсивный флаттинг (для произвольной глубины)

```python
def flatten_recursive(nested):
    for item in nested:
        if isinstance(item, (list, tuple)):
            yield from flatten_recursive(item)  # Рекурсия!
        else:
            yield item

data = [1, [2, 3, [4, 5, [6, 7]]], 8]
result = list(flatten_recursive(data))
print(result)  # [1, 2, 3, 4, 5, 6, 7, 8]
```

## 💻 Примеры: Матрицы

### Транспонирование матрицы

```python
# Оригинальная матрица (3x3)
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

# Транспонирование (с zip)
transposed = list(zip(*matrix))
print(transposed)
# [(1, 4, 7), (2, 5, 8), (3, 6, 9)]

# Или генератором
transposed = ((row[i] for row in matrix) for i in range(len(matrix[0])))
```

### Операции над элементами матрицы

```python
matrix = [[1, 2], [3, 4]]

# Удвоить все элементы
doubled = [[x*2 for x in row] for row in matrix]
print(doubled)  # [[2, 4], [6, 8]]

# Генератор вместо list comprehension
doubled_gen = ((x*2 for x in row) for row in matrix)
# Осторожно! Это вложенные генераторы!

# Лучше:
doubled_gen = (x*2 for row in matrix for x in row)
print(list(doubled_gen))  # [2, 4, 6, 8]
```

### Диагональ матрицы

```python
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

# Главная диагональ
diagonal = [matrix[i][i] for i in range(len(matrix))]
print(diagonal)  # [1, 5, 9]

# Или генератором
diagonal = (matrix[i][i] for i in range(len(matrix)))
print(list(diagonal))  # [1, 5, 9]
```

## 💻 Примеры: Сложные структуры

### Словари из вложенных структур

```python
# Исходные данные
people = [
    {'name': 'Alice', 'age': 30, 'skills': ['Python', 'SQL']},
    {'name': 'Bob', 'age': 25, 'skills': ['Java', 'C++']},
]

# Все скиллы одного человека
alice_skills = people[0]['skills']
print(alice_skills)  # ['Python', 'SQL']

# Все скиллы всех людей (плоский список)
all_skills = (skill for person in people for skill in person['skills'])
print(list(all_skills))  # ['Python', 'SQL', 'Java', 'C++']

# Пары (имя, скилл)
pairs = ((person['name'], skill) 
         for person in people 
         for skill in person['skills'])
print(list(pairs))
# [('Alice', 'Python'), ('Alice', 'SQL'), ('Bob', 'Java'), ('Bob', 'C++')]
```

### Обработка JSON-подобных структур

```python
data = {
    'users': [
        {'id': 1, 'name': 'Alice', 'posts': [
            {'id': 101, 'text': 'Hello'},
            {'id': 102, 'text': 'World'}
        ]},
        {'id': 2, 'name': 'Bob', 'posts': [
            {'id': 201, 'text': 'Hi'}
        ]},
    ]
}

# Все тексты всех постов всех пользователей
all_texts = (
    post['text'] 
    for user in data['users'] 
    for post in user['posts']
)
print(list(all_texts))  # ['Hello', 'World', 'Hi']

# С фильтром
texts_with_h = (
    post['text']
    for user in data['users']
    for post in user['posts']
    if 'H' in post['text']
)
print(list(texts_with_h))  # ['Hello', 'Hi']
```

## 💡 Сравнение: Выражение vs Функция

### Простой случай

```python
# Выражение (лучше)
result = (x*2 for x in range(5))

# Функция (лишнее)
def gen():
    for x in range(5):
        yield x*2
result = gen()
```

### Средней сложности

```python
# Выражение (хорошо)
result = (x*2 for x in range(10) if x % 2 == 0)

# Функция (тоже хорошо)
def gen():
    for x in range(10):
        if x % 2 == 0:
            yield x*2
result = gen()
```

### Сложный случай

```python
# Выражение (сложно читать)
result = (
    x*y 
    for x in range(10) 
    for y in range(10) 
    if x != y and (x+y) % 2 == 0
)

# Функция (понятнее!)
def gen():
    for x in range(10):
        for y in range(10):
            if x != y and (x+y) % 2 == 0:
                yield x*y
result = gen()
```

## ⚠️ Частые ошибки

### 1. Неправильный порядок циклов

```python
# ❌ Неправильно (результат неожиданный)
result1 = ((x, y) for y in range(3) for x in range(3))
# [(0, 0), (1, 0), (2, 0), (0, 1), (1, 1), (2, 1), ...] ← y медленнее!

# ✅ Правильно (как обычно пишут)
result2 = ((x, y) for x in range(3) for y in range(3))
# [(0, 0), (0, 1), (0, 2), (1, 0), (1, 1), (1, 2), ...] ← x медленнее!
```

### 2. Условие в неправильном месте

```python
# ❌ Неправильно (условие только для внутреннего цикла)
result1 = ((x, y) for x in range(5) for y in range(5) if y > x)

# ✅ Если нужно условие для внешнего цикла
result2 = ((x, y) for x in range(5) if x > 1 for y in range(5))

# ✅ Условие на оба цикла
result3 = ((x, y) for x in range(5) for y in range(5) if x > 1 and y > 1)
```

### 3. Читаемость вложенных циклов

```python
# ❌ Плохо (одна строка, сложно читать)
result = (x*y for x in range(10) for y in range(10) if x > 2 and y < 7 and (x+y) % 2 == 0)

# ✅ Хорошо (с переносами строк)
result = (
    x * y
    for x in range(10)
    for y in range(10)
    if x > 2 and y < 7 and (x+y) % 2 == 0
)
```

## 🎯 Рекомендации

### Когда использовать выражение

✅ **Простая логика** (один-два цикла, 1-2 условия)
✅ **Одна строка** (или переносы для читаемости)
✅ **Нужна компактность**

### Когда использовать функцию

✅ **Сложная логика** (много циклов, условий)
✅ **Нужны комментарии** (функция лучше читается)
✅ **Нужны дополнительные операции** (print для отладки и т.д.)
✅ **Нужна переиспользуемость** (функция легче вызывать)

## 📝 Шпаргалка

```python
# ДЕКАРТОВО ПРОИЗВЕДЕНИЕ
pairs = ((x, y) for x in list1 for y in list2)

# ФЛАТТИНГ (плоский список)
flat = (item for sublist in nested for item in sublist)

# С УСЛОВИЯМИ
result = (x*y for x in range(5) for y in range(5) if x != y)

# ФУНКЦИЯ-ГЕНЕРАТОР
def gen():
    for x in list1:
        for y in list2:
            if condition:
                yield (x, y)

# YIELD FROM
def delegating():
    yield from range(5)
    yield from 'ABC'

# РЕКУРСИВНЫЙ ФЛАТТИНГ
def flatten(nested):
    for item in nested:
        if isinstance(item, (list, tuple)):
            yield from flatten(item)
        else:
            yield item
```
