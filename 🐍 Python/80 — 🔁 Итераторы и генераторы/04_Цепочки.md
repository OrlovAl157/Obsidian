# 🔀 Цепочки и вложенные выражения

## 📌 Коротко

> Вложенные генераторные выражения и функции-генераторы с несколькими циклами. Порядок циклов совпадает с обычными циклами.

---

## 📖 Теория

### Порядок циклов

**Генераторное выражение:**
```python
result = ((x, y) for x in range(3) for y in range(3))
```

**Функция-генератор (эквивалентно):**
```python
def gen():
    for x in range(3):
        for y in range(3):
            yield (x, y)

result = gen()
```

**Результат одинаковый!**

```python
# [(0, 0), (0, 1), (0, 2), (1, 0), (1, 1), (1, 2), (2, 0), (2, 1), (2, 2)]
```

### Визуализация вложения

```
for x in range(3):        # Внешний цикл (медленный)
    for y in range(3):    # Внутренний цикл (быстрый)
        yield (x, y)

Итерация 1: x=0
    1.1: y=0 → (0, 0)
    1.2: y=1 → (0, 1)
    1.3: y=2 → (0, 2)
Итерация 2: x=1
    2.1: y=0 → (1, 0)
    ...
```

## 💻 Примеры

### Декартово произведение

```python
# Генераторное выражение
colors = ['red', 'green', 'blue']
sizes = ['S', 'M', 'L']

products = ((color, size) for color in colors for size in sizes)
print(list(products))
# [('red', 'S'), ('red', 'M'), ('red', 'L'),
#  ('green', 'S'), ('green', 'M'), ('green', 'L'),
#  ('blue', 'S'), ('blue', 'M'), ('blue', 'L')]
```

### Флаттинг (плоский список)

```python
# Вложенные списки → плоский список
nested = [[1, 2], [3, 4], [5, 6]]

# Порядок: внешний цикл для списков, внутренний для элементов
flat = (item for sublist in nested for item in sublist)
print(list(flat))  # [1, 2, 3, 4, 5, 6]

# Функция-генератор
def flatten(nested):
    for sublist in nested:
        for item in sublist:
            yield item
```

### Рекурсивный флаттинг

```python
def flatten_recursive(nested):
    for item in nested:
        if isinstance(item, (list, tuple)):
            yield from flatten_recursive(item)  # Рекурсия!
        else:
            yield item

data = [1, [2, 3, [4, 5, [6, 7]]], 8]
result = list(flatten_recursive(data))
print(result)  # [1, 2, 3, 4, 5, 6, 7, 8]
```

### Матрицы

```python
# Декартово произведение координат
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

# Главная диагональ
diagonal = [matrix[i][i] for i in range(len(matrix))]
print(diagonal)  # [1, 5, 9]

# Генератор
diagonal_gen = (matrix[i][i] for i in range(len(matrix)))
print(list(diagonal_gen))  # [1, 5, 9]
```

### Сложные структуры

```python
# Словари со списками
people = [
    {'name': 'Alice', 'skills': ['Python', 'SQL']},
    {'name': 'Bob', 'skills': ['Java', 'C++']},
]

# Все скиллы всех людей
all_skills = (skill for person in people for skill in person['skills'])
print(list(all_skills))  # ['Python', 'SQL', 'Java', 'C++']

# Пары (имя, скилл)
pairs = ((person['name'], skill) 
         for person in people 
         for skill in person['skills'])
print(list(pairs))
# [('Alice', 'Python'), ('Alice', 'SQL'), ('Bob', 'Java'), ('Bob', 'C++')]
```

## ⚠️ Частые ошибки

### Неправильный порядок циклов

```python
# ❌ Неправильно (y медленнее!)
result1 = ((x, y) for y in range(3) for x in range(3))
# [(0, 0), (1, 0), (2, 0), (0, 1), (1, 1), (2, 1), ...]

# ✅ Правильно (x медленнее)
result2 = ((x, y) for x in range(3) for y in range(3))
# [(0, 0), (0, 1), (0, 2), (1, 0), (1, 1), (1, 2), ...]
```

### Читаемость

```python
# ❌ Плохо (одна строка, сложно читать)
result = (x*y for x in range(10) for y in range(10) if x > 2 and y < 7)

# ✅ Хорошо (с переносами)
result = (
    x * y
    for x in range(10)
    for y in range(10)
    if x > 2 and y < 7
)
```
