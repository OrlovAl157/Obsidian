# 📚 Справочник: Основные понятия

## 📌 Коротко

> Четкие определения: Collection, Sequence, Iterable, Iterator, Generator, Range. Схемы связей и иерархия типов.

---

## 📖 Теория

### Collection (Коллекция)

**Коллекция** — объект, который хранит в себе набор значений одного или нескольких типов и позволяет обращаться к своим значениям.

Примеры: `list`, `tuple`, `set`, `dict`, `str`

### Sequence (Последовательность)

**Sequence** — коллекция с пронумерованными элементами в строгом порядке.

Примеры: `list`, `tuple`, `str`  
НЕ Sequence: `set`, `dict` (нет индексов)

### Iterable (Итерируемый объект)

**Iterable** — объект, на основе которого можно создать iterator. То, по чему можно ходить циклом for.

**Правило:** Если есть метод `__iter__()`, то итерируемый.

**Примеры:** Collections, range, iterators, file objects, генераторы

### Iterator (Итератор)

**Iterator** — специальный объект, который выдает следующее значение через `next()` и возбуждает `StopIteration` когда значения закончились.

**Характеристики:**
- Создается из итерируемого через `iter()`
- Выдает элементы через `next()` по одному
- Одноразовый (истощается)
- Ленивый (вычисляет по требованию)

**Правило:** Если есть методы `__iter__()` И `__next__()`, то итератор.

### Generator (Генератор)

**Generator** — частный случай итератора, созданный функцией с `yield` или генераторным выражением.

**Два типа:**
1. Функция-генератор — `def` с `yield`
2. Генераторное выражение — `(x for x in ...)`

**Особенности:**
- Это ТАКЖЕ итератор
- Имеет дополнительные методы: `.send()`, `.throw()`, `.close()`

### Range object

**Range object** — объект между sequence и iterator.

**Как Sequence:** индексация, срезы, `len()`, многоразовый  
**Как Iterator:** ленивые вычисления, не хранит все в памяти

```python
r = range(10000000000000)  # Мгновенно!
print(len(r))              # 10000000000000
print(r[999])              # O(1) операция
```

## 🔗 Схема связей

```
Collection (list, dict, set, tuple, str)
    ↓
Iterable (имеет __iter__())
    ├── Collections
    ├── range object
    └── iterator
          │
          ├── встроенные (map, filter, zip, enumerate)
          ├── generator (функция с yield)
          └── custom class (с __iter__ и __next__)
```
