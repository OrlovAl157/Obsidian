# 🎁 Генератор - Полная памятка

## 📌 Коротко

> Генератор — удобный способ создания итераторов через функцию с `yield`. Подкласс Iterator с дополнительными методами `.send()`, `.throw()`, `.close()` для управления выполнением. Два типа: функции-генераторы и генераторные выражения.

---

## 💡 Полезные советы

- **`yield` — приостановка с сохранением состояния** — ключевое слово генератора
- **Генератор ЯВЛЯЕТСЯ итератором** — имеет `__iter__()` и `__next__()`
- **Одноразовый** — истощается после использования, как все итераторы
- **Ленивый** — вычисляет значения по требованию, экономит память
- **`.send()` — уникально для генератора** — отправка данных В генератор
- **`yield from` — делегирование** — прямая передача в вложенный генератор
- **Генераторное выражение — компактный синтаксис** — `(x for x in ...)`

---

## 🚀 Быстрая справка

| Концепция | Синтаксис | Пример |
|---|---|---|
| Функция-генератор | `def func(): yield value` | `def count(): yield 1; yield 2` |
| Генераторное выражение | `(expr for item in it)` | `(x**2 for x in range(5))` |
| Получить элемент | `next(gen)` | `next(gen)` → значение |
| Отправить данные | `gen.send(value)` | `gen.send(42)` |
| Отправить исключение | `gen.throw(exc)` | `gen.throw(ValueError)` |
| Завершить | `gen.close()` | `gen.close()` |
| Делегирование | `yield from gen` | `yield from other_gen()` |

## 📖 Теория

### Что такое yield?

**`yield`** — ключевое слово, которое:
1. Возвращает значение (как `return`)
2. Приостанавливает функцию
3. Сохраняет состояние (все локальные переменные)
4. При следующем `next()` продолжает с той же позиции

```python
def simple_gen():
    print('Перед первым yield')
    yield 1
    print('Между yield')
    yield 2
    print('После последнего yield')

gen = simple_gen()
print(next(gen))  
# Вывод: Перед первым yield
#        1

print(next(gen))  
# Вывод: Между yield
#        2

# print(next(gen))  # StopIteration
```

**Состояние сохраняется между вызовами!**

### yield vs return

| Характеристика | `return` | `yield` |
|---|---|---|
| Завершает функцию | ✅ Да | ❌ Приостанавливает |
| Сохраняет локальные переменные | ❌ Нет | ✅ Да |
| Количество значений | 1 | Много |
| Возвращаемый тип | Любой | Generator |
| Можно вызвать несколько раз | ❌ Нет | ✅ Да |

### return в генераторах

```python
def generate_ints():
    yield 1
    yield 2
    return 3  # Возбуждает StopIteration
    yield 4   # Никогда не выполнится

for num in generate_ints():
    print(num)
# 1
# 2

# Получить значение return в исключении
gen = generate_ints()
try:
    while True:
        print(next(gen))
except StopIteration as e:
    print(f"Возвращено: {e.value}")
# 1
# 2
# Возвращено: 3
```

## 🎯 Функции-генераторы

### Базовая структура

```python
def my_generator():
    yield value1
    yield value2
    yield value3

gen = my_generator()
print(next(gen))  # value1
print(next(gen))  # value2
print(next(gen))  # value3
```

### Примеры функций-генераторов

**Счётчик:**
```python
def counter(low, high):
    for num in range(low, high + 1):
        yield num

for i in counter(3, 5):
    print(i)  # 3, 4, 5
```

**Бесконечный генератор чётных чисел:**
```python
def even_numbers(begin):
    begin += begin % 2
    while True:
        yield begin
        begin += 2

evens = even_numbers(10)
for i, num in enumerate(evens):
    if i > 3:
        break
    print(num)  # 10, 12, 14, 16
```

**Факториалы:**
```python
def factorials():
    value = 1
    index = 1
    while True:
        yield value
        index += 1
        value *= index

facts = factorials()
for i, num in enumerate(facts, 1):
    if i > 5:
        break
    print(f'{i}! = {num}')
# 1! = 1
# 2! = 2
# 3! = 6
# 4! = 24
# 5! = 120
```

**Фильтрация и преобразование:**
```python
def process_data(data):
    for item in data:
        if item > 0:
            yield item * 2

data = [-1, 2, -3, 4, 5]
result = list(process_data(data))
print(result)  # [4, 8, 10]
```

## 📦 Генераторные выражения

### Синтаксис и сравнение

```python
# List comprehension (список - квадратные скобки)
list_comp = [x**2 for x in range(5)]
print(list_comp)   # [0, 1, 4, 9, 16]
print(type(list_comp))  # <class 'list'>

# Generator expression (генератор - круглые скобки)
gen_expr = (x**2 for x in range(5))
print(gen_expr)    # <generator object>
print(type(gen_expr))   # <class 'generator'>
print(list(gen_expr))   # [0, 1, 4, 9, 16]
```

### Примеры генераторных выражений

```python
# Простое преобразование
squares = (i**2 for i in range(1, 7))
print(list(squares))  # [1, 4, 9, 16, 25, 36]

# С условием
evens = (x for x in range(10) if x % 2 == 0)
print(list(evens))  # [0, 2, 4, 6, 8]

# Вложенные циклы
pairs = ((x, y) for x in range(3) for y in range(3))
print(list(pairs))
# [(0,0), (0,1), (0,2), (1,0), (1,1), (1,2), (2,0), (2,1), (2,2)]

# В функциях (скобки можно опустить)
print(sum(x**2 for x in range(10)))  # 285
print(max(x for x in [3, 1, 4, 1, 5]))  # 5
```

### Память: список vs генератор

```python
from sys import getsizeof

# Список хранит все значения
list_obj = list(range(1000000))
print(getsizeof(list_obj))     # ~8000000 байт

# Генератор не хранит значения
gen_obj = (x for x in range(1000000))
print(getsizeof(gen_obj))      # ~104 байта

# Разница в ~77000 раз!
```

## 🔗 yield from - Делегирование

### Упрощение кода

```python
# БЕЗ yield from
def get_data_old():
    for num in range(5):
        yield num
    for char in 'ABC':
        yield char

# С yield from (компактнее!)
def get_data():
    yield from range(5)
    yield from 'ABC'

print(list(get_data()))  # [0, 1, 2, 3, 4, 'A', 'B', 'C']
```

### Композиция генераторов

```python
def chain(*iterables):
    for it in iterables:
        yield from it

result = chain('AB', [1, 2], (4, 5))
print(list(result))  # ['A', 'B', 1, 2, 4, 5]
```

### Вложенные структуры

```python
def flatten(nested):
    for item in nested:
        if isinstance(item, (list, tuple)):
            yield from flatten(item)  # Рекурсия!
        else:
            yield item

data = [1, [2, 3, [4, 5]], 6, [7]]
print(list(flatten(data)))  # [1, 2, 3, 4, 5, 6, 7]
```

## 📤 Методы генератора: .send(), .throw(), .close()

### .send() — Отправка данных В генератор

**Уникальное свойство генератора!** Можно отправлять данные в работающий генератор.

```python
def echo():
    value = None
    while True:
        value = yield value  # yield ПОЛУЧАЕТ значение через .send()

gen = echo()
next(gen)  # Инициализация

print(gen.send("Hello"))   # Hello
print(gen.send("World"))   # World
print(gen.send("Python"))  # Python
```

**Как это работает:**
1. `next(gen)` или `gen.send(None)` — инициализирует генератор, доводит до первого `yield`
2. `gen.send(value)` — отправляет значение, генератор продолжается, получает значение в переменную
3. Выполняется код, доходит до следующего `yield`, возвращает значение

```python
def add_generator():
    total = 0
    while True:
        x = yield total  # Получить x через .send()
        if x:
            total += x

gen = add_generator()
next(gen)  # Инициализация

print(gen.send(5))    # 5
print(gen.send(3))    # 8
print(gen.send(2))    # 10
```

### .throw() — Отправка исключения

```python
def gen_with_error_handling():
    try:
        yield 1
        yield 2
        yield 3
    except ValueError as e:
        print(f"Поймано: {e}")
        yield "После ошибки"

gen = gen_with_error_handling()
print(next(gen))               # 1
print(next(gen))               # 2
print(gen.throw(ValueError, "Test error"))
# Вывод: Поймано: Test error
#        После ошибки
```

### .close() — Завершение генератора

```python
def cleanup_gen():
    try:
        yield 1
        yield 2
        yield 3
    finally:
        print("Очистка ресурсов")

gen = cleanup_gen()
print(next(gen))  # 1
print(next(gen))  # 2
gen.close()       # Завершить генератор
# Вывод: Очистка ресурсов

# После close() генератор мёртв
try:
    next(gen)  # StopIteration
except StopIteration:
    print("Генератор закрыт")
```

## 🔀 Конвейеры генераторов

### Композиция генераторов

```python
def integers(n):
    for i in range(1, n + 1):
        yield i

def evens(iterable):
    for i in iterable:
        if i % 2 == 0:
            yield i

def squared(iterable):
    for i in iterable:
        yield i * i

def negated(iterable):
    for i in iterable:
        yield -i

# Создаём конвейер
chain = negated(squared(evens(integers(10))))
print(list(chain))  # [-4, -16, -36, -64, -100]
```

### С генераторными выражениями

```python
n = 10

integers = (i for i in range(1, n + 1))
evens = (i for i in integers if i % 2 == 0)
squared = (i * i for i in evens)
negated = (-i for i in squared)

print(list(negated))  # [-4, -16, -36, -64, -100]
```

### Обработка больших файлов

```python
# Читаем CSV построчно без загрузки в память
with open('data.csv', 'r', encoding='utf-8') as file:
    # Шаг 1: Читаем строки
    file_lines = (line.strip() for line in file)
    
    # Шаг 2: Разбиваем на значения
    line_values = (line.split(',') for line in file_lines)
    
    # Шаг 3: Извлекаем заголовки
    headers = next(line_values)
    
    # Шаг 4: Создаём словари
    line_dicts = (dict(zip(headers, data)) for data in line_values)
    
    # Шаг 5: Фильтруем и обрабатываем
    result = (
        (line['user_name'], line['user_ip'])
        for line in line_dicts
        if 'stepik.org' in line['page_url']
    )
    
    # Шаг 6: Используем данные
    for name, ip in result:
        print(f'{name} --- {ip}')
```

**Каждая строка обрабатывается по мере чтения, память постоянна!**

## ⚠️ Особенности и ловушки

### 1. Генераторы одноразовые

```python
gen = (x**2 for x in range(5))

print(list(gen))  # [0, 1, 4, 9, 16]
print(list(gen))  # [] (пуст!)
```

### 2. Циклы в генераторных выражениях

```python
# Порядок циклов совпадает с вложенностью
# for x in range(3) → внешний цикл
# for y in range(3) → внутренний цикл

pairs = ((x, y) for x in range(3) for y in range(3))
print(list(pairs))
# [(0,0), (0,1), (0,2), (1,0), (1,1), (1,2), (2,0), (2,1), (2,2)]

# Сравните с функцией-генератором
def pairs_func():
    for x in range(3):
        for y in range(3):
            yield (x, y)

print(list(pairs_func()))  # Такой же результат
```

### 3. Ошибка: Цикл без скобок

```python
# ❌ Ошибка - нет скобок
# result = x**2 for x in range(5)

# ✅ Правильно
result = (x**2 for x in range(5))

# ✅ В функциях скобки опустить можно
print(sum(x**2 for x in range(5)))
```

### 4. Переменные в генераторных выражениях

```python
# ⚠️ Ловушка: переменная захватывается ПО ССЫЛКЕ
funcs = [lambda: i for i in range(5)]
print([f() for f in funcs])  # [4, 4, 4, 4, 4] (!!)

# Решение: использовать дефолтный аргумент
funcs = [lambda i=i: i for i in range(5)]
print([f() for f in funcs])  # [0, 1, 2, 3, 4]

# С генераторами это не проблема
gens = [i for i in (x for x in range(5))]
print(gens)  # [0, 1, 2, 3, 4]
```

## 📊 Сравнение: Генератор vs Итератор vs Range

| Характеристика | Генератор | Итератор (класс) | Range |
|---|---|---|---|
| **Создание** | `yield` или `()` | Класс с `__iter__`/`__next__` | `range()` |
| **Тип** | `generator` | Свой класс | `range` |
| **len()** | ❌ Нет | ❌ Нет | ✅ Да |
| **Индексация** | ❌ Нет | ❌ Нет | ✅ Да |
| **Срезы** | ❌ Нет | ❌ Нет | ✅ Да |
| **Повторное использование** | ❌ Одноразовый | ❌ Одноразовый | ✅ Многоразовый |
| **in оператор** | ✅ Опустошает | ✅ Опустошает | ✅ Не опустошает |
| **.send()/.throw()** | ✅ Да | ❌ Нет | ❌ Нет |
