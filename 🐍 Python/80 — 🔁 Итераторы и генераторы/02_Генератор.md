# 🎁 Генератор в Python

## 📌 Коротко

> Генератор — удобный способ создания итераторов через функцию с `yield`. Подкласс Iterator с дополнительными методами `.send()`, `.throw()`, `.close()` для управления выполнением.

---

## 💡 Полезные советы

- **`yield` приостанавливает с сохранением состояния** — ключевое слово генератора
- **Генератор ЯВЛЯЕТСЯ итератором** — имеет `__iter__()` и `__next__()`
- **Одноразовый** — истощается после использования, как все итераторы
- **Ленивый** — вычисляет значения по требованию, экономит память
- **`.send()` уникален для генератора** — отправка данных В генератор
- **`yield from` делегирует** — передача в вложенный генератор
- **Генераторное выражение компактно** — `(x for x in ...)`

---

## 📖 Теория

### Что такое yield?

**`yield`** — ключевое слово, которое:
1. Возвращает значение (как `return`)
2. Приостанавливает функцию
3. Сохраняет состояние (все локальные переменные)
4. При следующем `next()` продолжает с той же позиции

```python
def simple_gen():
    print('Перед первым yield')
    yield 1
    print('Между yield')
    yield 2
    print('После последнего yield')

gen = simple_gen()
print(next(gen))  
# Вывод: Перед первым yield
#        1

print(next(gen))  
# Вывод: Между yield
#        2

# print(next(gen))  # StopIteration
```

**Состояние сохраняется между вызовами!**

### yield vs return

| Характеристика | `return` | `yield` |
|---|---|---|
| Завершает функцию | ✅ Да | ❌ Приостанавливает |
| Сохраняет локальные переменные | ❌ Нет | ✅ Да |
| Количество значений | 1 | Много |
| Возвращаемый тип | Любой | Generator |

### return в генераторах

```python
def generate_ints():
    yield 1
    yield 2
    return 3  # Возбуждает StopIteration
    yield 4   # Никогда не выполнится

for num in generate_ints():
    print(num)
# 1
# 2
```

## 💻 Функции-генераторы

### Базовая структура

```python
def my_generator():
    yield value1
    yield value2
    yield value3

gen = my_generator()
print(next(gen))  # value1
```

### Примеры

**Счётчик:**
```python
def counter(low, high):
    for num in range(low, high + 1):
        yield num

for i in counter(3, 5):
    print(i)  # 3, 4, 5
```

**Бесконечный генератор:**
```python
def even_numbers(begin):
    begin += begin % 2
    while True:
        yield begin
        begin += 2

evens = even_numbers(10)
for i, num in enumerate(evens):
    if i > 3:
        break
    print(num)  # 10, 12, 14, 16
```

**Факториалы:**
```python
def factorials():
    value = 1
    index = 1
    while True:
        yield value
        index += 1
        value *= index

facts = factorials()
for i, num in enumerate(facts, 1):
    if i > 5:
        break
    print(f'{i}! = {num}')
# 1! = 1
# 2! = 2
# 3! = 6
# 4! = 24
# 5! = 120
```

## 📦 Генераторные выражения

### Синтаксис

```python
# List comprehension (квадратные скобки)
list_comp = [x**2 for x in range(5)]
print(type(list_comp))  # <class 'list'>

# Generator expression (круглые скобки)
gen_expr = (x**2 for x in range(5))
print(type(gen_expr))   # <class 'generator'>
print(list(gen_expr))   # [0, 1, 4, 9, 16]
```

### Примеры

```python
squares = (i**2 for i in range(1, 7))
evens = (x for x in range(10) if x % 2 == 0)
pairs = ((x, y) for x in range(3) for y in range(3))

for num in squares:
    print(num)  # 1, 4, 9, 16, 25, 36
```

### Память

```python
from sys import getsizeof

list_obj = list(range(1000000))
gen_obj = (x for x in range(1000000))

print(getsizeof(list_obj))   # ~8MB
print(getsizeof(gen_obj))    # ~104 байта
```

## 🔗 yield from - Делегирование

```python
# БЕЗ yield from
def get_data_old():
    for num in range(5):
        yield num
    for char in 'ABC':
        yield char

# С yield from (компактнее!)
def get_data():
    yield from range(5)
    yield from 'ABC'

print(list(get_data()))  # [0, 1, 2, 3, 4, 'A', 'B', 'C']
```

### Вложенные генераторы

```python
def flatten(nested):
    for item in nested:
        if isinstance(item, (list, tuple)):
            yield from flatten(item)  # Рекурсия!
        else:
            yield item

data = [1, [2, 3, [4, 5]], 6, [7]]
print(list(flatten(data)))  # [1, 2, 3, 4, 5, 6, 7]
```

## 📤 Методы генератора: .send(), .throw(), .close()

### .send() — Отправка данных В генератор

**Уникальное свойство генератора!** Можно отправлять данные в работающий генератор.

```python
def echo():
    value = None
    while True:
        value = yield value  # yield ПОЛУЧАЕТ значение

gen = echo()
next(gen)  # Инициализация (важно!)

print(gen.send("Hello"))   # Hello
print(gen.send("World"))   # World
print(gen.send("Python"))  # Python
```

### Практический пример: Калькулятор

```python
def calculator():
    total = 0
    while True:
        x = yield total  # Получить число, вернуть сумму
        if x is not None:
            total += x

calc = calculator()
next(calc)  # Инициализация

print(calc.send(5))    # 5
print(calc.send(3))    # 8
print(calc.send(2))    # 10
print(calc.send(10))   # 20
```

### .throw() — Отправка исключения

```python
def gen_with_error_handling():
    try:
        yield 1
        yield 2
        yield 3
    except ValueError as e:
        print(f"Поймано: {e}")
        yield "После ошибки"

gen = gen_with_error_handling()
print(next(gen))               # 1
print(next(gen))               # 2
print(gen.throw(ValueError, "Test error"))
# Вывод: Поймано: Test error
#        После ошибки
```

### .close() — Завершение генератора

```python
def cleanup_gen():
    try:
        yield 1
        yield 2
        yield 3
    finally:
        print("Очистка ресурсов")

gen = cleanup_gen()
print(next(gen))  # 1
print(next(gen))  # 2
gen.close()       # Завершить генератор
# Вывод: Очистка ресурсов

try:
    next(gen)  # StopIteration
except StopIteration:
    print("Генератор закрыт")
```

## 🔀 Конвейеры генераторов

### Композиция генераторов

```python
def integers(n):
    for i in range(1, n + 1):
        yield i

def evens(iterable):
    for i in iterable:
        if i % 2 == 0:
            yield i

def squared(iterable):
    for i in iterable:
        yield i * i

# Создаём конвейер
chain = squared(evens(integers(10)))
print(list(chain))  # [4, 16, 36, 64, 100]
```

### С генераторными выражениями

```python
n = 10

integers = (i for i in range(1, n + 1))
evens = (i for i in integers if i % 2 == 0)
squared = (i * i for i in evens)

print(list(squared))  # [4, 16, 36, 64, 100]
```
