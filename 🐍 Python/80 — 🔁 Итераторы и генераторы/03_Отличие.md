# 🔄 Отличие: Iterator vs Generator

## 📌 Коротко

> Генератор ЯВЛЯЕТСЯ подтипом итератора. Итератор — протокол, генератор — реализация. Главное отличие: `yield` и методы `.send()`, `.throw()`, `.close()`.

---

## 📖 Теория

### Итератор — Протокол (интерфейс)

```
Iterator (протокол) = правила реализации
├── Методы:
│   ├── __iter__() — возвращает self
│   └── __next__() — возвращает следующий элемент или StopIteration
└── Примеры реализации:
    ├── Классы с __iter__ и __next__
    ├── Встроенные: map, filter, zip, enumerate
    └── Генераторы (тоже реализуют этот протокол)
```

### Генератор — Конкретная реализация

```
Generator (реализация) = функция с yield
├── Автоматически реализует Iterator протокол
├── Дополнительные методы:
│   ├── .send(value) — отправка данных
│   ├── .throw(exc) — отправка исключения
│   └── .close() — завершение
└── Типы:
    ├── Функция-генератор (def с yield)
    └── Генераторное выражение ((x for x in ...))
```

### Сравнение

| Характеристика | Итератор | Генератор |
|---|---|---|
| **Что это?** | Протокол (правила) | Реализация |
| **Создание** | Класс с `__iter__` и `__next__` | Функция с `yield` |
| **Код** | ~15-20 строк | ~5-10 строк |
| **`yield`** | ❌ Нет | ✅ Главное отличие! |
| **Доп. методы** | Только `__iter__`, `__next__` | + `.send()`, `.throw()`, `.close()` |
| **Отношение** | Базовый протокол | Подкласс итератора |
| **.send()** | ❌ Нельзя | ✅ Можно |

## 💻 Практический пример

### Итератор (класс) — 17 строк

```python
class Counter:
    def __init__(self, low, high):
        self.current = low
        self.high = high
    
    def __iter__(self):
        return self
    
    def __next__(self):
        if self.current > self.high:
            raise StopIteration
        else:
            self.current += 1
            return self.current - 1

counter = Counter(1, 5)
print(list(counter))  # [1, 2, 3, 4, 5]
```

### Генератор (функция) — 5 строк

```python
def counter(low, high):
    while low <= high:
        yield low
        low += 1

gen = counter(1, 5)
print(list(gen))  # [1, 2, 3, 4, 5]
```

**Генератор в 3-4 раза компактнее!**

## 🎯 Уникальная возможность: .send()

### Итератор не может получать данные

```python
class SimpleIterator:
    def __init__(self):
        self.value = 0
    
    def __iter__(self):
        return self
    
    def __next__(self):
        self.value += 1
        if self.value > 3:
            raise StopIteration
        return self.value

it = SimpleIterator()
# it.send(100)  # ❌ AttributeError!
```

### Генератор может получать данные

```python
def echo():
    value = None
    while True:
        value = yield value  # Получаем значение!

gen = echo()
next(gen)  # Инициализация

print(gen.send("Hello"))   # Hello
print(gen.send("World"))   # World
print(gen.send("Python"))  # Python

# ✅ Это работает!
```
