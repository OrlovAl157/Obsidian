## ⚡ Главное помнить

- Аннотации типов не меняют поведение программы сами по себе.
- Python остаётся языком с динамической типизацией.
- Основная польза аннотаций раскрывается вместе со статическим анализом.
- В современном Python предпочтителен новый синтаксис: `list[int]`, `str | None`, `type Alias = ...`.
