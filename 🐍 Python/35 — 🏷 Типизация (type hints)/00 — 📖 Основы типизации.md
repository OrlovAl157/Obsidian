# 🏷️ Основы типизации в Python

## 📌 Коротко

> Типизация (type hints) — это аннотации типов для переменных, параметров и возвращаемых значений. Помогает отловить ошибки и улучшить читаемость кода.

---

## 💡 Полезные советы

- **Type hints не влияют на выполнение** — это только для разработчиков
- **Используйте type checkers** — mypy, pyright для проверки
- **Аннотируйте сигнатуры функций** — это самое важное
- **Optional[T] вместо T | None** — для совместимости со старыми версиями Python
- **Union для нескольких типов** — `Union[int, str]`
- **Generics для контейнеров** — `List[int]`, `Dict[str, int]`

---

## 🚀 Быстрая справка

| Что | Синтаксис | Пример |
|---|---|---|
| **Переменная** | `var: Type = value` | `age: int = 25` |
| **Функция** | `def func(x: Type) -> Type:` | `def add(a: int, b: int) -> int:` |
| **Optional** | `Optional[Type]` | `name: Optional[str] = None` |
| **Union** | `Union[Type1, Type2]` | `value: Union[int, str]` |
| **List** | `List[Type]` | `numbers: List[int] = [1, 2, 3]` |
| **Dict** | `Dict[KeyType, ValueType]` | `data: Dict[str, int]` |

---

## 📖 Теория

### Зачем нужна типизация?

1. **Автоматическое обнаружение ошибок** — type checker найдёт несоответствия
2. **Улучшенная автодополнение** — IDE лучше поймёт код
3. **Самодокументирование** — сразу видно какие типы ожидаются
4. **Рефакторинг** — безопаснее менять код

### Базовые типы

```python
# Встроенные типы
age: int = 25
name: str = "Alice"
height: float = 1.75
is_active: bool = True
nothing: None = None

# Встроенные контейнеры
numbers: list = [1, 2, 3]
data: dict = {"key": "value"}
items: tuple = (1, 2, 3)
unique: set = {1, 2, 3}
```

### Функции с типами

```python
def greet(name: str) -> str:
    """Приветствие с типами"""
    return f"Hello, {name}!"

def add(a: int, b: int) -> int:
    """Сумма двух чисел"""
    return a + b

def process(data: list) -> bool:
    """Обработка данных"""
    return len(data) > 0
```

## 🎯 Примеры

### Простая функция

```python
def multiply(x: int, y: int) -> int:
    return x * y

# Type checker проверит:
result = multiply(3, 5)      # ✅ OK
result = multiply("3", 5)    # ❌ Error: Expected int
```

### С Optional

```python
from typing import Optional

def find_user(user_id: int) -> Optional[str]:
    """Возвращает имя или None"""
    if user_id == 1:
        return "Alice"
    return None

# Использование
user = find_user(1)  # str | None
```
