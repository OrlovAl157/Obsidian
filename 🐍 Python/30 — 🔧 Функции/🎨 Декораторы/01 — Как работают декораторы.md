# 📋 Памятка: Как работают декораторы в Python

## 🎯 Главное правило

**Декораторы применяются СНИЗУ ВВЕРХ, а выполняются СВЕРХУ ВНИЗ!**

---

## 📝 Пример кода

```python
@декоратор3
@декоратор2
@декоратор1
def функция():
    return 'результат'
```

**Это эквивалентно:**

```python
функция = декоратор3(декоратор2(декоратор1(функция)))
```

---

## ⚙️ Две фазы выполнения

### 🔵 ФАЗА 1: Декорирование (при загрузке модуля)

**Порядок:** изнутри наружу (снизу вверх)

```
1. декоратор1(функция) → создаёт wrapper1
2. декоратор2(wrapper1) → создаёт wrapper2
3. декоратор3(wrapper2) → создаёт wrapper3
4. функция = wrapper3
```

**Что выполняется:**

- Тела декораторов (print внутри `def декоратор()`)
- Создаются wrapper'ы
- НЕ выполняются wrapper'ы!

---

### 🟢 ФАЗА 2: Вызов функции (при вызове `функция()`)

**Порядок:** сверху вниз (от внешнего к внутреннему)

```
1. wrapper3 выполняется → вызывает func()
2. wrapper2 выполняется → вызывает func()
3. wrapper1 выполняется → вызывает func()
4. оригинальная функция выполняется
```

**Возврат значений:** снизу вверх

```
оригинальная функция → результат
       ↓
wrapper1 → преобразует результат
       ↓
wrapper2 → преобразует результат
       ↓
wrapper3 → преобразует результат
       ↓
итоговый результат
```

## 💡 Пример 1: С выводом при декорировании

```python
def декоратор1(func):
    print('A')  # ← Выполнится при декорировании
    def wrapper():
        print('1')  # ← Выполнится при вызове
        return func()
    return wrapper

def декоратор2(func):
    print('B')  # ← Выполнится при декорировании
    def wrapper():
        print('2')  # ← Выполнится при вызове
        return func()
    return wrapper

@декоратор2
@декоратор1
def hello():
    print('HELLO')
    return 'результат'

# Здесь уже вывелось: A, B

hello()  # Выведет: 2, 1, HELLO
```

**📊 Вывод программы:**

```
A
B
2
1
HELLO
```

## 🎯 Пример 2: Самый наглядный (ДО и ПОСЛЕ функции)

**Этот пример идеально показывает порядок выполнения!**

```python
def first(func):
    def wrapper():
        print('1')      # ← ДО вызова func()
        func()          # ← Вызываем следующую функцию
        print('2')      # ← ПОСЛЕ вызова func()
    return wrapper

def second(func):
    def wrapper():
        print('3')      # ← ДО вызова func()
        func()          # ← Вызываем следующую функцию
        print('4')      # ← ПОСЛЕ вызова func()
    return wrapper

@second
@first
def python():
    print('PYTHON')

python()
```

**📊 Вывод программы:**

```
3
1
PYTHON
2
4
```

### 🔍 Разбор по шагам:

```
Вызов python()
  ↓
wrapper из second:
  print('3')           → выводит 3
  вызывает func() ──────┐
                        ↓
                   wrapper из first:
                     print('1')      → выводит 1
                     вызывает func() ──────┐
                                           ↓
                                      оригинальная python():
                                        print('PYTHON') → выводит PYTHON
                                        возврат ←──────┘
                     print('2')      → выводит 2
                     возврат ←──────┘
  print('4')           → выводит 4
```

**📊 Визуально как "сэндвич":**

```
      3  ← second начинается
      1  ← first начинается
  PYTHON ← оригинальная функция
      2  ← first заканчивается
      4  ← second заканчивается
```

💡 **Запомни:** Декораторы работают как **матрёшки** — внешний декоратор начинается первым и заканчивается последним!

## 🔑 Ключевые моменты

### 1️⃣ Декорирование = вложенные вызовы

```python
@декоратор3
@декоратор2
@декоратор1
def func(): ...

# ↓ ↓ ↓

func = декоратор3(декоратор2(декоратор1(func)))
```

### 2️⃣ Каждый декоратор получает результат предыдущего

```
func → декоратор1 → wrapper1
wrapper1 → декоратор2 → wrapper2
wrapper2 → декоратор3 → wrapper3
```

### 3️⃣ При вызове `func()` — цепочка wrapper'ов

```
func() = wrapper3()
  wrapper3 вызывает wrapper2()
    wrapper2 вызывает wrapper1()
      wrapper1 вызывает оригинальную func()
```

### 4️⃣ Результат возвращается обратно по цепочке

```
оригинальная func → возврат
  wrapper1 преобразует → возврат
    wrapper2 преобразует → возврат
      wrapper3 преобразует → итоговый результат
```

---

## 🎓 Шпаргалка

|Что|Когда|Порядок|
|---|---|---|
|`print` внутри `def декоратор()`|При декорировании|Снизу вверх|
|`print` внутри `def wrapper()`|При вызове функции|Сверху вниз|
|Создание wrapper'ов|При декорировании|Снизу вверх|
|Выполнение wrapper'ов|При вызове функции|Сверху вниз|
|Возврат значений|При вызове функции|Снизу вверх|

## ✅ Быстрая проверка понимания

Если видишь:

```python
@A
@B
@C
def f(): return 'x'
```

**Спроси себя:**

1. При загрузке что выведется? → Выполнятся тела A, B, C (порядок: C, B, A)
2. При вызове `f()` что выведется? → Выполнятся wrapper'ы (порядок: A, B, C, оригинальная f)
3. Как преобразуется результат? → C получает 'x', преобразует → B преобразует → A преобразует

---

## 🚀 Запомни главное

**"Декораторы надеваются как одежда: сначала футболка, потом свитер, потом куртка. Снимаются в обратном порядке!"**

```
Надевание (декорирование):  футболка → свитер → куртка
Снимание (выполнение):      куртка → свитер → футболка
```
