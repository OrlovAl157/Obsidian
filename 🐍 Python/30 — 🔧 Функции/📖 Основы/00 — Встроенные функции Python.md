## Быстрый справочник

|Функция             |Назначение                      |Возвращает  |Пример                           |
|--------------------|--------------------------------|------------|---------------------------------|
|`callable(obj)`     |Проверка вызываемости объекта   |`True/False`|`callable(int)` → `True`         |
|`hasattr(obj, name)`|Проверка существования атрибута |`True/False`|`hasattr('hi', 'upper')` → `True`|
|`hash(obj)`         |Получение хеш-значения          |`int`       |`hash('text')` → `число`         |
|`help(obj)`         |Вывод документации              |`None`      |`help(print)` → документация     |
|`repr(obj)`         |Формальное представление объекта|`str`       |`repr([1, 2])` → `'[1, 2]'`      |
|`eval(expression)`  |Выполнение выражения            |результат   |`eval('2+2')` → `4`              |
|`exec(code)`        |Выполнение блока кода           |`None`      |`exec('x = 5')` → `None`         |

-----

## 1. callable()

**Проверяет, является ли объект вызываемым**

```python
callable(int)        # True
callable(list)       # True
callable(100)        # False
callable([1, 2, 3])  # False
```

**Вызываемые объекты:**

- Функции, методы
- Классы
- Объекты с методом `__call__()`

## 2. hasattr()

**Проверяет существование атрибута у объекта**

```python
hasattr('stepik', 'isalpha')  # True
hasattr([1, 2, 3], 'sort')    # True
hasattr(13, 'to_str')         # False
```

**Синтаксис:** `hasattr(object, name)`

**Связанные функции:**

- `getattr(obj, name, default)` — получить значение атрибута
- `setattr(obj, name, value)` — установить значение атрибута

## 3. hash()

**Возвращает хеш-значение объекта**

```python
hash(899)          # 899
hash('timyrik')    # 1468304915
hash((1, 2, 3))    # -378539185
hash(False)        # 0
hash(True)         # 1
```

### Хешируемость типов данных:

|Тип                                     |Хешируемый                 |
|----------------------------------------|---------------------------|
|`bool`, `int`, `float`, `complex`, `str`|✔️                          |
|`tuple`                                 |✔️ (если элементы хешируемы)|
|`frozenset`                             |✔️                          |
|`list`, `set`, `dict`                   |❌                          |

⚠️ **Особенности:**

- Хеш целых чисел = самому числу (кроме `-1`)
- `hash(-1) == hash(-2) == -2` (особенность CPython)
- Хеш-функция может меняться в разных версиях Python
- Используется алгоритм SipHash

## 4. help()

**Выводит документацию по объекту**

```python
help(print)     # Документация функции print
help('sorted')  # Документация функции sorted
help()          # Интерактивная справка (quit для выхода)
```

**Применение:** быстрое изучение API, параметров функций, примеров использования

## 5. repr()

**Возвращает формальное строковое представление объекта**

```python
repr('stepik')              # "'stepik'"
repr([1, 2, 3, 4])          # '[1, 2, 3, 4]'
repr(date(2022, 1, 16))     # 'datetime.date(2022, 1, 16)'
```

**Цель:** получить строку, понятную интерпретатору Python (для отладки)

## 6. eval()

**Выполняет строку-выражение и возвращает результат**

```python
eval('7 + 10')                    # 17
eval('len([1, 1, 1, 1, 1])')      # 5
eval('[i**2 for i in [1,2,3]]')   # [1, 4, 9]
```

### Парсинг объектов:

```python
eval("['Python', 'C#', 'Java']")      # список
eval('(1, 2, 3, 4, 5)')               # кортеж
eval("{1: 'January', 2: 'February'}") # словарь
```

### Контроль области видимости:

```python
# Ограничение доступа к переменным
eval('x + 10', {'x': 5})              # 15
eval('sum([1,2,3])', {'__builtins__': {}})  # Ошибка! Нет доступа к sum
```

**Что можно использовать:**

- Арифметические операции, сравнения
- Списочные выражения, генераторы
- Вызовы функций
- Доступ к переменным

**Что НЕЛЬЗЯ использовать:**

- `if`, `for`, `while`, `def`, `class`, `import`, `raise` и т.д.

⚠️ **Важно:**

- Имеет доступ к встроенным функциям и переменным
- **НЕБЕЗОПАСНА** — выполняет произвольный код!
- Используйте параметры `globals` и `locals` для контроля доступа

## 7. exec()

**Выполняет блок кода, возвращает None**

```python
code = '''a = 10
b = 20
print(a + b)'''
exec(code)  # Выводит: 30
```

### Контроль области видимости:

```python
# Изоляция выполнения кода
namespace = {}
exec('x = 100', namespace)
print(namespace['x'])  # 100 (x создан в namespace, а не в глобальной области)
```

### Отличия от eval():

|Критерий  |`eval()`            |`exec()`           |
|----------|--------------------|-------------------|
|Выполняет |Выражение           |Блок кода          |
|Возвращает|Результат выражения |Всегда `None`      |
|Операторы |Нельзя (`if`, `for`)|Можно              |
|Назначение|Вычисление          |Выполнение скриптов|

⚠️ **Важно:**

- Имеет доступ к встроенным функциям и переменным
- Может использовать любые операторы (`if`, `for`, `def` и т.д.)
- **НЕБЕЗОПАСНА** — выполняет произвольный код!
- Используйте параметры `globals` и `locals` для контроля доступа

## Практическое применение

**callable()** — проверка перед вызовом функции, работа с колбэками

```python
if callable(my_func):
    my_func()
```

**hasattr()** — проверка API объекта, утиная типизация

```python
if hasattr(obj, 'save'):
    obj.save()
```

**hash()** — использование объектов как ключей словаря, работа с множествами

```python
# Кортежи можно использовать как ключи
cache = {(1, 2): 'result'}
```

**repr()** — отладка, логирование

```python
print(f'Debug: {repr(my_object)}')
```

**eval()** — калькуляторы, парсинг простых данных (JSON-подобных строк)

```python
user_input = "2 * (3 + 5)"
result = eval(user_input)  # 16
```

**exec()** — динамическое выполнение скриптов, плагины

```python
plugin_code = "def greet(): return 'Hello'"
exec(plugin_code)
greet()  # 'Hello'
```

## Ключевые моменты безопасности

🔒 **Функции `eval()` и `exec()` выполняют произвольный код!**

- Никогда не используйте с ненадежным вводом пользователя
- Всегда ограничивайте доступ через `globals` и `locals`
- Рассмотрите альтернативы: `ast.literal_eval()` для парсинга данных
- Используйте только с полностью контролируемым кодом
