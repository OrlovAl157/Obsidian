# 🔧 Параметры и аргументы функций

## 📌 Коротко

> **Параметр** — то что функция ожидает получить (в объявлении).  
> **Аргумент** — то что передаётся при вызове.

```python
def greet(name):   # name — ПАРАМЕТР
    print(name)

greet('Саша')      # 'Саша' — АРГУМЕНТ
```

---

## 🚀 Быстрая справка

|Тип параметра|Синтаксис|Описание|
|---|---|---|
|Позиционный|`def f(a)`|передаётся по порядку|
|Со значением по умолчанию|`def f(a=1)`|необязательный|
|Только позиционный|`def f(a, /)`|нельзя передать по имени|
|Только именованный|`def f(*, a)`|нельзя передать позиционно|
|Сборщик позиционных|`def f(*args)`|собирает остаток в кортеж|
|Сборщик именованных|`def f(**kwargs)`|собирает остаток в словарь|

---

## 📖 Теория

### Полная сигнатура функции

```python
def f(a, b=1, /, c, *args, d, **kwargs):
    pass
#   ↑   ↑     ↑  ↑   ↑    ↑   ↑
#   |   |     |  |   |    |   kwargs — сборщик именованных
#   |   |     |  |   |    d — только именованный (после *)
#   |   |     |  |   *args — сборщик позиционных
#   |   |     |  c — обычный позиционный
#   |   |     / — всё ДО него только позиционно
#   |   b=1 — со значением по умолчанию
#   a — обычный позиционный
```

### Порядок параметров — обязательный!

```
def f( позиционные / обычные * keyword-only **kwargs )
       └── до / ──┘           └─── после * ───┘
```

---

## 💻 Примеры кода

### 1. Обычный позиционный параметр

```python
def f(a, b):
    print(a, b)

f(1, 2)        # ✅ → 1 2
f(b=2, a=1)   # ✅ → 1 2  (можно передать по имени)
```

### 2. Параметр со значением по умолчанию

```python
def greet(name, greeting='Привет'):
    print(f'{greeting}, {name}!')

greet('Саша')              # → Привет, Саша!
greet('Саша', 'Здарова')   # → Здарова, Саша!
```

> ⚠️ Параметры со значением по умолчанию всегда **после** обязательных!

### 3. Только позиционный (`/`)

```python
def f(a, b, /):
    print(a, b)

f(1, 2)        # ✅ → 1 2
f(a=1, b=2)   # ❌ TypeError — нельзя по имени!
```

### 4. Только именованный (`*`)

```python
def f(a, *, b, c):
    print(a, b, c)

f(1, b=3, c=4)   # ✅ → 1 3 4
f(1, 2, 3)       # ❌ TypeError — b и c только по имени!
```

### 5. Сборщик позиционных (`*args`)

```python
def f(a, *args):
    print(a, args)

f(1, 2, 3, 4)   # → 1 (2, 3, 4)
#              ↑ args — это кортеж остатка
```

### 6. Сборщик именованных (`**kwargs`)

```python
def f(**kwargs):
    print(kwargs)

f(name='Саша', age=20)   # → {'name': 'Саша', 'age': 20}
#                         ↑ kwargs — это словарь
```

### 7. Всё вместе

```python
def f(a, b=1, /, c, *args, d, **kwargs):
    print(a, b, c, args, d, kwargs)

f(1, 2, 3, 4, 5, d=6, x=7)
# a=1, b=2, c=3, args=(4,5), d=6, kwargs={'x': 7}
```

## ⚠️ Частые ошибки

```python
# ❌ Параметр со значением до обязательного
def f(a=1, b):   # SyntaxError!
    pass

# ❌ **kwargs не последний
def f(**kwargs, a):   # SyntaxError!
    pass

# ❌ Передаём позиционно после *
def f(a, *, b):
    pass
f(1, 2)   # TypeError — b только именованный!
```

## 💡 Тонкости

### `*` без имени vs `*args`

```python
# * без имени — просто разделитель, кортеж не создаётся
def f(a, *, b):
    print(a, b)        # args недоступен

# *args — создаёт кортеж из остатка
def f(a, *args, b):
    print(a, args, b)  # args доступен как кортеж
```

### `/` и `*` вместе

```python
def f(a, /, b, *, c):
#     ↑       ↑
# только      только
# позиционно  именованно
#        ↑
#     и так и так

f(1, 2, c=3)      # ✅
f(1, b=2, c=3)    # ✅
f(a=1, b=2, c=3)  # ❌ — a только позиционно
f(1, 2, 3)        # ❌ — c только именованно
```

### Опасный дефолтный аргумент — изменяемый объект!

```python
# ❌ Антипаттерн — список создаётся один раз при объявлении!
def f(items=[]):
    items.append(1)
    return items

print(f())   # [1]
print(f())   # [1, 1] — список накапливается!

# ✅ Правильно
def f(items=None):
    if items is None:
        items = []
    items.append(1)
    return items
```
