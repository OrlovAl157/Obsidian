## Что создаёт своё пространство имён?

|Конструкция|Создаёт пространство?|Что содержит|
|---|---|---|
|**Модуль**|✅ Да|Глобальные переменные|
|**Функция/метод**|✅ Да|Локальные аргументы и переменные|
|**Класс**|✅ Да|Атрибуты и методы класса|
|**Comprehension**|✅ Да|Переменные цикла comprehension|
|**Lambda**|✅ Да|Аргументы lambda|
|**Generator**|✅ Да|Переменные генератора|
|**Встроенные** (`builtins`)|✅ Да|`len`, `print`, `int`, etc.|
|**Import**|✅ Да|Объекты импортированного модуля|
|**exec/eval**|✅ Да|По словарю, переданному явно|

---

## Примеры

### Модуль

```python
# mymodule.py
x = 10  # В пространстве имён модуля
```

### Функция

```python
def func(a, b):  # a, b - в локальном пространстве
    c = a + b    # c - в локальном пространстве
    return c
```

### Класс

```python
class MyClass:
    x = 10          # В пространстве имён класса
    def method(self):  # В пространстве имён класса
        pass
```

### Comprehension

```python
# i существует только внутри comprehension
nums = [i*2 for i in range(5)]
# i недоступна снаружи!
```

### Lambda

```python
f = lambda x, y: x + y  # x, y - в пространстве lambda
```

### Generator

```python
def gen():
    i = 0  # В пространстве генератора
    while i < 5:
        yield i
        i += 1
```

### Встроенные

```python
# len, print, int - всегда доступны
# Находятся в встроенном пространстве имён
len([1, 2, 3])
```

### Import

```python
import math
# math и всё его содержимое - в своём пространстве
math.pi
```

### exec/eval

```python
namespace = {'x': 10}
exec('y = x + 5', namespace)
print(namespace['y'])  # 15
```

## Что НЕ создаёт пространство имён?

|Конструкция|Использует пространство|
|---|---|
|`if/else`|❌ Родительское|
|`for/while`|❌ Родительское|
|`try/except`|❌ Родительское|
|`with`|❌ Родительское|

```python
def func():
    if True:
        x = 10  # x в пространстве func, не if
    print(x)  # 10 - доступна!
    
    for i in range(5):
        y = i  # y в пространстве func
    print(y)  # 4 - доступна!
```
