# 📍 Переменные в SQL

## Пользовательские переменные

Начинаются с `@`. Живут в рамках сессии.

### Объявление и использование

```sql
SET @variable := 1;

SELECT @variable;
-- 1
```

### Несколько переменных сразу

```sql
SET @variable1 := 1,
    @variable2 := 2;

SELECT @variable1, @variable2;
-- 1 | 2
```

### Присвоение из SELECT

```sql
-- Способ 1: через SET
SET @min_price := (SELECT MIN(price) FROM Books);
SELECT @min_price;

-- Способ 2: через INTO
SELECT MIN(price) INTO @min_price FROM Books;
SELECT @min_price;

-- Несколько значений
SELECT MIN(price), MAX(price) INTO @min_price, @max_price
FROM Books;

SELECT @min_price, @max_price;
```

## Системные переменные

Начинаются с `@@`. Делятся на глобальные (`@@global`) и сессионные (`@@session`).

### Установка значений

```sql
SET @@sort_buffer_size := 262146;
-- сессионная, по умолчанию

SET @@session.sort_buffer_size := 262146;
-- сессионная, явно

SET @@global.sort_buffer_size := 262146;
-- глобальная
```

### Просмотр значений

```sql
SELECT @@version;
-- версия сервера

SELECT @@global.sort_buffer_size;
-- глобальное значение

SELECT @@session.sort_buffer_size;
-- сессионное значение
```

### Системная переменная для рекурсивных CTE

```sql
SET @@cte_max_recursion_depth := 10000;
-- увеличиваем глубину рекурсии, по умолчанию 1000
```
