# ⚡ Триггеры

## Синтаксис создания

Синтаксис с изменением разделителя.

```sql
DELIMITER //
CREATE TRIGGER trigger_name
{BEFORE | AFTER} {INSERT | UPDATE | DELETE}
ON table_name
FOR EACH ROW
BEGIN
    -- тело триггера
END //
DELIMITER ;
```

## Упрощённый синтаксис

Упрощённый вариант без `BEGIN` / `END`.

```sql
CREATE TRIGGER trigger_name
{BEFORE | AFTER} {INSERT | UPDATE | DELETE}
ON table_name
FOR EACH ROW
тело_триггера;
```

## Типы триггеров и доступ к данным

| Триггер | Доступные псевдонимы | Можно изменять |
|---------|----------------------|----------------|
| `BEFORE INSERT` | `NEW` | да, изменяет вставляемые значения |
| `AFTER INSERT` | `NEW` | нет |
| `BEFORE UPDATE` | `OLD`, `NEW` | да, изменяет `NEW` |
| `AFTER UPDATE` | `OLD`, `NEW` | нет |
| `BEFORE DELETE` | `OLD` | нет |
| `AFTER DELETE` | `OLD` | нет |

## Удаление триггера

```sql
DROP TRIGGER trigger_name;
```

## Примеры

### BEFORE INSERT

Автоматически заполняет `created_at`.

```sql
CREATE TRIGGER set_created_at
BEFORE INSERT ON users
FOR EACH ROW
SET NEW.created_at = NOW();
```

### BEFORE UPDATE

Проверяет, что цена не отрицательная.

```sql
CREATE TRIGGER check_price
BEFORE UPDATE ON products
FOR EACH ROW
BEGIN
    IF NEW.price < 0 THEN
        SET NEW.price = 0;
    END IF;
END;
```
