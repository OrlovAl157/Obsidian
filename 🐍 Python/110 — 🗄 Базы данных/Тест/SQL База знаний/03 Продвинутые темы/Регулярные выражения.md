# 🔍 Регулярные выражения в SQL

## Метасимволы

| Метасимвол | Что делает | Пример | Результат |
|------------|------------|--------|-----------|
| . | один любой символ | `.......` | beegeek |
| {n} | ровно n раз | `be{5}e` | beeeee |
| {n,k} | от n до k раз | `a{3,5}` | aaa, aaaa, aaaaa |
| {n,} | не менее n раз | `a{4,}` | aaaa, aaaaa |
| {,n} | не более n раз | `a{,4}` | aaa, aaaa |
| \ | экранирование | `pygen\.ru` | pygen.ru |
| \d | цифра 0-9 | `\d` | 9 |
| \D | не цифра | `bee\Dgee\D` | bee-geek |
| \s | пробельный символ | `...\s....` | bee geek |
| \S | не пробел | `...\S....` | bee-geek |
| \w | буква, цифра, _ | `\w\w\w\w` | _x1_ |
| \W | не буква, не цифра, не _ | `\W\W` | !? |
| [...] | один из набора | `[bd]` | b |
| [.-.] | диапазон | `[a-z]` | f |
| [^...] | не в диапазоне | `[^7-9]` | 6 |
| \| | ИЛИ | `[b-k]\|[4-8]` | d |
| + | 1 раз и более | `\d+` | 19587792597 |
| * | 0 раз и более | `aaa\d*` | aaa, aaa8657 |
| ? | 0 или 1 раз | `https?://` | https://, http:// |
| .*? | ленивый поиск | `<B>.*?</B>` | подстрока в тегах |
| \b | граница слова | `\bcat\b` | cat |
| \B | не граница слова | `\Bcat` | scat, ducati |
| ^ | начало строки | `^[Hh]ello` | Hello... |
| $ | конец строки | `[Gg]oodbye$` | ...Goodbye |
| (...) | группировка | `(SELECT){2}` | SELECTSELECT |
| (...)\n | ссылка назад | `(\b\w+\b)[ ]+\1` | повтор слов |
| (?P<name>) | именованная группа | `(?P<word>\b\w+\b)` | — |
| (?P=name) | использование имени | `(?P=word)` | — |

## Флаги
| Флаг | Что даёт |
|------|----------|
| c | учитывает регистр |
| m | \n учитывается как новая строка |
| n | `.` учитывает \n |

## Функции регулярных выражений

### REGEXP_LIKE()
Есть ли совпадение
```sql
REGEXP_LIKE('beegeek', 'bee', 'c')  -- 1
REGEXP_LIKE('beegeek', 'Bee', 'c')  -- 0
```

### REGEXP / RLIKE
```sql
'beegeek' REGEXP 'bee'      -- 1
'beegeek' NOT REGEXP 'bee'  -- 0
```

### REGEXP_INSTR()
```sql
REGEXP_INSTR('beegeek', 'e{2}', 1, 1, 1)  -- 4
```

### REGEXP_SUBSTR()
```sql
REGEXP_SUBSTR('beegeeek', 'e{2,}', 1, 1)  -- ee
REGEXP_SUBSTR('beegeeek', 'e{2,}', 1, 2)  -- eee
```

### REGEXP_REPLACE()
```sql
REGEXP_REPLACE('beeeeegeeek', 'a', 'i')   -- beeeeegeeek
```
