# 🔢 Числовые функции SQL

## DIV
Целочисленное деление
```sql
SELECT 11 DIV 3;  -- 3
```

## MOD (или %)
Остаток от деления
```sql
SELECT 11 MOD 3;  -- 2
SELECT 11 % 3;    -- 2
SELECT MOD(11, 3); -- 2
```

## ABS
Модуль числа
```sql
SELECT ABS(-2.1234);  -- 2.1234
```

## ROUND
Округление
```sql
SELECT ROUND(23.1653, 2);  -- 23.17
```

## POW
Возведение в степень
```sql
SELECT POW(2, 3);  -- 8
```

## SQRT
Квадратный корень
```sql
SELECT SQRT(9);  -- 3
```

## RAND
Случайное число от 0 до 1
```sql
SELECT RAND();  -- 0.8266121575856694
```

## FLOOR / CEILING
Округление вниз/вверх
```sql
SELECT FLOOR(-11.22);    -- -12
SELECT CEILING(18.39);   -- 19
```

## DEGREES / RADIANS
```sql
SELECT DEGREES(3.14);   -- 179.9087476710785
SELECT RADIANS(180);    -- 3.141592653589793
```

## PI()
```sql
SELECT PI();  -- 3.141593
```

## SIN / COS / TAN / COT
```sql
SELECT SIN(PI());   -- 0
SELECT COS(PI());   -- -1
SELECT TAN(PI());   -- 0
SELECT COT(PI());   -- -8165619676597685
```

## LEAST / GREATEST
```sql
SELECT LEAST(2, 1, 3, 5, 4);     -- 1
SELECT GREATEST(2, 1, 3, 5, 4);  -- 5
```

## CONV
Перевод между системами счисления
```sql
SELECT CONV('10F', 16, 2);  -- 100001111
```

## FORMAT
Форматирование числа
```sql
SELECT FORMAT(1000000.1653, 2);  -- 1,000,000.17
```

## LOG2 / LOG10
```sql
SELECT LOG2(65536);  -- 16
SELECT LOG10(100);   -- 2
```
