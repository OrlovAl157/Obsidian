# Оконные функции SQL

## Ранжирующие функции

### ROW_NUMBER()
Нумерация строк
```sql
SELECT full_name, department, salary,
       ROW_NUMBER() OVER (PARTITION BY department ORDER BY salary DESC) AS row_num
FROM Employees;
```

### DENSE_RANK()
Ранг без пропусков
```sql
SELECT full_name, salary,
       DENSE_RANK() OVER (ORDER BY salary DESC) AS place
FROM Employees;
```

### RANK()
Ранг с пропусками
```sql
SELECT full_name, salary,
       RANK() OVER (ORDER BY salary DESC) AS place
FROM Employees;
```

### NTILE(n)
Разбиение на n групп
```sql
SELECT id, full_name,
       NTILE(3) OVER (ORDER BY id) AS group_number
FROM Employees;
```

## Агрегатные функции как оконные
```sql
SELECT full_name, department, salary,
       AVG(salary) OVER (PARTITION BY department) AS department_avg,
       SUM(salary) OVER () AS total_salary,
       COUNT(*) OVER (PARTITION BY department) AS employees_in_dept,
       MIN(salary) OVER (PARTITION BY department) AS min_in_dept,
       MAX(salary) OVER (PARTITION BY department) AS max_in_dept
FROM Employees;
```

## WINDOW
```sql
SELECT full_name, salary,
       ROW_NUMBER() OVER salary_desc AS row_num,
       DENSE_RANK() OVER salary_desc AS dense_rank_num
FROM Employees
WINDOW salary_desc AS (ORDER BY salary DESC);
```

## Границы окон

### ROWS
```sql
SELECT id, full_name, salary,
       AVG(salary) OVER (ORDER BY id 
                         ROWS BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW) AS avg_salary
FROM Employees;
```

### RANGE
```sql
SELECT amount,
       COUNT(amount) OVER(ORDER BY amount 
                          RANGE BETWEEN 20 PRECEDING AND 20 FOLLOWING) - 1 AS diff_twenty
FROM Orders;

SELECT *,
       AVG(amount) OVER(ORDER BY date 
                        RANGE BETWEEN INTERVAL 2 DAY PRECEDING AND CURRENT ROW) AS avg_amount
FROM Orders;
```

## Функции смещения

### FIRST_VALUE() / LAST_VALUE()
```sql
SELECT id, full_name, salary,
       FIRST_VALUE(salary) OVER (ORDER BY id) AS first_salary,
       LAST_VALUE(salary) OVER (ORDER BY id 
                                ROWS BETWEEN UNBOUNDED PRECEDING AND UNBOUNDED FOLLOWING) AS last_salary
FROM Employees;
```

### NTH_VALUE()
```sql
SELECT id, full_name, salary,
       NTH_VALUE(salary, 2) OVER id_asc AS second_salary
FROM Employees
WINDOW id_asc AS (ORDER BY id ROWS BETWEEN UNBOUNDED PRECEDING AND UNBOUNDED FOLLOWING);
```

### LAG() / LEAD()
```sql
SELECT id, full_name, salary,
       LAG(salary, 1, 0) OVER (ORDER BY id) AS prev_salary,
       LEAD(salary, 1, 'empty') OVER (ORDER BY id) AS next_salary
FROM Employees;
```
