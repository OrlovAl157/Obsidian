# Переменные в SQL

## Пользовательские переменные

### Объявление и использование
```sql
SET @variable := 1;
SELECT @variable;  -- 1
```

### Несколько переменных
```sql
SET @var1 := 1, @var2 := 2;
SELECT @var1, @var2;
```

### Из подзапроса (SET)
```sql
SET @min_price := (SELECT MIN(price) FROM Books);
SELECT @min_price;
```

### Из подзапроса (INTO)
```sql
SELECT MIN(price) INTO @min_price FROM Books;
SELECT @min_price;
```

### Несколько значений INTO
```sql
SELECT MIN(price), MAX(price) INTO @min_price, @max_price FROM Books;
SELECT @min_price, @max_price;
```

## Системные переменные

### Сессионные
```sql
SET @@sort_buffer_size := 262146;
SET @@session.sort_buffer_size := 262146;
```

### Глобальные
```sql
SET @@global.sort_buffer_size := 262146;
```

### Просмотр значений
```sql
SELECT @@version;
SELECT @@global.sort_buffer_size, @@session.sort_buffer_size;
```
