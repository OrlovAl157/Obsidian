# Триггеры в SQL

## Синтаксис создания
```sql
DELIMITER //
CREATE TRIGGER trigger_name
{BEFORE | AFTER} {INSERT | UPDATE | DELETE}
ON table_name
FOR EACH ROW
BEGIN
    -- тело триггера
END //
DELIMITER ;
```

## Времена и операции

| Комбинация | Доступ к данным | Можно изменять |
|---|---|---|
| BEFORE INSERT | NEW.column | ДОБАВЛЯЕМУЮ запись |
| AFTER INSERT | NEW.column | ДОБАВЛЕННУЮ запись |
| BEFORE UPDATE | OLD.column, NEW.column | НОВЫЕ значения |
| AFTER UPDATE | OLD.column, NEW.column | — |
| BEFORE DELETE | OLD.column | — |
| AFTER DELETE | OLD.column | — |

## Упрощённый вариант
```sql
CREATE TRIGGER trigger_name
BEFORE INSERT ON table_name
FOR EACH ROW
SET NEW.created_at = NOW();
```

## Удаление триггера
```sql
DROP TRIGGER trigger_name;
```
