# 🔧 Дополнительные функции SQL

## ISNULL
Проверка на NULL
```sql
SELECT ISNULL(NULL);  -- 1
```

## IF
Условный выбор
```sql
SELECT IF(1 < 0, 'bee', 'geek');  -- geek
```

## COALESCE
Первое непустое значение
```sql
SELECT COALESCE(NULL, NULL, 2, 3);  -- 2
```

## IFNULL
Если первое NULL, то второе
```sql
SELECT IFNULL(NULL, 'geek');  -- geek
```

## NULLIF
Сравнение двух значений
```sql
SELECT NULLIF('bee', 'bee');   -- NULL
SELECT NULLIF('bee', 'geek');  -- bee
```

## CONVERT / CAST
Приведение типов
```sql
SELECT CONVERT(100.78, DECIMAL(4, 1));  -- 100.8
SELECT CAST(100.78 AS DECIMAL(3, 0));   -- 101
```

## CASE
Множественное условие

Вариант 1: сравнение с конкретными значениями
```sql
CASE quantity
    WHEN 0 THEN 'Out of stock'
    WHEN 1 THEN 'One in stock'
    WHEN 2 THEN 'Little in stock'
    ELSE 'A lot in stock'
END
```

Вариант 2: логические условия
```sql
CASE
    WHEN quantity = 0 THEN 'Out of stock'
    WHEN quantity = 1 THEN 'One in stock'
    WHEN quantity BETWEEN 2 AND 5 THEN 'Little in stock'
    ELSE 'A lot in stock'
END
```
