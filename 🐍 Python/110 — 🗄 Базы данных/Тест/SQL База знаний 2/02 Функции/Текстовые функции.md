# 📝 Текстовые функции SQL

## CONCAT
Объединяет строки
```sql
SELECT CONCAT('bee', 'geek');  -- beegeek
```

## CONCAT_WS
Объединяет с разделителем
```sql
SELECT CONCAT_WS('_', 'bee', 'geek');  -- bee_geek
```

## GROUP_CONCAT
Складывает значения из разных строк
```sql
SELECT GROUP_CONCAT(name SEPARATOR ', ') FROM users;
```

## CHAR_LENGTH()
Длина строки
```sql
SELECT CHAR_LENGTH('beegeek');  -- 7
```

## LOWER / UPPER
```sql
SELECT LOWER('BeeGeek');  -- beegeek
SELECT UPPER('BeeGeek');  -- BEEGEEK
```

## LTRIM / RTRIM
Удаляет пробелы слева/справа
```sql
SELECT LTRIM('   bee');   -- bee
SELECT RTRIM('bee   ');   -- bee
```

## LPAD / RPAD
Дополнение строки символами
```sql
SELECT LPAD('bee', 6, '-+-');   -- -+-bee
SELECT RPAD('bee', 6, '-+-');   -- bee-+-
```

## LEFT / RIGHT
```sql
SELECT LEFT('beegeek', 3);   -- bee
SELECT RIGHT('beegeek', 3);  -- eek
```

## SUBSTRING
Извлечение подстроки
```sql
SELECT SUBSTRING('beegeek', 4, 2);  -- ge
SELECT SUBSTRING('beegeek', -4);    -- geek
```

## SUBSTRING_INDEX
Извлечение по разделителю
```sql
SELECT SUBSTRING_INDEX('g-e-e-k', '-', 2);   -- g-e
SELECT SUBSTRING_INDEX('g-e-e-k', '-', -2);  -- e-k
```

## REPLACE
Замена (регистр важен!)
```sql
SELECT REPLACE('beegEek', 'e', 'i');  -- biigEik
```

## REVERSE
```sql
SELECT REVERSE('beegeek');  -- keegeeb
```

## REPEAT
```sql
SELECT REPEAT('bee', 3);  -- beebeebee
```

## LOCATE
Поиск позиции
```sql
SELECT LOCATE('eg', 'beegeek');  -- 3
```

## TRIM
Удаление символов по краям
```sql
SELECT TRIM(LEADING '-' FROM '--bee--');  -- bee--
SELECT TRIM(TRAILING '-' FROM '--bee--'); -- --bee
SELECT TRIM(BOTH '-' FROM '--bee--');     -- bee
```

## ORD / CHAR
```sql
SELECT ORD('A');           -- 65
SELECT CHAR(65);           -- A
```
