# Регулярные выражения

> Источник: лист Excel-конспекта. Перенесено в Markdown для Obsidian.

| Мета-символ | Что делает | Пример | Результат | Колонка 5 | Флаги | Что дают |
| --- | --- | --- | --- | --- | --- | --- |
| . | один любой символ | ....... | beegeek |  | c | учитывается регистр в регулярном выражении |
| {n} | n -количество повторяющихся символов | be{5}ge{4}k | beeeeegeeeek |  | m | метасимвол '\n' учитывается как новая строка |
| {n,k} | диапазон количества повторяющихся символов. от n до k. | a{3,5} | aaa<br>aaaa<br>aaaaa |  | n | метасимвол '.' учитывал '/n', как символ |
| {n,} | не менее n -количество повторяющихся символов | a{4,} | aaaa<br>aaaaa |  |  |  |
| {,n} | не более n -количество повторяющихся символов | a{,4} | aaa<br>aaaa |  |  |  |
| \ | экранирование символов | pygen\.ru | pygen.ru |  |  |  |
| \d | digit любая цифра от 0 до 9 | \d | 9 |  |  |  |
| \D | NON-digit любой нецифровой символ | bee\Dgee\D | bee-geek |  |  |  |
| \s | space пробелььный символ | ...\s.... | bee geek |  |  |  |
| \S | NON-space любой непробельный символ | ...\S.... | bee-geek |  |  |  |
| \w | word буквы в любом регистре, цифры, нижнее подчёркивание _ | \w\w\w\w | _x1_ |  |  |  |
| \W | NON-word символ НЕ буква, цифра или нижнее подчёркивание _ | \W\W | !? |  |  |  |
| [...] | соответствие одному из нескольких (набора) символов | [bd]<br>[12] | b<br>2 |  |  |  |
| [.-.] | соответствию символу одному из диапазаона | [a-z]<br>[1-9] | f<br>6 |  |  |  |
|  | несколько диапазонов: | [A-Za-z0-9]<br>[12a-p34]<br>[12-57] | [ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789]<br>[12abcdefghijklmnop34]<br>[123457] |  |  |  |
| [^...] | символ не включён в диапазон | [^7-9]<br>[^CIKLOV] | 6<br>B |  |  |  |
| \| | ИЛИ. соответствие одному из шаблонов | [b-k]\|[4-8] | d |  |  |  |
| + | данный символ может быть 1 раз и более подряд. {1,} | \d+<br>a+<br>[a-z.]+@[a-z.]+\.[a-z]+ | 19587792597<br>aaaaaaaa<br>spam@beegeek.com |  |  |  |
| * | данный символ может быть 0 раз и более подряд {0,} | aaa\d* | aaa<br>aaa8657<br>aaa4 |  |  |  |
| ? | данный символ может быть 0 или 1 раз. {0,1} | https?:// | https://<br>http:// |  |  |  |
|  | использование минимального расстояния для поиска по шаблону. Жадный/ленивый. | <B>.*</B> -- жадный | Подписку нельзя оформить клиентам, живущим в <B>России</B>, а также <B>Беларуси</B>. |  |  |  |
|  |  | <B>.*?</B> -- ленивый | Подписку нельзя оформить клиентам, живущим в <B>России</B>, а также <B>Беларуси</B>. |  |  |  |
| \b | border. Ищет символы по границе слова (пробел или _) | \bcat | Some words contain the word cat: scat, ducati, alcatel, catalyst, worldcat and of course cat! |  |  |  |
|  |  | cat\b  | Some words contain the word cat: scat, ducati, alcatel, catalyst, worldcat and of course cat! |  |  |  |
|  |  | \bcat\b | Some words contain the word cat: scat, ducati, alcatel, catalyst, worldcat and of course cat! |  |  |  |
| \B | NON-border. Символ находится не на границе слова. | \Bcat | Some words contain the word cat: scat, ducati, alcatel, catalyst, worldcat and of course cat! |  |  |  |
|  |  | cat\B  | Some words contain the word cat: scat, ducati, alcatel, catalyst, worldcat and of course cat! |  |  |  |
|  |  | \Bcat\B | Some words contain the word cat: scat, ducati, alcatel, catalyst, worldcat and of course cat! |  |  |  |
| ^ | граница начала всей строки (не предложения!) | ^[Hh]ello | Hello, Timur. I haven't seen you for ages. I want to be his favorite hello and his hardest goodbye. |  |  |  |
| $ | граница конца  всей строки (не предложения!) | [Gg]oodbye$ | Goodbye my dear friend, goodbye! I miss you so much. Goodbye |  |  |  |
| (...) | группировка выражения (подвыражение) | (SELECT){2} | SELECTSELECT |  |  |  |
| (...)\n | поиск одинаковых подвыражений (ссылка назад). \1 - первое подвыражение, \2 — второе, \3 — третье и так далее.<br>ссылки назад не видят \b | (\b\w+\b)[ ]+\1 | Это текст, в котором котором несколько слов    слов повторяются, хотя они и не   не должны повторяться повторяться. |  |  |  |
|  |  | <[Hh]([1-6])>.*</[Hh]\1> | <H2>ООП</H2><br><h2>Алгоритмы и структуры данных</h3> |  |  |  |
| (?P<name><regex>) | задание имени подвыражения | (?P<word>\b\w+\b)[ ]+(?P=word) | Это текст, в котором котором несколько слов    слов повторяются, хотя они и не   не должны повторяться повторяться. |  |  |  |
| (?P=name) | использование имени подвыражения |  |  |  |  |  |
| Функция | Что делает | Пример | Результат |  |  |  |
| REGEXP_LIKE(1, 2, 3)<br><br>1 - исходная строка,<br>2 - шаблон рег. выражения,<br>3 - флаги сопоставления. | показывает есть ли совпадение с шаблоном регулярного выражения в строке. | REGEXP_LIKE('beegeek', 'bee', 'c') AS result1,<br>REGEXP_LIKE('beegeek', 'Bee', 'c') AS result2,<br>REGEXP_LIKE('beegeek', 'Bee') AS result3,<br>REGEXP_LIKE('Foo\nbar.', '^[A-Z].*\\.$', 'mc') AS result4 | +---------+----------+----------+----------+<br>\| result1 \| result2 \| result3 \| result4 \|<br>+---------+----------+----------+----------+<br>\| 1          \| 0         \| 1           \| 0           \|<br>+---------+----------+----------+----------+ |  |  |  |
| REGEXP<br>RLIKE | показывает есть ли совпадение с шаблоном регулярного выражения в строке | beegeek' REGEXP 'bee' AS result1,<br>'beegeek' RLIKE 'bee' AS result2,<br>'beegeek' NOT REGEXP 'bee' AS result3,<br>'beegeek' NOT RLIKE 'bee' AS result4; | +---------+----------+----------+----------+<br>\| result1 \| result2 \| result3 \| result4 \|<br>+---------+----------+----------+----------+<br>\| 1          \| 1          \| 0          \| 0          \|<br>+---------+----------+----------+----------+ |  |  |  |
| REGEXP_INSTR(1, 2, 3, 4, 5, 6)<br><br>1 - исходная строка,<br>2 - шаблон рег. выражения,<br>3 - позиция начала поиска,<br>4 - номер совпадения,<br>5 - тип возвращаемой позиции,<br>6 - флаги сопоставления. | определения местоположения подстроки, которая совпадает с шаблоном регулярного выражения. | REGEXP_INSTR('beegeek', 'e{2}', 1, 1, 1) AS result1, <br>REGEXP_INSTR('beegeek', 'e{2}', 1, 2, 1) AS result2;   | +---------+---------+<br>\| result1 \| result2 \|<br>+---------+---------+<br>\| 4         \| 7          \|<br>+---------+---------+ |  |  |  |
| REGEXP_SUBSTR(1, 2, 3, 4, 5)<br><br>1 -  исходная строка,<br>2 - шаблон рег. выражения,<br>3 - позиция начала поиска,<br>4 - номер совпадения,<br>5 - флаги сопоставления. | извлечения подстроки, которая совпадает с шаблоном регулярного выражения | REGEXP_SUBSTR('beegeeek', 'e{2,}', 1, 1) AS result1,  <br>REGEXP_SUBSTR('beegeeek', 'e{2,}', 1, 2) AS result2,  <br>REGEXP_SUBSTR('beegeeek', 'e{2,}', 1, 3) AS result3; | +---------+---------+---------+<br>\| result1 \| result2 \| result3 \|<br>+---------+---------+---------+<br>\| ee       \| eee      \| NULL    \|<br>+---------+---------+---------+ |  |  |  |
| REGEXP_REPLACE(1, 2, 3, 4, 5, 6)<br><br>1 - исходная строка,<br>2 - шаблон рег. выражения,<br>3 - строка замены,<br>4 - позиция начала поиска,<br>5 - номер совпадения,<br>6 - флаги сопоставления. | замена подстроки, которая совпадает с шаблоном регулярного выражения | REGEXP_REPLACE('beeeeegeeek', 'a', 'i') AS result1,<br>REGEXP_REPLACE('beegeeg', 'eee', 'ee') AS result2; | +-------------------+------------+<br>\| result1            \| result2     \|<br>+-------------------+------------+<br>\| beeeeegeeek \| beegeeg  \|<br>+-------------------+------------+ |  |  |  |
