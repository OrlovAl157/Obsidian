# Текстовые функции

> Источник: лист Excel-конспекта. Перенесено в Markdown для Obsidian.

| Функция | Что делает | Пример | Результат |
| --- | --- | --- | --- |
| Текстовые функции SQL |  |  |  |
| CONCAT | соединяет строки | CONCAT('bee', 'geek') | beegeek |
| CONCAT_WS | соединяет строки с разделителем | CONCAT_WS('_', 'bee', 'geek', 'SQL') | bee_geek_SQL |
| GROUP_CONCAT | складывает (как строки) содержимое одного поля из разных строк | GROUP_CONCAT(name SEPARATOR ', ') | Peter, Gwen, Flash, Mary, Eddie |
| CHAR_LENGTH() | длина строки | CHAR_LENGTH('beegeek') | 7 |
| ORD()  | получить ANSII-код символа | ORD('A') | 65 |
| CHAR() | получить символ по ANSII-коду | CHAR(65)<br>CONVERT(CHAR(65), CHAR) | b'A'<br>A |
| LOWER() | буквы маленькие | LOWER('BeeGeek') | beegeek |
| UPPER() | буквы большие | UPPER('BeeGeek') | BEEGEEK |
| LTRIM() | удаляет пробелы слева | LTRIM('   bee') | bee |
| RTRIM() | удаляет пробелы справа | LTRIM('bee        ') | bee |
| REVERSE() | обратный порядок | REVERSE('beegeek') | keegeeb |
| REPEAT() | повтор символов | REPEAT('bee', 3) | beebeebee |
| LPAD() | увеличение строки слева символами | LPAD('bee', 6, '-+-+') | -+-bee |
|  |  | LPAD('bee', 1, '-') | b |
| RPAD() | увеличение строки справа символами | RPAD('bee', 6, '-+-+') | bee-+- |
| LEFT() | выводит символы слева | LEFT('beegeek', 3) | bee |
| RIGHT() | выводит символы справа | RIGHT('beegeek', 3) | eek |
| LOCATE() | расположение символов в строке | LOCATE('eg', 'beegeek') | 3 |
| REPLACE() | замена символов в строке (важен регистр) | REPLACE('beegEek', 'e', 'i') | biigEik  |
| SUBSTRING() | извлечение строки из подстроки | SUBSTRING('beegeek', 4, 2) | ge |
|  |  | SUBSTRING('beegeek', -4) | geek |
| SUBSTRING_INDEX() | извлечение строки из подстроки делителем,<br>отрицательный - с конца (важен регистр) | SUBSTRING_INDEX('g-e-e-k', '-', 2) | g-e |
|  |  | SUBSTRING_INDEX('g-e-e-k', '-', -2) | e-k |
| TRIM() | удаление вхождений в строке:<br> LEADING — из начала строки<br> TRAILING  — из конца строки<br> BOTH — из начала и конца строки  | TRIM(LEADING '-' FROM '--bee--')<br>TRIM(TRAILING '-' FROM '--bee--')<br>TRIM(BOTH '-' FROM '--bee--'); | bee--<br>--bee<br>bee |
