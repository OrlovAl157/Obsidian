# Переменные

> Источник: лист Excel-конспекта. Перенесено в Markdown для Obsidian.

## Пользовательские переменные

```sql
SET @variable := 1;
    
SELECT @variable;
```

```sql
SET @variable1 := 1,
        @variable2 := 2;

SELECT @variable1, @variable2;
```

```sql
SET @min_price := (SELECT MIN(price)
                                 FROM Books);
    
SELECT @min_price;
```

```sql
SELECT MIN(price) INTO @min_price
FROM Books;

SELECT @min_price;
```

```sql
SELECT MIN(price), MAX(price) INTO @min_price, @max_price
FROM Books;

SELECT @min_price, @max_price;
```
