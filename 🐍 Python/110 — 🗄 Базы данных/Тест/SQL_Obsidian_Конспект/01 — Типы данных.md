# Типы данных

> Источник: лист Excel-конспекта. Перенесено в Markdown для Obsidian.

| Тип данных | Размер в байтах | Количество символов |
| --- | --- | --- |
| Строковые типы данных |  |  |
| CHAR() |  | [0; 255] |
| VARCHAR() |  | [0; 65535] |
| TINYTEXT | 255 | [0; 255] |
| TEXT | 65535 | 2^16 - 1 |
| MEDIUMTEXT | 16777215 | 2^24 - 1 |
| LONGTEXT | 4294967295 | 2^32 - 1 |
| Числовые типы данных |  |  |
|  Целые числа |  |  |
| TINYINT | 1 | [-128; 127] или [0; 255] |
| SMALLINT | 2 | [-2^15; 2^15 - 1] или [0; 2^16 - 1] |
| MEDIUMINT | 3 | [-2^23; 2^23 - 1] или [0; 2^24 - 1] |
| INT | 4 | [-2^31; 2^31 - 1] или [0; 2^32 - 1] |
| BIGINT | 8 | [-2^63; 2^63 - 1] или [0; 2^64 - 1] |
|  Числа с плавающей точкой |  |  |
| FLOAT | 4 | [1.17 * 10^-38; 3.4 * 10^38] |
| DOUBLE | 8 | [2.2 * 10^-308; 1.7 * 10^308] |
|  Числа с фиксированной точкой |  |  |
| DECIMAL(M, D) | M – общее количество цифр, D – количество цифр, используемое для дробной части. Максимум 65 цифр. |  |
| NUMERIC(M, D) |  |  |
| Типы данных даты и времени |  |  |
| DATE | 3 | 1000-01-01 – 9999-12-31 |
| TIME | 3 | 00:00:00 – 23:59:59 |
| DATETIME | 8 | 1000-01-01 00:00:00 – 9999-12-31 23:59:59 |
| TIMESTAMP | 4 | 1970-01-01 00:00:00 – 2038-01-19 03:14:07 |
| YEAR | 1 | 1901 – 2155 |
