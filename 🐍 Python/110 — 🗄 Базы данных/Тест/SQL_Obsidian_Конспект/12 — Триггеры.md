## Создание триггера

| Колонка 1 | Колонка 2 | Колонка 3 |
| --- | --- | --- |
| DELIMITER // | Устанавливаем новый разделитель (//) |  |

| Колонка 1 | Колонка 2 | Колонка 3 |
| --- | --- | --- |
| CREATE TRIGGER <имя триггера> | Создаём триггер |  |

| Колонка 1 | Колонка 2 | Колонка 3 |
| --- | --- | --- |
| <время срабатывания триггера> <операция, которая вызывает триггер> | Выбираем до/после чего (столбец справа) ⭢ | BEFORE INSERT — имеет доступ к значениям добавляемой записи; может изменять значения добавляемой записи |

| Колонка 1 | Колонка 2 | Колонка 3 |
| --- | --- | --- |
| ON <имя таблицы> | Имя изменённой таблицы | AFTER INSERT — имеет доступ к значениям добавленной записи |

| Колонка 1 | Колонка 2 | Колонка 3 |
| --- | --- | --- |
| FOR EACH ROW | Для каждой строки | BEFORE UPDATE — имеет доступ к старым и новым значениям обновляемой записи; может изменять новые значения обновляемой записи |

| Колонка 1 | Колонка 2 | Колонка 3 |

| --- | --- | --- |
| <тело триггера>; | Производим операцию | BEFORE DELETE — имеет доступ к значениям удаляемой записи |

| --- | --- | --- |
| END // | Конец | AFTER DELETE — имеет доступ к значениям удаленной записи |

| DELIMITER ; | Возвращаем старый разделитель (;) |  |

## Упрощённый вариант создания триггера

```sql
CREATE TRIGGER <имя триггера>
<время срабатывания триггера> <операция, которая вызывает триггер>
ON <имя таблицы>
FOR EACH ROW
<тело триггера>
```

## Удаление триггера

```sql
DROP TRIGGER <имя триггера>;
```
