# Функции даты и времени

> Источник: лист Excel-конспекта. Перенесено в Markdown для Obsidian.

| Функция | Что делает | Пример | Результат | Единицы измерений | Колонка 6 |
| --- | --- | --- | --- | --- | --- |
| Функции даты и времени |  |  |  |  |  |
| INTERVAL | изменение интервала времени | 2023-01-01' + INTERVAL '10-2' YEAR_MONTH | 2033-03-01 | микросекунда  | MICROSECOND |
| NOW() | текущее дата и время сервера | NOW() | 2023-06-27 10:14:48 | секунда  | SECOND |
| CURDATE() | текущее дата сервера | CURDATE() | 2023-06-27 | минута  | MINUTE |
| CURTIME() | текущее время сервера | CURTIME() | 10:23:15 | час  | HOUR |
| UTC_TIMESTAMP() | текущее дата и время UTC | UTC_TIMESTAMP() | 2023-06-28 05:44:53 | день  | DAY |
| UTC_DATE() | текущее дата UTC | UTC_DATE() | 2023-10-17 | неделя  | WEEK |
| UTC_TIME() | текущее время UTC | UTC_TIME() | 13:57:34 | месяц  | MONTH |
| MAKEDATE() | создаёт дату (год, номер дня в году) | MAKEDATE(2023, 365) | 2023-12-31 | квартал  | QUARTER |
| MAKETIME() | создание времени (час, мин., сек.) | MAKETIME(21, 11, 42) | 21:11:42 | год  | YEAR |
| TIME_TO_SEC() | преобразование времени в секунды | TIME_TO_SEC('01:00:00')  | 3600 | секунды.микросекунды | SECOND_MICROSECOND  |
| SEC_TO_TIME() | преобразование секунд во время | SEC_TO_TIME(3600)  | 01:00:00 | минуты:секунды.микросекунды | MINUTE_MICROSECOND  |
| TO_DAYS() | количество дней от даты до нулевого года | TO_DAYS('2023-04-01')  | 3923-03-31 | минуты:секунды | MINUTE_SECOND  |
| TO_SECONDS() | количество секунда от даты до нулевого года | TO_SECONDS('2023-04-01')  | 63847526400 | часы:минуты:секунды.микросекунды | HOUR_MICROSECOND  |
| LAST_DAY() | замена дня на последний день месяца | LAST_DAY('2023-02-14')  | 2023-02-28 | часы:минуты:секунды | HOUR_SECOND  |
| DATE() | извлекает дату из штампа | DATE('2023-12-31 10:00:20') | 2023-12-31 | часы:минуты | HOUR_MINUTE  |
| TIME() | извлекает время из штампа | TIME('2023-12-31 10:00:20') | 10:00:20 | дни часы:минуты:секунды.микросекунды | DAY_MICROSECOND  |
| YEAR() | извлекает год из штампа | YEAR('2027-12-31') | 2027 | дни часы:минуты:секунды | DAY_SECOND  |
| MONTH() | извлекает месяц из штампа | MONTH('2023-10-20') | 10 | дни часы:минуты | DAY_MINUTE  |
| DAY() | извлекает день из штампа | DAY('2023-10-20') | 20 | дни часы | DAY_HOUR  |
| HOUR() | извлекает часы из штампа | HOUR('18:20:00') | 18 | годы-месяцы | YEAR_MONTH  |
| MINUTE() | извлекает минуты из штампа | MINUTE('12:30:10.1415') | 30 |  |  |
| SECOND() | извлекает секунды из штампа | SECOND('12:30:10.1415') | 10 | Спецификатор  | Описание |
| MICROSECOND() | извлекает микросекунды из штампа | MICROSECOND('12:30:10.1415') | 141500 | %M  | Полное название месяца (January..December) |
| DAYOFYEAR() | вычисление номера дня по дате | DAYOFYEAR('2023-12-31') | 365 | %b  | Сокращенное название месяца (Jan..Dec) |
| WEEK() | вычисление номера недели по дате | WEEK('2023-12-31') | 53 | %W  | Полное название дня недели (Sunday..Saturday) |
| WEEKDAY() | вычисление номера дня недели по дате (от 0 до 6) | WEEKDAY('2023-10-16') - - понедельник | 0 | %a  | Сокращенное название дня недели (Sun..Sat) |
| QUARTER() | вычисление квартала по дате | QUARTER('2023-10-01') | 4 | %Y  | Год (0000..9999) |
| DAYNAME() | вычисление название дня недели по дате | DAYNAME('2023-10-16') | Monday | %m  | Номер месяца (00..12) |
| MONTHNAME() | вычисление название месяца по дате | MONTHNAME('2023-07-01') | July | %d  | День месяца (00..31) |
| lc_time_names | локализация названия дня недели или месяца | SET lc_time_names = 'ru_RU';<br>SELECT MONTHNAME('2023-10-20')  | Октября | %w  | Номер дня недели (0..6) |
| ADDDATE() | прибавляет интервал даты или времени к дате | ADDDATE('2023-10-20 07:30:00', INTERVAL 4 HOUR) | 2023-10-20 03:30:00 | %H  | Часы (00..23) |
| SUBDATE() | вычитает интервал даты или времени от даты | SUBDATE('2023-10-20', INTERVAL 4 DAY) | 2023-10-16 | %i  | Минуты (00..59) |
| ADDTIME() | прибавляет время к дате | ADDTIME('2023-10-20 21:00:00', '06:00:00') | 2023-10-21 03:00:00 | %S  | Секунды (00..59) |
| SUBTIME() | вычитает время от даты | SUBTIME('20:00:00', '30:00:00') | -10:00:00 | %f  | Микросекунды (000000..999999) |
| DATEDIFF() | вычисление разницы между датами | DATEDIFF('2023-10-20 07:30:00', '2023-10-20 8:00:00') | 0 | %T  | Время в формате HH:MM:SS |
| TIMEDIFF() | вычисление разницы между временем | TIMEDIFF('2023-10-25 9:00:00', '2023-10-20 07:00:00') | 122:00:00 |  |  |
| TIMESTAMPDIFF() | вычисление разницы по дате/времени по штампу | TIMESTAMPDIFF(HOUR, '2023-10-20 07:00:00', '2023-10-20 12:00:00') | -5 |  |  |
| TIMESTAMP() | объединение даты и времени | TIMESTAMP('2023-10-20', '28:00') | 2023-10-21 04:00:00 |  |  |
| DATE_FORMAT() | форматирование даты и времени | DATE_FORMAT('2023-10-20 07:30:00', '%H:%i, %d.%m.%Y') | 07:30, 20.10.2023 |  |  |
| TIME_FORMAT() | форматирование времени | TIME_FORMAT('12:30:00', 'Time: %H::%i') | Time: 12::30 |  |  |
| STR_TO_DATE() | обратное форматирование даты и времени | STR_TO_DATE('07:30, 20.10.2023', '%H:%i, %d.%m.%Y') | 2023-10-20 07:30:00  |  |  |
| GET_FORMAT() | получение форматирование даты и времени по региону (USA, JIS, ISO, EUR) | GET_FORMAT(DATE, 'EUR') | %d.%m.%Y |  |  |
