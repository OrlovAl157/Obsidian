# 📄 Работа с текстовыми файлами

## 📌 Коротко

> Чтение и запись текстовых файлов с использованием `open()`, контекстного менеджера `with`, различных режимов доступа и методов обработки больших файлов.

---

## 💡 Полезные советы

- **Всегда используйте `with`** — автоматически закрывает файл даже при ошибках
- **Явно указывайте кодировку** — `encoding='utf-8'` для работы с кириллицей
- **Режимы**: `'r'` (чтение), `'w'` (запись, стирает!), `'a'` (добавление), `'x'` (создание нового)
- **Для больших файлов** — используйте построчное чтение, не `read()` целиком
- **`\n` vs `\r\n`** — Windows использует `\r\n`, Unix/Mac - `\n`
- **Проверяйте существование** — используйте `pathlib.Path.exists()` перед чтением
- **Обрабатывайте ошибки** — `FileNotFoundError`, `PermissionError`, `UnicodeDecodeError`

---

## 🚀 Быстрая справка

| Задача | Метод | Сложность | Пример |
|---|---|---|---|
| Открыть файл | `open(file, mode)` | O(1) | `open('data.txt', 'r')` |
| Прочитать всё | `.read()` | O(n) | `f.read()` |
| Прочитать строки | `.readlines()` | O(n) | `f.readlines()` |
| Прочитать строку | `.readline()` | O(m) | `f.readline()` |
| Записать текст | `.write()` | O(n) | `f.write('Hello')` |
| Записать строки | `.writelines()` | O(n) | `f.writelines(lines)` |
| Перебор строк | `for line in f:` | O(n) | `for line in f: ...` |
| Позиция в файле | `.tell()` | O(1) | `f.tell()` |
| Переместить курсор | `.seek(offset)` | O(1) | `f.seek(0)` |
| Закрыть файл | `.close()` | O(1) | `f.close()` |

## 📖 Теория

### Что такое файл?

Файл — это последовательность данных, хранящихся на диске. В Python мы работаем с файлами через объект файла, который предоставляет методы для чтения и записи.

### Режимы открытия файлов

```
ТЕКСТОВЫЕ РЕЖИМЫ:
'r'   - Чтение (файл должен существовать)
'w'   - Запись (СТИРАЕТ существующий файл!)
'a'   - Добавление в конец (не стирает)
'x'   - Создание нового (ошибка если существует)
'r+'  - Чтение и запись (не стирает)
'w+'  - Запись и чтение (СТИРАЕТ при открытии!)
'a+'  - Добавление и чтение (не стирает)

БИНАРНЫЕ РЕЖИМЫ (добавьте 'b'):
'rb'  - Чтение в бинарном режиме
'wb'  - Запись в бинарном режиме
'ab'  - Добавление в бинарном режиме
```

### Почему использовать `with`?

```python
# ❌ БЕЗ with — файл может остаться открытым при ошибке
f = open('file.txt')
data = f.read()
f.close()  # Может не выполниться!

# ✅ С with — гарантированное закрытие
with open('file.txt') as f:
    data = f.read()
# Файл закроется автоматически, даже при ошибке
```

## 💻 Чтение файлов

### Способ 1: Прочитать весь файл

```python
with open('file.txt', 'r', encoding='utf-8') as f:
    content = f.read()
    print(content)

# Для маленьких файлов (< 10 MB) — OK
# Для больших файлов — плохо (весь файл в памяти)
```

### Способ 2: Прочитать все строки

```python
with open('file.txt', 'r', encoding='utf-8') as f:
    lines = f.readlines()
    # lines = ['Hello\n', 'World\n', ...]

# Убрать \n:
lines = [line.rstrip('\n') for line in lines]
```

### Способ 3: Построчное чтение (РЕКОМЕНДУЕТСЯ!)

```python
# Экономит память для больших файлов
with open('file.txt', 'r', encoding='utf-8') as f:
    for line in f:
        print(line.strip())  # strip() убирает \n
        # Обрабатываем по одной строке за раз

# Это самый эффективный способ!
```

### Способ 4: Читать по одной строке

```python
with open('file.txt', 'r', encoding='utf-8') as f:
    line1 = f.readline()  # → "Hello\n"
    line2 = f.readline()  # → "World\n"
    line3 = f.readline()  # → "" (конец файла)
```

### Обработка ошибок при чтении

```python
try:
    with open('file.txt', 'r', encoding='utf-8') as f:
        content = f.read()
except FileNotFoundError:
    print("❌ Файл не найден")
except PermissionError:
    print("❌ Нет доступа к файлу")
except UnicodeDecodeError:
    print("❌ Ошибка кодировки (попробуйте другую)")
```

## ✍️ Запись в файлы

### Способ 1: Простая запись

```python
# ПЕРЕЗАПИШЕТ файл (стирает старое содержимое!)
with open('output.txt', 'w', encoding='utf-8') as f:
    f.write('Hello World\n')
    f.write('Python is awesome\n')
```

### Способ 2: Запись списка строк

```python
lines = ['Line 1\n', 'Line 2\n', 'Line 3\n']

with open('output.txt', 'w', encoding='utf-8') as f:
    f.writelines(lines)  # Пишет все сразу
```

### Способ 3: Запись через print()

```python
# print() удобнее: автоматически добавляет \n
with open('output.txt', 'w', encoding='utf-8') as f:
    print('Hello World', file=f)
    print('Python', 'is', 'awesome', file=f)
```

### Способ 4: Добавление в файл (не стирает!)

```python
# Режим 'a' — добавляет в конец
with open('log.txt', 'a', encoding='utf-8') as f:
    f.write('New log entry\n')

# С временной меткой
from datetime import datetime

with open('log.txt', 'a', encoding='utf-8') as f:
    timestamp = datetime.now().strftime('%Y-%m-%d %H:%M:%S')
    f.write(f"[{timestamp}] Event occurred\n")
```

## 🔄 Работа с позицией в файле

```python
with open('file.txt', 'r', encoding='utf-8') as f:
    # tell() — текущая позиция в байтах
    print(f.tell())  # → 0 (начало)
    
    first_line = f.readline()
    print(f.tell())  # → позиция после первой строки
    
    # seek() — переместить курсор
    f.seek(0)       # В начало файла
    start = f.read(5)  # Первые 5 символов
    
    # seek с режимами:
    # 0 — от начала файла (по умолчанию)
    # 1 — от текущей позиции
    # 2 — от конца файла
    f.seek(-10, 2)  # На 10 символов от конца
```

## 📂 Работа с путями (pathlib)

```python
from pathlib import Path

# Создание пути
file_path = Path('data.txt')
nested = Path('documents') / 'reports' / '2024.txt'

# Проверка существования
if file_path.exists():
    print("✅ Файл существует")

# Проверка типа
if file_path.is_file():
    print("Это файл")

# Информация о файле
print(file_path.stat().st_size)  # Размер в байтах
print(file_path.stem)             # Имя без расширения
print(file_path.suffix)           # Расширение (.txt)

# Чтение и запись через pathlib
file_path.write_text('Hello', encoding='utf-8')
content = file_path.read_text(encoding='utf-8')
```

## 💻 Практические примеры

### Подсчёт строк, слов, символов

```python
def file_stats(filename):
    """Статистика файла"""
    lines = words = chars = 0
    
    with open(filename, 'r', encoding='utf-8') as f:
        for line in f:
            lines += 1
            words += len(line.split())
            chars += len(line)
    
    return {'lines': lines, 'words': words, 'chars': chars}

stats = file_stats('document.txt')
print(f"Строк: {stats['lines']}, Слов: {stats['words']}")
```

### Фильтрация строк

```python
def filter_lines(input_file, output_file, condition):
    """Отфильтровать строки по условию"""
    with open(input_file, 'r', encoding='utf-8') as infile:
        with open(output_file, 'w', encoding='utf-8') as outfile:
            for line in infile:
                if condition(line):
                    outfile.write(line)

# Пример: оставить только строки длиннее 50 символов
filter_lines('input.txt', 'output.txt', lambda line: len(line) > 50)
```

### Удаление дубликатов

```python
def remove_duplicates(input_file, output_file):
    """Удалить дубликаты строк"""
    seen = set()
    with open(input_file, 'r', encoding='utf-8') as infile:
        with open(output_file, 'w', encoding='utf-8') as outfile:
            for line in infile:
                clean = line.strip()
                if clean not in seen:
                    outfile.write(line)
                    seen.add(clean)

remove_duplicates('data.txt', 'unique.txt')
```

### Объединение файлов

```python
def merge_files(input_files, output_file):
    """Объединить несколько файлов"""
    with open(output_file, 'w', encoding='utf-8') as outfile:
        for filename in input_files:
            with open(filename, 'r', encoding='utf-8') as infile:
                outfile.write(infile.read())
                outfile.write('\n')  # Разделитель

merge_files(['file1.txt', 'file2.txt', 'file3.txt'], 'merged.txt')
```

## 🔍 Работа с большими файлами

```python
# ❌ ПЛОХО: весь файл в памяти
with open('huge_file.txt', 'r') as f:
    content = f.read()  # Может занять гигабайты!

# ✅ ХОРОШО: построчная обработка
with open('huge_file.txt', 'r', encoding='utf-8') as f:
    for line in f:
        process(line)

# Быстрый подсчёт строк
def count_lines(filename):
    with open(filename, 'r', encoding='utf-8') as f:
        return sum(1 for _ in f)
```

## ⚠️ Частые ошибки

```python
# ❌ Ошибка 1: Забыли with (файл не закроется при ошибке)
f = open('file.txt')
content = f.read()
f.close()  # Может не выполниться!

# ✅ ВСЕГДА используйте with
with open('file.txt', 'r', encoding='utf-8') as f:
    content = f.read()

# ❌ Ошибка 2: Забыли encoding (проблемы с кириллицей)
with open('russian.txt', 'r') as f:  # Может быть ошибка!
    content = f.read()

# ✅ Всегда указывайте encoding
with open('russian.txt', 'r', encoding='utf-8') as f:
    content = f.read()

# ❌ Ошибка 3: Перепутали 'w' и 'a'
with open('log.txt', 'w') as f:  # СТИРАЕТ весь файл!
    f.write('Entry\n')

# ✅ Используйте 'a' для добавления
with open('log.txt', 'a') as f:  # Добавляет в конец
    f.write('Entry\n')

# ❌ Ошибка 4: Забыли \n при записи
with open('list.txt', 'w') as f:
    f.write('Item 1')
    f.write('Item 2')  # → "Item 1Item 2" без переноса!

# ✅ Добавляйте \n явно
with open('list.txt', 'w') as f:
    f.write('Item 1\n')
    f.write('Item 2\n')

# ❌ Ошибка 5: Использовали бинарный режим без 'b'
with open('image.png', 'r') as f:  # ❌ Ошибка!
    data = f.read()

# ✅ Используйте 'rb' для бинарных файлов
with open('image.png', 'rb') as f:
    data = f.read()
```

## 🎯 Когда использовать что

**`.read()` — когда:**
- Файл маленький (< 10 MB)
- Нужна вся информация сразу
- Производите замены всего текста

**Построчное чтение — когда:**
- Файл большой (> 10 MB)
- Обрабатываете построчно
- Важна экономия памяти

**pathlib — когда:**
- Работаете с путями
- Нужна кроссплатформенность
- Проверяете существование файлов
