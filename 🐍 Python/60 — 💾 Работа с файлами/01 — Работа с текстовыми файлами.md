# 📄 Работа с текстовыми файлами

## 📌 Коротко

> Чтение и запись текстовых файлов с использованием `open()`, контекстного менеджера `with`, различных режимов доступа и методов обработки больших файлов.

---

## 💡 Полезные советы

- **Всегда используйте `with`** — автоматически закрывает файл даже при ошибках
- **Явно указывайте кодировку** — `encoding='utf-8'` для работы с кириллицей
- **Режимы**: `'r'` (чтение), `'w'` (запись, стирает!), `'a'` (добавление), `'x'` (создание нового)
- **Для больших файлов** — используйте построчное чтение, не `read()` целиком
- **`\n` vs `\r\n`** — Windows использует `\r\n`, Unix/Mac - `\n`
- **Проверяйте существование** — используйте `pathlib.Path.exists()` перед чтением
- **Обрабатывайте ошибки** — `FileNotFoundError`, `PermissionError`, `UnicodeDecodeError`

---

## 📖 Теория

### Что такое файл?

Файл — это последовательность данных, хранящихся на диске. В Python мы работаем с файлами через объект файла, который предоставляет методы для чтения и записи.

## 🔄 Режимы открытия файлов

```python
# ========== ТЕКСТОВЫЕ РЕЖИМЫ ==========

# Только чтение
'r'     # Чтение (файл должен существовать, ошибка если нет)
        # Курсор в начале файла

# Только запись
'w'     # Запись (создаёт новый файл или ПОЛНОСТЬЮ перезаписывает существующий!)
        # Курсор в начале файла
        # ОСТОРОЖНО: стирает всё содержимое!

# Добавление
'a'     # Добавление в конец (создаёт файл если не существует)
        # Курсор в конце файла
        # НЕ стирает существующее содержимое

# Эксклюзивное создание
'x'     # Создание нового файла (ошибка если уже существует)
        # Защита от случайной перезаписи

# ========== КОМБИНИРОВАННЫЕ РЕЖИМЫ ==========

# Чтение + Запись
'r+'    # Чтение и запись (файл должен существовать)
        # Курсор в начале файла
        # НЕ стирает содержимое
        # Можно читать и писать в любом месте

'w+'    # Чтение и запись (создаёт новый или СТИРАЕТ существующий!)
        # Курсор в начале файла
        # ОСТОРОЖНО: стирает содержимое при открытии!

'a+'    # Чтение и добавление
        # Курсор в конце для записи
        # НЕ стирает содержимое
        # Можно читать с любого места, но запись только в конец

# ========== БИНАРНЫЕ РЕЖИМЫ ==========
# Для работы с не-текстовыми файлами (изображения, видео, архивы)

'rb'    # Чтение в бинарном режиме
'wb'    # Запись в бинарном режиме
'ab'    # Добавление в бинарном режиме
'rb+'   # Чтение и запись в бинарном режиме
'wb+'   # Запись и чтение в бинарном режиме
'ab+'   # Добавление и чтение в бинарном режиме

# ========== ПРИМЕРЫ ==========

# Только чтение
with open('file.txt', 'r') as f:
    content = f.read()

# Только запись (ОСТОРОЖНО: стирает файл!)
with open('file.txt', 'w') as f:
    f.write('New content')

# Добавление в конец
with open('file.txt', 'a') as f:
    f.write('Added line\n')

# Создание нового файла (защита от перезаписи)
try:
    with open('new.txt', 'x') as f:
        f.write('Content')
except FileExistsError:
    print("Файл уже существует")

# Чтение и запись (без стирания)
with open('file.txt', 'r+') as f:
    content = f.read()          # Прочитать
    f.write('\nNew line')       # Дописать

# ========== СРАВНЕНИЕ РЕЖИМОВ ==========

# 'w' vs 'a' vs 'r+'
# 'w'  - СТИРАЕТ файл при открытии
# 'a'  - НЕ стирает, добавляет в конец
# 'r+' - НЕ стирает, можно писать в любое место

# Пример разницы:
# Исходный файл содержит: "Hello"

# С 'w':
with open('test.txt', 'w') as f:
    f.write('World')
# Результат: "World" (Hello стёрто!)

# С 'a':
with open('test.txt', 'a') as f:
    f.write('World')
# Результат: "HelloWorld" (добавлено в конец)

# С 'r+':
with open('test.txt', 'r+') as f:
    f.write('World')
# Результат: "Worldo" (перезаписано с начала, остальное осталось)
```

### Аргумент `newline` — переносы строк

Аргумент `newline` контролирует работу режима новой строки. Работает только с текстовым режимом. Допустимые аргументы: `None`, `''`, `'\n'`, `'\r'` и `'\r\n'`.

- если аргумент `newline=None` или не указан, то символами окончания строк `'\n'`, `'\r'` или `'\r\n'` на выходе преобразуются в разделитель строки `'\n'`
```python
open('file.txt', 'r', encoding='utf-8', newline=None)
# '\r\n' → '\n'
# '\r'   → '\n'
# '\n'   → '\n'
```

- если `newline=''`, то получим результат с вырезанными символами окончания строк `'\n'`, `'\r'` или `'\r\n'`, то есть файл прочитается как одна большая строка
```python
open('file.txt', 'r', encoding='utf-8', newline='')
# '\r\n' → '\r\n'  (без изменений)
# '\r'   → '\r'    (без изменений)
# '\n'   → '\n'    (без изменений)
```

- если `newline='\n'`, то в результате чтения функция `open()` уберет только символ `\n` из результата, если `newline='\r'`, то только `\r`, если `newline='\r\n'`, то только сочетание `'\r\n'` не будет учитываться
```python
open('file.txt', 'r', encoding='utf-8', newline='\r\n')
# '\r\n' → обрабатывается
# '\r'   → остаётся как есть
# '\n'   → остаётся как есть
```

### Почему использовать `with`?

```python
# ❌ БЕЗ with — файл может остаться открытым при ошибке
f = open('file.txt')
data = f.read()
f.close()  # Может не выполниться!

# ✅ С with — гарантированное закрытие
with open('file.txt') as f:
    data = f.read()
# Файл закроется автоматически, даже при ошибке
```

## 💻 Чтение файлов

### Способ 1: Прочитать весь файл

```python
with open('file.txt', 'r', encoding='utf-8') as f:
    content = f.read()
    print(content)

# Для маленьких файлов (< 10 MB) — OK
# Для больших файлов — плохо (весь файл в памяти)
```

### Способ 2: Прочитать все строки

```python
with open('file.txt', 'r', encoding='utf-8') as f:
    lines = f.readlines()
    # lines = ['Hello\n', 'World\n', ...]

# Убрать \n:
lines = [line.rstrip('\n') for line in lines]
```

### Способ 3: Построчное чтение (РЕКОМЕНДУЕТСЯ!)

```python
# Экономит память для больших файлов
with open('file.txt', 'r', encoding='utf-8') as f:
    for line in f:
        print(line.strip())  # strip() убирает \n
        # Обрабатываем по одной строке за раз

# Это самый эффективный способ!
```

### Способ 4: Читать по одной строке

```python
with open('file.txt', 'r', encoding='utf-8') as f:
    line1 = f.readline()  # → "Hello\n"
    line2 = f.readline()  # → "World\n"
    line3 = f.readline()  # → "" (конец файла)
```

### Обработка ошибок при чтении

```python
try:
    with open('file.txt', 'r', encoding='utf-8') as f:
        content = f.read()
except FileNotFoundError:
    print("❌ Файл не найден")
except PermissionError:
    print("❌ Нет доступа к файлу")
except UnicodeDecodeError:
    print("❌ Ошибка кодировки (попробуйте другую)")
```

## ✍️ Запись в файлы

### Способ 1: Простая запись

```python
# ПЕРЕЗАПИШЕТ файл (стирает старое содержимое!)
with open('output.txt', 'w', encoding='utf-8') as f:
    f.write('Hello World\n')
    f.write('Python is awesome\n')
```

### Способ 2: Запись списка строк

```python
lines = ['Line 1\n', 'Line 2\n', 'Line 3\n']

with open('output.txt', 'w', encoding='utf-8') as f:
    f.writelines(lines)  # Пишет все сразу
```

### Способ 3: Запись через print()

```python
# print() удобнее: автоматически добавляет \n
with open('output.txt', 'w', encoding='utf-8') as f:
    print('Hello World', file=f)
    print('Python', 'is', 'awesome', file=f)
```

### Способ 4: Добавление в файл (не стирает!)

```python
# Режим 'a' — добавляет в конец
with open('log.txt', 'a', encoding='utf-8') as f:

# С временной меткой
from datetime import datetime

with open('log.txt', 'a', encoding='utf-8') as f:
```

## 🔄 Управление позицией в файле (seek и tell)

### tell() — узнать текущую позицию курсора

```python
with open('file.txt', 'r', encoding='utf-8') as f:
    print(f.tell())             # → 0 (курсор в начале)
    f.read(5)                   # Прочитать 5 символов
    print(f.tell())             # → 5 (курсор на 5-й позиции)
    f.readline()                # Прочитать строку
    print(f.tell())             # → позиция после строки
```

### seek() — переместить курсор в нужную позицию

```python
# seek(offset, whence)
# offset - смещение в байтах
# whence - откуда считать:
#   0 - от начала файла (по умолчанию)
#   1 - от текущей позиции (только в бинарном режиме!)
#   2 - от конца файла (только в бинарном режиме!)

with open('file.txt', 'r') as f:
    f.read()                    # Прочитать весь файл
    f.seek(0)                   # Вернуться в начало
    content = f.read()          # Прочитать снова

with open('file.txt', 'r') as f:
    pos = f.tell()              # Запомнить позицию
    f.read(100)
    f.seek(pos)                 # Вернуться к позиции — всегда ОК
```

### Бинарный режим — полный контроль

```python
with open('file.txt', 'rb') as f:
    f.seek(10, 0)               # От начала файла
    f.seek(10, 1)               # От текущей позиции
    f.seek(-10, 2)              # От конца файла (назад на 10 байт)
    
    file_size = f.seek(0, 2)    # Узнать размер файла
    f.seek(0)                   # Вернуться в начало
```

### Практические примеры

**Чтение последних N строк (без загрузки всего в память):**

```python
def read_last_lines(filename, n=10):
    with open(filename, 'rb') as f:
        f.seek(0, 2)
        file_size = f.tell()
        lines_found = 0
        position = file_size
        while position > 0 and lines_found <= n:
            position -= 1
            f.seek(position)
            if f.read(1) == b'\n':
                lines_found += 1
        if position > 0:
            f.seek(position + 1)
        else:
            f.seek(0)
        return f.read().decode('utf-8').splitlines()[-n:]

last_10_lines = read_last_lines('huge_log.txt', 10)
```

### ⚠️ Важные замечания

```python
# ❌ ОПАСНО: seek() работает с БАЙТАМИ, не символами!
with open('russian.txt', 'r', encoding='utf-8') as f:
    f.seek(1)                   # Может разорвать UTF-8 символ!

# ✅ БЕЗОПАСНО:
with open('file.txt', 'r', encoding='utf-8') as f:
    f.seek(0)                   # Начало файла — всегда ОК
    pos = f.tell()
    f.seek(pos)                 # Сохранённая позиция — всегда ОК

# ❌ В текстовом режиме whence может быть только 0!
with open('file.txt', 'r') as f:
    f.seek(10, 0)               # ✅ OK
    # f.seek(10, 1)             # ❌ OSError!
    # f.seek(-10, 2)            # ❌ OSError!
```

## 💻 Практические примеры

### Подсчёт строк, слов, символов

```python
def file_stats(filename):
    lines = words = chars = 0
    with open(filename, 'r', encoding='utf-8') as f:
        for line in f:
            lines += 1
            words += len(line.split())
            chars += len(line)
    return {'lines': lines, 'words': words, 'chars': chars}
```

### Фильтрация строк

```python
def filter_lines(input_file, output_file, condition):
    with open(input_file, 'r', encoding='utf-8') as infile:
        with open(output_file, 'w', encoding='utf-8') as outfile:
            for line in infile:
                if condition(line):
                    outfile.write(line)

filter_lines('input.txt', 'output.txt', lambda line: len(line) > 50)
```

### Удаление дубликатов

```python
def remove_duplicates(input_file, output_file):
    seen = set()
    with open(input_file, 'r', encoding='utf-8') as infile:
        with open(output_file, 'w', encoding='utf-8') as outfile:
            for line in infile:
                clean = line.strip()
                if clean not in seen:
                    outfile.write(line)
                    seen.add(clean)
```

## ⚠️ Частые ошибки

```python
# ❌ Забыли with
f = open('file.txt')
content = f.read()
f.close()  # Может не выполниться!

# ✅ ВСЕГДА используйте with
with open('file.txt', 'r', encoding='utf-8') as f:
    content = f.read()

# ❌ Забыли encoding
with open('russian.txt', 'r') as f:
    content = f.read()

# ✅ Всегда указывайте encoding
with open('russian.txt', 'r', encoding='utf-8') as f:
    content = f.read()

# ❌ Перепутали 'w' и 'a'
with open('log.txt', 'w') as f:  # СТИРАЕТ весь файл!
    f.write('Entry\n')

# ✅ Используйте 'a' для добавления
with open('log.txt', 'a') as f:
    f.write('Entry\n')

# ❌ Забыли \n при записи
with open('list.txt', 'w') as f:
    f.write('Item 1')
    f.write('Item 2')  # → "Item 1Item 2" без переноса!

# ✅ Добавляйте \n явно
with open('list.txt', 'w') as f:
    f.write('Item 1\n')
    f.write('Item 2\n')

# ❌ Бинарный файл без 'b'
with open('image.png', 'r') as f:  # ❌ Ошибка!
    data = f.read()

# ✅ Используйте 'rb'
with open('image.png', 'rb') as f:
    data = f.read()
```

## 🎯 Когда использовать что

**`.read()` — когда:**
- Файл маленький (< 10 MB)
- Нужна вся информация сразу

**Построчное чтение — когда:**
- Файл большой (> 10 MB)
- Важна экономия памяти

**`pathlib` — когда:**
- Работаете с путями
- Нужна кроссплатформенность
