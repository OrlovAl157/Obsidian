## 🎯 Выбор правильного инструмента

### Чем различаются форматы?

| Формат | Лучше для | Плюсы | Минусы |
|---|---|---|---|
| **Текстовые файлы** | Логи, конфиги, простой текст | Простой, универсальный | Нужна ручная обработка |
| **CSV** | Табличные данные, Excel | Читается человеком, стандарт | Только простые типы |
| **JSON** | Веб-API, конфигурация | Безопасно, универсально | Не все типы Python |
| **Pickle** | Состояние, кэш, Python объекты | Быстро, компактно | ⚠️ ОПАСНО из интернета! |
| **ZIP** | Архивы, резервные копии | Упаковывает много файлов | Нужно распаковывать |

## 🔄 Типичные сценарии

**📥 Читаю данные из файла:**
- Текстовый файл → `open() + read()`
- CSV таблица → `csv.DictReader()`
- JSON → `json.load()`
- Pickle (свой код) → `pickle.load()`

**📤 Сохраняю данные:**
- Простой текст → `open() + write()`
- Таблица → `csv.DictWriter()`
- Для веб → `json.dump()`
- Python объект → `pickle.dump()`

**🌊 Работаю с потоками:**
- Читаю с клавиатуры → `sys.stdin`
- Вывожу результат → `sys.stdout`
- Большой объём → построчное чтение

**📦 Архивирую:**
- Несколько файлов → `ZipFile.write()`
- Распаковка → `ZipFile.extractall()`

## ⚡ Общие правила

```python
# ✅ ВСЕГДА используйте with
with open('file.txt', 'r', encoding='utf-8') as f:
    data = f.read()

# ✅ ВСЕГДА указывайте encoding='utf-8'
with open('file.txt', 'r', encoding='utf-8') as f:
    ...

# ✅ Для больших файлов — построчное чтение
with open('huge.txt') as f:
    for line in f:
        process(line)

# ✅ Для бинарных файлов — режим 'b'
with open('data.pkl', 'rb') as f:
    data = pickle.load(f)

# ❌ НЕ загружайте pickle из интернета!
# pickle может выполнить произвольный код

# ✅ Для веб-данных используйте JSON
data = json.loads(response.text)
```

## 📚 Быстрая справка по всем модулям

```python
# ТЕКСТОВЫЕ ФАЙЛЫ
with open('file.txt', 'r', encoding='utf-8') as f:
    content = f.read()

# CSV
import csv
with open('data.csv') as f:
    reader = csv.DictReader(f)
    for row in reader:
        print(row['name'])

# JSON
import json
with open('data.json') as f:
    data = json.load(f)
json.dump(data, open('out.json', 'w'), indent=2)

# PICKLE
import pickle
with open('data.pkl', 'wb') as f:
    pickle.dump(obj, f)
with open('data.pkl', 'rb') as f:
    obj = pickle.load(f)

# ZIP
from zipfile import ZipFile
with ZipFile('archive.zip') as zf:
    zf.extractall('output')

# STDIN/STDOUT
import sys
for line in sys.stdin:
    print(line.strip())
```

## ❓ Часто задаваемые вопросы

**Q: Какой формат выбрать для сохранения данных?**
A: JSON для обмена с другими системами, Pickle только для своего кода, CSV для таблиц.

**Q: Почему pickle опасен?**
A: Может выполнить произвольный Python код при десериализации. Используйте только свои файлы!

**Q: Как работать с большими файлами?**
A: Построчное чтение вместо `read()` целиком. Используйте `for line in file:` вместо загрузки в память.

**Q: Что означает `encoding='utf-8'`?**
A: Это правильная кодировка для русских текстов и эмодзи. Без неё на Windows будут ошибки.
