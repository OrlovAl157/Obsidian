
|Формат|Значение|Пример|
|---|---|---|
|**День недели**|||
|`%a`|Сокращенное название дня недели|Sun, Mon, …, Sat (en_US)<br>Пн, Вт, …, Вс (ru_RU)|
|`%A`|Полное название дня недели|Sunday, Monday, …, Saturday (en_US)<br>понедельник, …, воскресенье (ru_RU)|
|`%w`|Номер дня недели [0, …, 6]|0, 1, …, 6 (0=воскресенье, 6=суббота)|
|**День**|||
|`%d`|День месяца [01, …, 31]|01, 02, …, 31|
|`%j`|День года [001, 366]|001, 002, …, 366|
|**Месяц**|||
|`%b`|Сокращенное название месяца|Jan, Feb, …, Dec (en_US)<br>янв, фев, …, дек (ru_RU)|
|`%B`|Полное название месяца|January, February, …, December (en_US)<br>Январь, Февраль, …, Декабрь (ru_RU)|
|`%m`|Номер месяца [01, …, 12]|01, 02, …, 12|
|**Год**|||
|`%y`|Год без века [00, …, 99]|00, 01, …, 99|
|`%Y`|Год с веком|0001, 0002, …, 2025, …, 9999|
|**Час**|||
|`%H`|Час (24-часовой формат) [00, …, 23]|00, 01, …, 23|
|`%I`|Час (12-часовой формат) [01, …, 12]|01, 02, …, 12|
|`%p`|До полудня или после|AM, PM (en_US)|
|**Минуты и секунды**|||
|`%M`|Число минут [00, …, 59]|00, 01, …, 59|
|`%S`|Число секунд [00, …, 59]|00, 01, …, 59|
|`%f`|Число микросекунд|000000, 000001, …, 999999|
|**Временная зона**|||
|`%z`|Разница с UTC в формате ±HHMM[SS[.ffffff]]|+0000, -0400, +1030, +063415, …|
|`%Z`|Временная зона|UTC, EST, CST|
|**Неделя**|||
|`%U`|Номер недели в году (неделя с воскресенья) [00, …, 53]|00, 01, …, 53|
|`%W`|Номер недели в году (неделя с понедельника) [00, …, 53]|00, 01, …, 53|
|**Комбинированные**|||
|`%c`|Дата и время|Tue Aug 16 21:30:00 1988 (en_US)<br>03.01.2019 23:18:32 (ru_RU)|
|`%x`|Дата|08/16/88 (None)<br>08/16/1988 (en_US)<br>03.01.2019 (ru_RU)|
|`%X`|Время|21:30:00|
|**Специальные**|||
|`%%`|Символ процента|%|

## Частые шаблоны

|Описание|Формат|Пример вывода|
|---|---|---|
|Российская дата|`%d.%m.%Y`|16.10.2025|
|Российская дата + время|`%d.%m.%Y %H:%M:%S`|16.10.2025 14:30:45|
|Американская дата|`%m/%d/%Y`|10/16/2025|
|ISO дата|`%Y-%m-%d`|2025-10-16|
|ISO дата + время|`%Y-%m-%d %H:%M:%S`|2025-10-16 14:30:45|
|Полная дата (текст)|`%A, %B %d, %Y`|Thursday, October 16, 2025|
|Краткая дата (текст)|`%a, %b %d, %Y`|Thu, Oct 16, 2025|
|Время 24ч|`%H:%M:%S`|14:30:45|
|Время 12ч|`%I:%M:%S %p`|02:30:45 PM|
|С микросекундами|`%Y-%m-%d %H:%M:%S.%f`|2025-10-16 14:30:45.123456|
|RFC 2822|`%a, %d %b %Y %H:%M:%S`|Thu, 16 Oct 2025 14:30:45|

## Использование

python

```python
from datetime import datetime

dt = datetime(2025, 10, 16, 14, 30, 45)

# Форматирование в строку
formatted = dt.strftime('%d.%m.%Y %H:%M:%S')  # '16.10.2025 14:30:45'

# Парсинг из строки
parsed = datetime.strptime('16.10.2025 14:30:45', '%d.%m.%Y %H:%M:%S')
```
