# 🔍 Представление строк: str() и repr()

## 📌 Коротко

> **str()** — красивое представление для пользователей, **repr()** — точное представление для разработчиков. Для datetime они отличаются форматом и назначением.

---

## 💡 Полезные советы

- **str()** → в `print()` и для вывода пользователю
- **repr()** → в отладке, консоли и логах
- **repr() должен быть точным** — идеально если `eval(repr(obj)) == obj`
- **Для datetime используйте strftime()** → если нужен свой формат
- **В логах используйте repr()** → полная информация об объекте

---

## 🚀 Быстрая справка

| Параметр | `str()` | `repr()` |
|---|---|---|
| **Для кого** | 👤 Пользователей | 👨‍💻 Разработчиков |
| **Цель** | 📖 Читаемость | 🎯 Точность |
| **Вызов** | `str(obj)` или `print(obj)` | `repr(obj)` или в консоли |
| **Метод** | `__str__()` | `__repr__()` |

---

## 📖 Теория

### `str()` — Для пользователей

```python
from datetime import datetime

dt = datetime(2025, 10, 16, 14, 30, 45)

# Красивый вывод
print(str(dt))      # 2025-10-16 14:30:45
print(dt)           # то же самое (print использует str())
```

**💡 Характеристики:**
- ✅ Красивый, понятный формат
- ✅ Можно опускать технические детали
- ✅ Для `print()`, `format()`, f-strings
- ❌ Может быть неполный

### `repr()` — Для разработчиков

```python
from datetime import datetime

dt = datetime(2025, 10, 16, 14, 30, 45)

# Точное представление
print(repr(dt))     # datetime.datetime(2025, 10, 16, 14, 30, 45)
dt                  # в консоли выводит то же самое
```

**💡 Характеристики:**
- ✅ Максимально точное представление
- ✅ Содержит всю важную информацию
- ✅ Идеально: `eval(repr(obj)) == obj`
- ✅ Для консоли, отладки, логов

## 💻 Примеры

### Разница для datetime

```python
from datetime import datetime, timezone, timedelta

dt = datetime(2025, 10, 16, 14, 30, 45)

# Простой datetime
print(str(dt))      # 2025-10-16 14:30:45
print(repr(dt))     # datetime.datetime(2025, 10, 16, 14, 30, 45)

# С микросекундами
dt_micro = datetime(2025, 10, 16, 14, 30, 45, 123456)
print(str(dt_micro))    # 2025-10-16 14:30:45.123456
print(repr(dt_micro))   # datetime.datetime(2025, 10, 16, 14, 30, 45, 123456)

# С временной зоной
tz = timezone(timedelta(hours=3))
dt_tz = datetime(2025, 10, 16, 14, 30, 45, tzinfo=tz)
print(str(dt_tz))       # 2025-10-16 14:30:45+03:00
print(repr(dt_tz))      # datetime.datetime(2025, 10, 16, 14, 30, 45, tzinfo=datetime.timezone(datetime.timedelta(seconds=10800)))
```

### Разница для других типов

```python
# Строки
s = "Hello\nWorld"
print(str(s))       # Hello
                    # World
print(repr(s))      # 'Hello\nWorld'

# Числа (обычно одинаковые)
n = 42
print(str(n))       # 42
print(repr(n))      # 42

# None
print(str(None))    # None
print(repr(None))   # None

# Списки
lst = [1, 2, 3]
print(str(lst))     # [1, 2, 3]
print(repr(lst))    # [1, 2, 3]
```

## 🎯 Когда что использовать

| Ситуация | Использовать |
|---|---|
| Вывод для пользователя | `str()` |
| Отладка | `repr()` |
| Логирование | `repr()` |
| Консоль Python | `repr()` |
| Форматирование для UI | `str()` |
| Сохранение в файл (для отладки) | `repr()` |
