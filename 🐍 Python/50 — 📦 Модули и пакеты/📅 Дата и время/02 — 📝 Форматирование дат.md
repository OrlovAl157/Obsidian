# 📝 Форматирование дат в Python

## 📌 Коротко

> Форматирование дат — это преобразование объектов `datetime` в строки и обратно. Используется метод `strftime()` для преобразования в строку и `strptime()` для парсинга из строки.

---

## 💡 Полезные советы

- **strftime()** — преобразует datetime в строку (дата→строка)
- **strptime()** — парсит строку в datetime (строка→дата)
- **Коды формата чувствительны к регистру** — `%Y` и `%y` это разные годы
- **Локаль влияет на названия** — месяцы/дни на английском или русском
- **Используйте ISO формат для логов** — `%Y-%m-%d %H:%M:%S`
- **Российский формат для отчётов** — `%d.%m.%Y %H:%M:%S`

---

## 🚀 Быстрая справка

| Что нужно | Формат | Пример |
|---|---|---|
| Российская дата | `%d.%m.%Y` | 16.10.2025 |
| Российская дата + время | `%d.%m.%Y %H:%M:%S` | 16.10.2025 14:30:45 |
| ISO дата | `%Y-%m-%d` | 2025-10-16 |
| ISO дата + время | `%Y-%m-%d %H:%M:%S` | 2025-10-16 14:30:45 |
| Время 24-часовое | `%H:%M:%S` | 14:30:45 |
| Время 12-часовое | `%I:%M:%S %p` | 02:30:45 PM |

## Коды форматирования

### 📅 День недели

| Код | Значение | Пример |
|---|---|---|
| `%a` | Сокращённый день недели | Sun, Mon, ..., Sat |
| `%A` | Полный день недели | Sunday, Monday, ..., Saturday |
| `%w` | Номер дня [0-6] | 0 (вс), 6 (сб) |

### 📆 День

| Код | Значение | Пример |
|---|---|---|
| `%d` | День месяца [01-31] | 01, 02, ..., 31 |
| `%j` | День года [001-366] | 001, 002, ..., 366 |
| `%e` | День месяца без лидирующего нуля | 1, 2, ..., 31 |

### 📋 Месяц

| Код | Значение | Пример |
|---|---|---|
| `%b` | Сокращённый месяц | Jan, Feb, ..., Dec |
| `%B` | Полный месяц | January, February, ..., December |
| `%m` | Номер месяца [01-12] | 01, 02, ..., 12 |

### 📊 Год

| Код | Значение | Пример |
|---|---|---|
| `%y` | Год без века [00-99] | 00, 01, ..., 99 |
| `%Y` | Год с веком | 0001, ..., 2025, ..., 9999 |

### ⏰ Время

| Код | Значение | Пример |
|---|---|---|
| `%H` | Час [00-23] | 00, 01, ..., 23 |
| `%I` | Час [01-12] | 01, 02, ..., 12 |
| `%M` | Минуты [00-59] | 00, 01, ..., 59 |
| `%S` | Секунды [00-59] | 00, 01, ..., 59 |
| `%f` | Микросекунды | 000000 - 999999 |
| `%p` | AM/PM | AM, PM |

### 🌍 Временная зона

| Код | Значение | Пример |
|---|---|---|
| `%z` | UTC смещение | +0000, -0400, +1030 |
| `%Z` | Название зоны | UTC, EST, CST |

### 📍 Неделя

| Код | Значение | Пример |
|---|---|---|
| `%U` | Неделя [00-53] (неделя с вс) | 00, 01, ..., 53 |
| `%W` | Неделя [00-53] (неделя с пн) | 00, 01, ..., 53 |

### 🔀 Комбинированные

| Код | Значение | Пример |
|---|---|---|
| `%c` | Дата и время | Tue Aug 16 21:30:00 1988 |
| `%x` | Дата | 08/16/1988 |
| `%X` | Время | 21:30:00 |
| `%%` | Процент | % |

## 📚 Частые шаблоны

### Форматы по регионам

```python
from datetime import datetime

dt = datetime(2025, 10, 16, 14, 30, 45)

# 🇷🇺 Российский формат
dt.strftime('%d.%m.%Y')              # '16.10.2025'
dt.strftime('%d.%m.%Y %H:%M:%S')     # '16.10.2025 14:30:45'

# 🇺🇸 Американский формат
dt.strftime('%m/%d/%Y')              # '10/16/2025'
dt.strftime('%m/%d/%Y %I:%M %p')     # '10/16/2025 02:30 PM'

# 🌍 ISO формат (международный)
dt.strftime('%Y-%m-%d')              # '2025-10-16'
dt.strftime('%Y-%m-%d %H:%M:%S')     # '2025-10-16 14:30:45'

# 📧 RFC 2822 (для email)
dt.strftime('%a, %d %b %Y %H:%M:%S') # 'Thu, 16 Oct 2025 14:30:45'

# 📝 Полная дата (текст)
dt.strftime('%A, %B %d, %Y')         # 'Thursday, October 16, 2025'

# ⏱️ Только время
dt.strftime('%H:%M:%S')              # '14:30:45'
dt.strftime('%I:%M:%S %p')           # '02:30:45 PM'

# 🔬 С микросекундами
dt.strftime('%Y-%m-%d %H:%M:%S.%f')  # '2025-10-16 14:30:45.123456'
```

## 🔄 Преобразования

### Дата → Строка (strftime)

```python
from datetime import datetime

dt = datetime.now()

# Форматирование в разные форматы
print(dt.strftime('%d.%m.%Y'))           # 16.10.2025
print(dt.strftime('%Y-%m-%d'))           # 2025-10-16
print(dt.strftime('%A, %d %B %Y'))       # Thursday, 16 October 2025
```

### Строка → Дата (strptime)

```python
from datetime import datetime

# Парсинг из строки
date_string = '16.10.2025 14:30:45'
dt = datetime.strptime(date_string, '%d.%m.%Y %H:%M:%S')

print(dt)                    # 2025-10-16 14:30:45
print(dt.day)               # 16
print(dt.month)             # 10
print(dt.year)              # 2025
```

## 💻 Практические примеры

### Формирование имени файла с датой

```python
from datetime import datetime

dt = datetime.now()

# Имя файла с датой
filename = f"report_{dt.strftime('%Y%m%d_%H%M%S')}.txt"
# report_20251016_143045.txt

# Или более читаемый формат
filename = f"report_{dt.strftime('%d.%m.%Y')}.pdf"
# report_16.10.2025.pdf
```

### Логирование с временными метками

```python
from datetime import datetime

def log_message(msg):
    timestamp = datetime.now().strftime('%Y-%m-%d %H:%M:%S')
    print(f"[{timestamp}] {msg}")

log_message("Application started")
# [2025-10-16 14:30:45] Application started
```

### Вывод даты для пользователя

```python
from datetime import datetime
import locale

# Установка русской локали (если доступна)
try:
    locale.setlocale(locale.LC_TIME, 'ru_RU.UTF-8')
except:
    pass

dt = datetime.now()

# Красивый формат для пользователя
user_format = dt.strftime('%d %B %Y года, %A')
# 16 October 2025 года, Thursday (или на русском если локаль установлена)
```

### Парсинг дат из разных источников

```python
from datetime import datetime

# Парсинг российского формата
dt1 = datetime.strptime('16.10.2025', '%d.%m.%Y')

# Парсинг американского формата
dt2 = datetime.strptime('10/16/2025', '%m/%d/%Y')

# Парсинг ISO формата
dt3 = datetime.strptime('2025-10-16', '%Y-%m-%d')

# Все три представляют одну дату!
print(dt1 == dt2 == dt3)  # True
```

## ⚠️ Частые ошибки

```python
from datetime import datetime

# ❌ Ошибка 1: Перепутан %Y и %y
dt_string = '25'
# dt = datetime.strptime(dt_string, '%Y')  # ❌ Неверно! %Y ждёт 4 символа
dt = datetime.strptime('2025', '%Y')      # ✅ Правильно

# ❌ Ошибка 2: Неправильный формат при парсинге
# datetime.strptime('16.10.2025', '%Y-%m-%d')  # ❌ ValueError!
dt = datetime.strptime('16.10.2025', '%d.%m.%Y')  # ✅ Правильно

# ❌ Ошибка 3: %H вместо %I для 12-часового формата
dt = datetime.now()
# dt.strftime('%I:%M:%S %p')  # ✅ Правильно (12-часовой)
# dt.strftime('%H:%M:%S %p')  # ⚠️ Можно, но %H — 24-часовой

# ❌ Ошибка 4: Забыли про микросекунды
try:
    dt = datetime.strptime('2025-10-16 14:30:45.123456', '%Y-%m-%d %H:%M:%S')
    # ❌ ValueError: time data does not match format
except ValueError:
    dt = datetime.strptime('2025-10-16 14:30:45.123456', '%Y-%m-%d %H:%M:%S.%f')
    # ✅ Правильно
```
