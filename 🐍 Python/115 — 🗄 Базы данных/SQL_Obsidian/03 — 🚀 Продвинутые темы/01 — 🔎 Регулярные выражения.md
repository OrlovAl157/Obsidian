# 🔍 Регулярные выражения в SQL

## Метасимволы

| Метасимвол | Что делает | Пример | Результат |
|------------|------------|--------|-----------|
| `.` | один любой символ | `.......` | beegeek |
| `{n}` | ровно `n` повторений | `be{5}ge{4}k` | beeeeegeeeek |
| `{n,k}` | от `n` до `k` повторений | `a{3,5}` | aaa, aaaa, aaaaa |
| `{n,}` | не менее `n` повторений | `a{4,}` | aaaa, aaaaa... |
| `{,n}` | не более `n` повторений | `a{,4}` | a, aa, aaa, aaaa |
| `\` | экранирование символов | `pygen\.ru` | pygen.ru |
| `\|` | ИЛИ | `[b-k]\|[4-8]` | d |
| `[...]` | один из набора | `[bd]` | b или d |
| `[.-.]` | диапазон | `[a-z]` | f |
| `[^...]` | НЕ из диапазона | `[^7-9]` | 6 |
| `^` | начало строки | `^[Hh]ello` | Hello... |
| `$` | конец строки | `[Gg]oodbye$` | ...Goodbye |
| `\d` | цифра 0-9 | `\d` | 9 |
| `\D` | не цифра | `bee\Dgee\D` | bee-geek |
| `\s` | пробельный символ | `...\s....` | bee geek |
| `\S` | не пробельный символ | `...\S....` | bee-geek |
| `\w` | буква, цифра, `_` | `\w\w\w\w` | _x1_ |
| `\W` | не буква, не цифра, не `_` | `\W\W` | !? |
| `\b` | граница слова | `\bcat\b` | cat отдельно |
| `\B` | не граница слова | `\Bcat` | scat, ducati, alcatel |
| `+` | 1 и более раз | `\d+` | 19587792597 |
| `*` | 0 и более раз | `aaa\d*` | aaa, aaa8657 |
| `?` | 0 или 1 раз | `https?://` | http:// или https:// |
| `(...)` | группировка | `(SELECT){2}` | SELECTSELECT |
| `(...)\n` | ссылка на группу | `(\b\w+\b)[ ]+\1` | повторяющиеся слова |
| `(?P<name>)` | именованная группа | `(?P<word>\b\w+\b)[ ]+(?P=word)` | повторяющиеся слова |
| `.*?` | ленивый поиск | `<B>.*?</B>` | минимальное совпадение |

## Ленивый и жадный поиск

### Жадный поиск: `.*`

```sql
'<B>.*</B>'
-- Подписку нельзя оформить клиентам, живущим в <B>России</B>, а также <B>Беларуси</B>.
-- Найдёт: от первого <B> до последнего </B>
```

### Ленивый поиск: `.*?`

```sql
'<B>.*?</B>'
-- Найдёт: <B>России</B> и <B>Беларуси</B> по отдельности
```

## Примеры границ слов

```sql
-- \bcat  — находит cat в начале слова: cat, catalyst
-- cat\b  — находит cat в конце слова: scat, ducati
-- \bcat\b — находит отдельное слово cat
-- \Bcat  — находит cat не в начале слова: scat, ducati
-- cat\B  — находит cat не в конце слова: catalyst
-- \Bcat\B — находит cat внутри слова: alcatel
```

## Флаги

| Флаг | Что даёт |
|------|----------|
| `c` | учитывается регистр в регулярном выражении |
| `m` | метасимвол `\n` учитывается как новая строка |
| `n` | метасимвол `.` учитывает `\n` как символ |

## Функции регулярных выражений

### REGEXP_LIKE(строка, шаблон, флаги)

Показывает, есть ли совпадение с шаблоном.

```sql
SELECT REGEXP_LIKE('beegeek', 'bee', 'c') AS result1,        -- 1
       REGEXP_LIKE('beegeek', 'Bee', 'c') AS result2,        -- 0
       REGEXP_LIKE('beegeek', 'Bee') AS result3,             -- 1
       REGEXP_LIKE('Foo\nbar.', '^[A-Z].*\\.$', 'mc') AS result4;  -- 0
```

### REGEXP / RLIKE

Аналогично `REGEXP_LIKE`.

```sql
SELECT 'beegeek' REGEXP 'bee' AS result1,      -- 1
       'beegeek' RLIKE 'bee' AS result2,       -- 1
       'beegeek' NOT REGEXP 'bee' AS result3,  -- 0
       'beegeek' NOT RLIKE 'bee' AS result4;   -- 0
```

### REGEXP_INSTR(строка, шаблон, позиция, номер_совпадения, тип_возврата, флаги)

Определение местоположения подстроки.

```sql
SELECT REGEXP_INSTR('beegeek', 'e{2}', 1, 1, 1) AS result1,  -- 4
       REGEXP_INSTR('beegeek', 'e{2}', 1, 2, 1) AS result2;  -- 7
```

### REGEXP_SUBSTR(строка, шаблон, позиция, номер_совпадения, флаги)

Извлечение подстроки.

```sql
SELECT REGEXP_SUBSTR('beegeeek', 'e{2,}', 1, 1) AS result1,  -- ee
       REGEXP_SUBSTR('beegeeek', 'e{2,}', 1, 2) AS result2,  -- eee
       REGEXP_SUBSTR('beegeeek', 'e{2,}', 1, 3) AS result3;  -- NULL
```

### REGEXP_REPLACE(строка, шаблон, замена, позиция, номер_совпадения, флаги)

Замена подстроки.

```sql
SELECT REGEXP_REPLACE('beeeeegeeek', 'a', 'i') AS result1,  -- beeeeegeeek
       REGEXP_REPLACE('beegeeg', 'eee', 'ee') AS result2;   -- beegeeg
```
