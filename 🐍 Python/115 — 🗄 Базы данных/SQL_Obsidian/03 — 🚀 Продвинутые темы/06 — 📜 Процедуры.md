# 📦 Процедуры (Stored Procedures)

## Синтаксис создания

```sql
DELIMITER //
CREATE PROCEDURE procedure_name(param1 TYPE, param2 TYPE, ...)
BEGIN
    -- тело процедуры
END //
DELIMITER ;
```

## Вызов процедуры

```sql
CALL procedure_name(параметры);
```

## Типы параметров

| Тип | Описание |
|-----|----------|
| `IN` | входной параметр: значение передаётся в процедуру |
| `OUT` | выходной параметр: значение возвращается из процедуры |
| `INOUT` | входной и выходной параметр |

## Пример 1: отображение таблицы

```sql
DELIMITER //
CREATE PROCEDURE HIGH_GRADE_STUDENTS()
BEGIN
    SELECT *
    FROM Math
    WHERE grade >= 4;
END //
DELIMITER ;

CALL HIGH_GRADE_STUDENTS();

-- Результат:
-- id | name | surname | grade
-- 2 | Peter | Parker | 5
-- 4 | Gwen | Stacy | 4
-- 5 | Harry | Osborn | 5
-- 7 | Miles | Morales | 5
-- 9 | Curtis | Connors | 4
```

## Пример 2: изменение исходной таблицы

```sql
DELIMITER //
CREATE PROCEDURE DELETE_LOW_GRADE_STUDENTS()
BEGIN
    DELETE FROM Math
    WHERE grade <= 2;
END //
DELIMITER ;

CALL DELETE_LOW_GRADE_STUDENTS();

SELECT * FROM Math;
-- Останутся только студенты с оценками 3, 4, 5
```

## Пример 3: счётчик удалённых записей

```sql
DELIMITER //
CREATE PROCEDURE DELETE_LOW_GRADE_STUDENTS()
BEGIN
    DECLARE low_grade_students INT;

    SET low_grade_students := (SELECT COUNT(*)
                               FROM Math
                               WHERE grade <= 2);

    DELETE FROM Math
    WHERE grade <= 2;

    SELECT low_grade_students AS deleted_students;
END //
DELIMITER ;

CALL DELETE_LOW_GRADE_STUDENTS();

-- Результат:
-- deleted_students
-- 4
```

## Пример 4: передача данных в процедуру (IN)

```sql
DELIMITER //
CREATE PROCEDURE STUDENTS_WITH_GRADE_BETWEEN(IN min_score INT, IN max_score INT)
BEGIN
    SELECT id, name, surname, grade
    FROM Math
    WHERE grade BETWEEN min_score AND max_score;
END //
DELIMITER ;

CALL STUDENTS_WITH_GRADE_BETWEEN(2, 4);

-- Результат:
-- id | name | surname | grade
-- 1 | Flash | Thompson | 2
-- 3 | Mary | Jane | 2
-- 4 | Gwen | Stacy | 4
-- 9 | Curtis | Connors | 4
-- 10 | Edward | Brock | 3
```

## Пример 5: вывод данных из процедуры (OUT)

```sql
DELIMITER //
CREATE PROCEDURE MAX_GRADE(OUT score INT)
BEGIN
    SET score := (SELECT MAX(grade) FROM Math);
END //
DELIMITER ;

SET @max_grade = NULL;
CALL MAX_GRADE(@max_grade);
SELECT @max_grade;

-- Результат: 5
```

## Пример 6: ввод и вывод данных (INOUT)

```sql
DELIMITER //
CREATE PROCEDURE PROC(INOUT parameter INT)
BEGIN
    SELECT parameter;
END //
DELIMITER ;

SET @variable := 1;
CALL PROC(@variable);

-- Результат: 1
```
