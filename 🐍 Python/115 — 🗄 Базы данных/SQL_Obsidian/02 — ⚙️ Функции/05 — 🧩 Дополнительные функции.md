# 🔧 Дополнительные функции SQL

## ISNULL()

Проверяет, что значение равно `NULL`.

```sql
SELECT ISNULL(NULL);
-- 1
```

## IF()

Выбор одного из двух значений.

```sql
SELECT IF(1 < 0, 'bee', 'geek');
-- geek
```

## COALESCE()

Поиск первого непустого значения.

```sql
SELECT COALESCE(NULL, NULL, 2, 3);
-- 2
```

## IFNULL()

Если первое значение `NULL`, выбирает второе.

```sql
SELECT IFNULL(NULL, 'geek');
-- geek
```

## NULLIF()

Сравнение двух значений.

```sql
SELECT NULLIF('bee', 'bee');
-- NULL

SELECT NULLIF('bee', 'geek');
-- bee
```

## CONVERT()

Приводит значение к нужному типу.

```sql
SELECT CONVERT(100.78, DECIMAL(4, 1));
-- 100.8
```

## CAST()

Приведение типа.

```sql
SELECT CAST(100.78 AS DECIMAL(3, 0));
-- 101
```

## CASE

Множественное условие, похожее на `ELSE IF`.

### Вариант 1: сравнение с выражением

```sql
CASE quantity
    WHEN 0 THEN 'Out of stock'
    WHEN 1 THEN 'One in stock'
    WHEN 2 THEN 'Little in stock'
    WHEN 3 THEN 'Little in stock'
    WHEN 4 THEN 'Little in stock'
    WHEN 5 THEN 'Little in stock'
    ELSE 'A lot in stock'
END AS availability
```

### Вариант 2: логические условия

```sql
CASE
    WHEN quantity = 0 THEN 'Out of stock'
    WHEN quantity = 1 THEN 'One in stock'
    WHEN quantity BETWEEN 2 AND 5 THEN 'Little in stock'
    ELSE 'A lot in stock'
END AS availability
```

### Результат выполнения

```text
+--------------------+
| availability       |
+--------------------+
| A lot in stock     |
| One in stock       |
| Little in stock    |
| Little in stock    |
| Out of stock       |
+--------------------+
```
