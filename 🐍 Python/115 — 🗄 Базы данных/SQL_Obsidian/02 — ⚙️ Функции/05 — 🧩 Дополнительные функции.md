## Работа с NULL

### `ISNULL()`

Проверяет, равно ли значение `NULL`.

```sql
SELECT ISNULL(NULL);
-- 1
```

### `COALESCE()`

Возвращает первое непустое значение из списка.

```sql
SELECT COALESCE(NULL, NULL, 2, 3);
-- 2
```

### `IFNULL()`

Если первое значение `NULL`, возвращает второе.

```sql
SELECT IFNULL(NULL, 'geek');
-- geek
```

### `NULLIF()`

Сравнивает два значения и возвращает `NULL`, если они равны.

```sql
SELECT NULLIF('bee', 'bee');
-- NULL

SELECT NULLIF('bee', 'geek');
-- bee
```

## Условные функции

### `IF()`

Выбирает одно из двух значений по условию.

```sql
SELECT IF(1 < 0, 'bee', 'geek');
-- geek
```

## Приведение типов

### `CONVERT()`

Приводит значение к нужному типу.

```sql
SELECT CONVERT(100.78, DECIMAL(4, 1));
-- 100.8
```

### `CAST()`

Преобразует значение к указанному типу.

```sql
SELECT CAST(100.78 AS DECIMAL(3, 0));
-- 101
```

## Конструкция CASE

### Что делает `CASE`

`CASE` позволяет задать несколько условий, похожих на `ELSE IF`.

### Вариант 1: сравнение с выражением

```sql
CASE quantity
    WHEN 0 THEN 'Out of stock'
    WHEN 1 THEN 'One in stock'
    WHEN 2 THEN 'Little in stock'
    WHEN 3 THEN 'Little in stock'
    WHEN 4 THEN 'Little in stock'
    WHEN 5 THEN 'Little in stock'
    ELSE 'A lot in stock'
END AS availability
```

### Вариант 2: логические условия

```sql
CASE
    WHEN quantity = 0 THEN 'Out of stock'
    WHEN quantity = 1 THEN 'One in stock'
    WHEN quantity BETWEEN 2 AND 5 THEN 'Little in stock'
    ELSE 'A lot in stock'
END AS availability
```

### Результат выполнения

```text
+--------------------+
| availability       |
+--------------------+
| A lot in stock     |
| One in stock       |
| Little in stock    |
| Little in stock    |
| Out of stock       |
+--------------------+
```
