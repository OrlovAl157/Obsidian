# 🔢 Числовые функции SQL

## DIV()

Целочисленное деление.

```sql
SELECT 11 DIV 3;
-- 3
```

## MOD()

Остаток от деления.

```sql
SELECT 11 MOD 3;
-- 2

SELECT 11 % 3;
-- 2

SELECT MOD(11, 3);
-- 2
```

## ABS()

Модуль числа.

```sql
SELECT ABS(-2.1234);
-- 2.1234
```

## ROUND()

Округление.

```sql
SELECT ROUND(23.1653, 2);
-- 23.17
```

## POW()

Возведение в степень.

```sql
SELECT POW(2, 3);
-- 8
```

## SQRT()

Квадратный корень.

```sql
SELECT SQRT(9);
-- 3
```

## RAND()

Случайное число от 0 до 1.

```sql
SELECT RAND();
-- 0.8266121575856694
```

## FLOOR()

Округление вниз.

```sql
SELECT FLOOR(-11.22);
-- -12
```

## CEILING()

Округление вверх.

```sql
SELECT CEILING(18.39);
-- 19
```

## DEGREES()

Перевод из радиан в градусы.

```sql
SELECT DEGREES(3.14);
-- 179.9087476710785
```

## RADIANS()

Перевод из градусов в радианы.

```sql
SELECT RADIANS(180);
-- 3.141592653589793
```

## PI()

Число Пи.

```sql
SELECT PI();
-- 3.141593
```

## SIN() / COS() / TAN() / COT()

Тригонометрические функции.

```sql
SELECT SIN(PI());
-- 0

SELECT COS(PI());
-- -1

SELECT TAN(PI());
-- 0

SELECT COT(PI());
-- -8165619676597685
```

## LEAST() / GREATEST()

Минимум и максимум из списка.

```sql
SELECT LEAST(2, 1, 3, 5, 4);
-- 1

SELECT GREATEST(2, 1, 3, 5, 4);
-- 5
```

## CONV()

Перевод числа из одной системы счисления в другую.

```sql
SELECT CONV('10F', 16, 2);
-- 100001111
```

## FORMAT()

Округление и форматирование числа. Добавляет разделители тысяч.

```sql
SELECT FORMAT(1000000.1653, 2);
-- 1,000,000.17
```

## LOG2()

Двоичный логарифм.

```sql
SELECT LOG2(65536);
-- 16
```

## LOG10()

Логарифм по основанию 10.

```sql
SELECT LOG10(100);
-- 2
```
