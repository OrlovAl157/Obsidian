## Деление и остаток

### `DIV`

Целочисленное деление.

```sql
SELECT 11 DIV 3;
-- 3
```

### `MOD()`

Возвращает остаток от деления.

```sql
SELECT 11 MOD 3;
-- 2

SELECT 11 % 3;
-- 2

SELECT MOD(11, 3);
-- 2
```

## Округление и модуль

### `ABS()`

Возвращает модуль числа.

```sql
SELECT ABS(-2.1234);
-- 2.1234
```

### `ROUND()`

Округляет число до нужного количества знаков.

```sql
SELECT ROUND(23.1653, 2);
-- 23.17
```

### `FLOOR()`

Округляет вниз.

```sql
SELECT FLOOR(-11.22);
-- -12
```

### `CEILING()`

Округляет вверх.

```sql
SELECT CEILING(18.39);
-- 19
```

## Степени и корни

### `POW()`

Возводит число в степень.

```sql
SELECT POW(2, 3);
-- 8
```

### `SQRT()`

Возвращает квадратный корень.

```sql
SELECT SQRT(9);
-- 3
```

### `RAND()`

Генерирует случайное число от `0` до `1`.

```sql
SELECT RAND();
-- 0.8266121575856694
```

## Тригонометрия и углы

### `DEGREES()`

Переводит радианы в градусы.

```sql
SELECT DEGREES(3.14);
-- 179.9087476710785
```

### `RADIANS()`

Переводит градусы в радианы.

```sql
SELECT RADIANS(180);
-- 3.141592653589793
```

### `PI()`

Возвращает число Пи.

```sql
SELECT PI();
-- 3.141593
```

### `SIN()` / `COS()` / `TAN()` / `COT()`

Тригонометрические функции.

```sql
SELECT SIN(PI());
-- 0

SELECT COS(PI());
-- -1

SELECT TAN(PI());
-- 0

SELECT COT(PI());
-- -8165619676597685
```

## Минимум, максимум и форматирование

### `LEAST()` / `GREATEST()`

Возвращают минимум или максимум из списка.

```sql
SELECT LEAST(2, 1, 3, 5, 4);
-- 1

SELECT GREATEST(2, 1, 3, 5, 4);
-- 5
```

### `FORMAT()`

Округляет и форматирует число, добавляя разделители тысяч.

```sql
SELECT FORMAT(1000000.1653, 2);
-- 1,000,000.17
```

## Логарифмы и системы счисления

### `CONV()`

Переводит число из одной системы счисления в другую.

```sql
SELECT CONV('10F', 16, 2);
-- 100001111
```

### `LOG2()`

Возвращает двоичный логарифм.

```sql
SELECT LOG2(65536);
-- 16
```

### `LOG10()`

Возвращает логарифм по основанию `10`.

```sql
SELECT LOG10(100);
-- 2
```
