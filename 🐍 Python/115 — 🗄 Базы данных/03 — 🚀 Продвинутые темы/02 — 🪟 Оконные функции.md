# Windows Оконные функции SQL

## Синтаксис

```sql
SELECT full_name, department, salary,
       ROW_NUMBER() OVER (PARTITION BY department ORDER BY salary DESC) AS row_num
FROM Employees;
```

### Именованное окно

```sql
SELECT full_name, department, salary,
       ROW_NUMBER() OVER salary_desc_part_by_department AS row_num
FROM Employees
WINDOW salary_desc_part_by_department AS (
    PARTITION BY department
    ORDER BY salary DESC
    ROWS BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW
);
```

## Ранжирующие функции

### ROW_NUMBER()

Нумерация записей.

```sql
SELECT full_name, department, salary,
       ROW_NUMBER() OVER (ORDER BY salary DESC) AS row_num
FROM Employees;

-- Результат:
-- Sergey Brin | Engineering | 10000 | 1
-- Alice Johnson | Engineering | 10000 | 2
-- Susan Wojcicki | Engineering | 8000 | 3
-- Larry Page | Engineering | 7000 | 4
-- John Smith | Engineering | 7000 | 5
-- Marissa Mayer | Marketing | 9000 | 1
-- Sheryl Sandberg | Marketing | 9000 | 2
-- Eric Schmidt | Marketing | 8000 | 3
-- Sundar Pichai | Sales | 11000 | 1
-- John Doerr | Sales | 10000 | 2
```

### RANK()

Ранг с пропусками: `1, 2, 2, 4`.

```sql
SELECT full_name, salary,
       RANK() OVER (ORDER BY salary DESC) AS row_rank
FROM Employees;
-- 11000: 1, 10000: 2, 10000: 2, 9000: 4, 9000: 4, 8000: 6...
```

### DENSE_RANK()

Ранг без пропусков: `1, 2, 2, 3`.

```sql
SELECT full_name, salary,
       DENSE_RANK() OVER (ORDER BY salary DESC) AS row_dense_rank
FROM Employees;
-- 11000: 1, 10000: 2, 10000: 2, 9000: 3, 9000: 3...
```

### Сравнение ROW_NUMBER(), DENSE_RANK() и RANK()

```sql
SELECT full_name, salary,
       ROW_NUMBER() OVER salary_desc AS row_num,
       DENSE_RANK() OVER salary_desc AS row_dense_rank,
       RANK() OVER salary_desc AS row_rank
FROM Employees
WINDOW salary_desc AS (ORDER BY salary DESC);

-- Результат:
-- Sundar Pichai | 11000 | 1 | 1 | 1
-- Sergey Brin | 10000 | 2 | 2 | 2
-- John Doerr | 10000 | 3 | 2 | 2
-- Alice Johnson | 10000 | 4 | 2 | 2
-- Marissa Mayer | 9000 | 5 | 3 | 5
-- Sheryl Sandberg | 9000 | 6 | 3 | 5
-- Eric Schmidt | 8000 | 7 | 4 | 7
-- Susan Wojcicki | 8000 | 8 | 4 | 7
-- Larry Page | 7000 | 9 | 5 | 9
-- John Smith | 7000 | 10 | 5 | 9
```

### NTILE(n)

Разбивает окно на `n` групп с равным количеством записей.

```sql
SELECT id, full_name,
       NTILE(3) OVER (ORDER BY id) AS group_number
FROM Employees;

-- Результат:
-- 1 | Sergey Brin | 1
-- 2 | John Doerr | 1
-- 3 | Larry Page | 1
-- 4 | Eric Schmidt | 1
-- 5 | Sundar Pichai | 2
-- 6 | Marissa Mayer | 2
-- 7 | Susan Wojcicki | 2
-- 8 | John Smith | 3
-- 9 | Sheryl Sandberg | 3
-- 10 | Alice Johnson | 3
```

## Агрегатные функции как оконные

### AVG()

Среднеарифметическое по окну.

```sql
SELECT full_name, department, salary,
       AVG(salary) OVER (PARTITION BY department) AS department_avg_salary
FROM Employees;
-- Engineering: 8750.0000
-- Marketing: 8500.0000
-- Sales: 10500.0000
```

### SUM()

Сумма по окну.

```sql
SELECT full_name, department, salary,
       SUM(salary) OVER () AS sum_salary,
       SUM(salary) OVER (PARTITION BY department) AS department_sum_salary
FROM Employees;
-- sum_salary: 73000
-- Engineering: 35000, Marketing: 17000, Sales: 21000
```

### COUNT()

Подсчёт не-NULL значений.

```sql
SELECT full_name, department,
       COUNT(*) OVER (PARTITION BY department) AS employees_in_department
FROM Employees;
-- Engineering: 5, Marketing: 3, Sales: 2
```

### MIN() / MAX()

Минимум и максимум по окну.

```sql
SELECT full_name, department,
       MIN(salary) OVER (PARTITION BY department) AS department_min_salary,
       MAX(salary) OVER (PARTITION BY department) AS department_max_salary
FROM Employees;
-- Engineering: 7000 | 10000
-- Marketing: 8000 | 9000
-- Sales: 10000 | 11000
```

## Границы окон

### ROWS

Граница по строкам.

| Значение | Описание |
|----------|----------|
| `UNBOUNDED PRECEDING` | самая первая запись окна или секции |
| `n PRECEDING` | `n` записей перед текущей |
| `CURRENT ROW` | текущая запись |
| `n FOLLOWING` | `n` записей после текущей |
| `UNBOUNDED FOLLOWING` | самая последняя запись окна или секции |

```sql
SELECT id, full_name, salary,
       AVG(salary) OVER (
           ORDER BY id
           ROWS BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW
       ) AS avg_salary
FROM Employees;

-- Результат:
-- 1 | Sergey Brin | 10000 | 10000.0000
-- 2 | John Doerr | 10000 | 10000.0000
-- 3 | Larry Page | 7000 | 9000.0000
-- 4 | Eric Schmidt | 8000 | 8750.0000
-- 5 | Sundar Pichai | 11000 | 9200.0000
-- 6 | Marissa Mayer | 9000 | 9166.6667
-- 7 | Susan Wojcicki | 8000 | 9000.0000
-- 8 | John Smith | 7000 | 8750.0000
-- 9 | Sheryl Sandberg | 9000 | 8777.7778
-- 10 | Alice Johnson | 10000 | 8900.0000
```

### RANGE

Граница по значениям. Используется только для дат, времени и чисел.

```sql
-- Для чисел: разница в 20 от текущего значения
SELECT amount,
       COUNT(amount) OVER (
           ORDER BY amount
           RANGE BETWEEN 20 PRECEDING AND 20 FOLLOWING
       ) - 1 AS diff_twenty
FROM Orders;

-- Результат:
-- 75 | 1
-- 90 | 2
-- 100 | 2
-- 120 | 1
-- 150 | 0
-- 180 | 1
-- 200 | 2
-- 210 | 1
-- 250 | 0
-- 300 | 0
```

```sql
-- Для дат: интервал в 2 дня
SELECT Orders.*,
       AVG(amount) OVER (
           ORDER BY date
           RANGE BETWEEN INTERVAL 2 DAY PRECEDING AND CURRENT ROW
       ) AS avg_amount
FROM Orders;
```

## Функции смещения

### FIRST_VALUE()

Значение из первой записи окна.

```sql
SELECT id, full_name, salary,
       FIRST_VALUE(salary) OVER (ORDER BY id) AS first_employee_salary
FROM Employees;
-- У всех будет 10000: зарплата Sergey Brin
```

### LAST_VALUE()

Значение из последней записи окна.

```sql
SELECT id, full_name, salary,
       LAST_VALUE(salary) OVER (
           ORDER BY id
           ROWS BETWEEN UNBOUNDED PRECEDING AND UNBOUNDED FOLLOWING
       ) AS last_employee_salary
FROM Employees;
-- У всех будет 12000: зарплата Alice Johnson
```

### NTH_VALUE(column, n)

Значение из n-ой записи окна.

```sql
SELECT id, full_name, salary,
       NTH_VALUE(salary, 2) OVER id_asc AS second_employee_salary,
       NTH_VALUE(salary, 3) OVER id_asc AS third_employee_salary
FROM Employees
WINDOW id_asc AS (
    ORDER BY id
    ROWS BETWEEN UNBOUNDED PRECEDING AND UNBOUNDED FOLLOWING
);
-- second: 10000 (John Doerr), third: 7000 (Larry Page)
```

### LAG(column, n, default)

Значение на `n` строк назад.

```sql
SELECT id, full_name, salary,
       LAG(salary, 1, 0) OVER (ORDER BY id) AS prev_employee_salary
FROM Employees;
-- Sergey Brin: 0
-- John Doerr: 10000
-- Larry Page: 10000
-- Eric Schmidt: 7000...
```

### LEAD(column, n, default)

Значение на `n` строк вперёд.

```sql
SELECT id, full_name, salary,
       LEAD(salary, 1, 'empty') OVER (ORDER BY id) AS next_employee_salary
FROM Employees;
-- Sergey Brin: 10000
-- John Doerr: 7000
-- Larry Page: 8000
-- ...
-- Alice Johnson: empty
```

## Создание нескольких окон

```sql
SELECT full_name, salary
FROM Employees
WHERE department = 'Engineering'
WINDOW salary_asc AS (ORDER BY salary),
       salary_desc AS (ORDER BY salary DESC);
```
