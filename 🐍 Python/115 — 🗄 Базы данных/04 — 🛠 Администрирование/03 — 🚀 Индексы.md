# 🔍 Индексы

Индексы ускоряют поиск данных ценой дополнительного места на диске и замедления `INSERT`, `UPDATE`, `DELETE`.

## Создание индекса

```sql
CREATE INDEX index_name
ON table_name(column_name);
```

## Просмотр индексов

```sql
SHOW INDEX FROM table_name;
```

## Удаление индекса

```sql
DROP INDEX index_name
ON table_name;
```

## Переименование индекса

```sql
ALTER TABLE table_name
RENAME INDEX current_index_name TO new_index_name;
```

## Примечания

- Первичный ключ (`PRIMARY KEY`) и уникальные ограничения (`UNIQUE`) автоматически создают индексы.
- Индексы особенно полезны для полей, которые часто используются в `WHERE`, `JOIN`, `ORDER BY`.
- Не стоит создавать индексы на маленьких таблицах или полях с низкой селективностью.
