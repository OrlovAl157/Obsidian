# 👁 Представления (VIEW)

Представление — это виртуальная таблица, содержимое которой определяется запросом.

## Создание представления

```sql
CREATE VIEW view_name AS
SELECT column1, column2, ...
FROM table_name
WHERE condition;
```

## Изменение представления

```sql
ALTER VIEW view_name AS
SELECT column1, column2, ...
FROM table_name
WHERE new_condition;
```

## Удаление представления

```sql
DROP VIEW view_name;
```

## Пример

```sql
-- создаём представление с книгами после 2000 года
CREATE VIEW RecentBooks AS
SELECT title, author, release_year
FROM Books
WHERE release_year > 2000;

-- используем представление как обычную таблицу
SELECT * FROM RecentBooks
WHERE author = 'Stephen King';
```

## Преимущества представлений

- Безопасность: можно скрыть чувствительные колонки.
- Упрощение сложных запросов: часто используемые `JOIN` можно вынести в представление.
- Логическая независимость данных: изменение структуры таблицы можно скрыть за представлением.
