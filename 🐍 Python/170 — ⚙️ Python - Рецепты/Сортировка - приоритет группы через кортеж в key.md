
## Код

```python
def sort_priority(values, group):
    values.sort(key=lambda x: (x in group, x))
```

## Как работает

`key` возвращает кортеж `(bool, value)` — Python сортирует кортежи поэлементно:
- сначала по первому элементу: `False < True`, значит не-группа идёт раньше
- потом по второму: обычная числовая сортировка внутри каждой половины

```
values = [5, 2, 8, 1, 3]
group  = {2, 5}

5 → (True, 5)    # в группе
2 → (True, 2)    # в группе
8 → (False, 8)   # не в группе
1 → (False, 1)   # не в группе
3 → (False, 3)   # не в группе

Результат: [1, 3, 8, 2, 5]
# сначала не-группа по возрастанию, потом группа по возрастанию
```

## Инвертировать приоритет

Чтобы группа шла **первой**:

```python
values.sort(key=lambda x: (x not in group, x))
# [2, 5, 1, 3, 8]
```

## Суть приёма
Булево значение в кортеже — это неявный числовой вес (`False=0, True=1`). Можно добавлять любое количество уровней сортировки просто расширяя кортеж.
