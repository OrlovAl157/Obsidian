# kwargs в атрибуты через `__dict__.update`

## Суть
Универсальный класс, который принимает любые именованные аргументы и превращает их в атрибуты объекта.

## Код

```python
class AnyClass:
    def __init__(self, **kwargs) -> None:
        self.__dict__.update(kwargs)

cowboy = AnyClass(name='John', surname='Marston')
print(cowboy.__dict__)
# {'name': 'John', 'surname': 'Marston'}
```

## Как работает
- `**kwargs` собирает все именованные аргументы в словарь
- `self.__dict__` — это и есть словарь атрибутов объекта
- `.update(kwargs)` просто вливает переданные пары ключ-значение прямо в этот словарь

## Когда полезно
Когда заранее не знаешь набор полей — например, при маппинге строк из БД или JSON в объект.
