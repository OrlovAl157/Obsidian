```python
# 🔀 MATCH-CASE — ОСНОВЫ (Python 3.10+)
match value:
    case 42:             # литерал — точное совпадение
    case 'bee' | 'geek': # OR — одно из значений
    case x:              # захват — любое значение → x
    case x if x > 0:     # охранник — захват + условие
    case _:              # заглушка — всегда совпадает
```

# 🔀 Памятка: match-case — основы и простые шаблоны

`match-case` — оператор сопоставления с шаблонами (Python 3.10+). Проверяет значение сверху вниз до первого совпадения, автоматически выходит (не нужен `break`).

### 💡 Ключевые особенности

- **Порядок важен** — шаблоны проверяются сверху вниз, первое совпадение выигрывает.
- **`case x:`** — это не сравнение, это **захват**. Любое одиночное имя — шаблон захвата.
- **`case _:`** — заглушка, всегда совпадает. Только в конце и только один раз.
- **`True`/`False`/`None`** сравниваются через `is`, а не `==`.
- **`match` и `case`** — "мягкие" ключевые слова, их можно использовать как имена переменных (но не рекомендуется).
- **match-case медленнее `if-elif`** — если критична производительность, используй `if`.

### 💡 Полезные советы

- Чтобы сравнить с **переменной** (не захватить) — используй охранник: `case x if x == target:`.
- Шаблон захвата **перезаписывает** переменную с тем же именем во внешней области видимости.
- `case _:` нельзя использовать дважды и нельзя ставить не в конце — `SyntaxError`.
- Захваченное имя **недоступно** в других ветках `case` (если только не захвачено в первом `case`).

### 🚀 Быстрая справка

| **Шаблон** | **Синтаксис** | **Что делает** |
|---|---|---|
| Литерал | `case 42:` | точное совпадение по значению |
| OR | `case 1 \| 2 \| 3:` | одно из значений |
| Захват | `case x:` | захватывает любое значение в `x` |
| Заглушка | `case _:` | всегда совпадает, значение недоступно |
| Охранник | `case x if x > 0:` | захват + дополнительное условие |

## 🔤 Синтаксис

**Назначение:** сопоставить результат выражения с шаблонами по очереди сверху вниз.

```python
match выражение:
    case шаблон_1:
        действие_1
    case шаблон_2:
        действие_2
    case _:
        по_умолчанию
```

```python
# В match можно передавать любое выражение Python
match bee + geek:
    case 'beegeek':
        print('Строка beegeek')

# Вызов функции
match division(121, 11):
    case int():
        print('Делится без остатка')
    case float():
        print('Делится с остатком')
```

**Важное:** `match` и `case` — мягкие ключевые слова. В отличие от `if`, их можно использовать как имена переменных, но это не рекомендуется.

## 🔵 Литеральные шаблоны

**Назначение:** проверить точное совпадение с конкретным значением.

| Тип данных | Тип сравнения |
|---|---|
| `int`, `float`, `complex`, `str`, `bytes` | равенство (`==`) |
| `True`, `False`, `None` | идентичность (`is`) |

**Примеры:**

```python
match s:
    case 'bee':
        print('Строка bee')
    case 'geek':
        print('Строка geek')

# Если тип не совпадает — ошибки нет, просто не совпадёт
s = 69
match s:
    case 'bee':   # не совпадёт — число не равно строке
        print('bee')
    case _:
        print('другое')   # → другое
```

```python
# True/False/None — через is, не ==
match value:
    case True:    # is True
        print('Истина')
    case None:    # is None
        print('Пусто')
```

**Важное:** первой проверяется принадлежность типу, потом значение. Число `69` никогда не совпадёт со строкой `'bee'` — ошибки не будет, просто пропустит.

## 🟡 Шаблон OR

**Назначение:** проверить совпадение с одним из нескольких значений в одном `case`.

**Примеры:**

```python
match s:
    case 'bee' | 'geek':
        print('Строка bee или geek')

match digit:
    case 0 | 3 | 6 | 9:
        print('Делится на 3')
    case 1 | 4 | 7:
        print('Остаток 1')
    case _:
        print('Другое')
```

```python
# | можно использовать и для типов
match num:
    case int() | float():
        print('Число')
```

**Важное:** `|` объединяет шаблоны внутри одного `case`. Для объединения целых кортежей нужны скобки:

```python
case (1, 2, 'bee') | (1, 2, 'geek'):   # ✅
case 1, 2, 'bee' | 1, 2, 'geek':       # ❌ неожиданный результат
```

## 🟢 Шаблон захвата

**Назначение:** захватить значение в переменную для использования в теле `case`.

**Примеры:**

```python
# Одиночное имя = шаблон захвата (никогда не ложный)
match s:
    case 'bee':
        print('Строка bee')
    case default:         # захватывает любое значение
        print(default)    # → pygen
```

```python
# ❗ Шаблон захвата перезаписывает внешнюю переменную
s = 'pygen'
default = 'beegeek'

print(default)   # beegeek

match s:
    case default:
        print(default)   # pygen

print(default)   # pygen -- внешняя переменная перезаписана!
```

```python
# ❗ Шаблон захвата ≠ сравнение с переменной
target = 42
match value:
    case target:   # ❌ НЕ сравнивает с target, захватывает в target!
        ...

# ✅ Правильно — охранник
match value:
    case x if x == target:
        ...
```

**Важное:** любое одиночное имя в `case` — шаблон захвата. Чтобы сравнить с переменной, используй охранник `if`.

---

## ⚫ Шаблон-заглушка `_`

**Назначение:** совпасть с любым значением без захвата (аналог `default` в других языках).

**Примеры:**

```python
match s:
    case 'bee':
        print('Строка bee')
    case 'geek':
        print('Строка geek')
    case _:
        print('Что-то другое')
```

```python
# ❗ _ недоступен в теле case
case _:
    print(_)   # ❌ NameError: name '_' is not defined

# ✅ Если нужно значение — используй имя
case other:
    print(other)   # ✅
```

**Важное:** `_` ставится только в конце и только один раз. Два `case _:` или `_` не в конце — `SyntaxError`.

---

## 🔶 Охранники (Guard)

**Назначение:** добавить дополнительное условие после совпадения с шаблоном.

**Примеры:**

```python
# Сначала совпадение с шаблоном, потом проверка условия
match numbers:
    case [_, _, _, *tail] if len(numbers) <= 5:
        print(tail)

# Захваченное имя доступно в охраннике
match len(s):
    case length if length <= 5:
        print(f'Короткая: {length}')
    case length if length <= 10:
        print(f'Средняя: {length}')
    case length:
        print(f'Длинная: {length}')
```

```python
# Захват из первого case доступен в следующих через _
match len(s):
    case length if length <= 5:
        print(f'Короткая: {length}')
    case _ if length <= 10:
        print(f'Средняя: {length}')
    case _:
        print(f'Длинная: {length}')
```

**Важное:** охранник проверяется **после** совпадения с шаблоном. Если шаблон совпал, но охранник ложен — переходит к следующему `case`.

## ⚠️ Важные особенности

```python
# ❗ case _: только в конце
match s:
    case _:        # ❌ SyntaxError
        ...
    case 'bee':
        ...

# ❗ case _: не может быть два раза
match s:
    case 'bee': ...
    case _: ...
    case _: ...    # ❌ SyntaxError

# ❗ match/case — мягкие ключевые слова
from keyword import softkwlist
print(softkwlist)  # ['_', 'case', 'match', 'type']

# ❗ match-case медленнее if-elif
# Для высоконагруженных участков кода предпочти if-elif
```
