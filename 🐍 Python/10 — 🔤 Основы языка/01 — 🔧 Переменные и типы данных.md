# 🔧 Переменные и типы данных

## 📌 Коротко

> Переменная — контейнер для хранения значений. Python — динамически типизированный язык, поэтому тип определяется автоматически при присваивании.

---

## 💡 Полезные советы

- **Имена переменных в snake_case** — `my_variable` а не `myVariable`
- **Используйте осмысленные имена** — `user_age` вместо `a`
- **Переменные чувствительны к регистру** — `Age` и `age` это разные переменные
- **Тип можно узнать через type()** — `type(x)`
- **Можно переприсваивать тип** — `x = 5; x = "text"`

---

## 🚀 Быстрая справка

| Тип | Описание | Пример |
|---|---|---|
| **int** | Целое число | `age = 25` |
| **float** | Дробное число | `height = 1.75` |
| **str** | Текст | `name = "Alice"` |
| **bool** | Истина/Ложь | `is_active = True` |
| **None** | Отсутствие значения | `value = None` |

## 📖 Теория

### Создание переменных

```python
# Целые числа
age = 25
temperature = -5
year = 2025

# Дробные числа
height = 1.75
pi = 3.14159

# Строки
name = "Alice"
message = 'Hello, World!'
multi_line = """Это
многострочная
строка"""

# Логические значения
is_active = True
is_empty = False

# Отсутствие значения
nothing = None
```

### Определение типа

```python
x = 42
print(type(x))      # <class 'int'>

y = 3.14
print(type(y))      # <class 'float'>

name = "Bob"
print(type(name))   # <class 'str'>

flag = True
print(type(flag))   # <class 'bool'>
```

### Преобразование типов

```python
# Строка в число
text = "42"
number = int(text)          # 42

# Число в строку
num = 100
string = str(num)           # "100"

# Дробное в целое
value = float("3.14")       # 3.14
whole = int(value)          # 3 (отбрасывает дробную часть)

# В булево значение
bool(1)             # True
bool(0)             # False
bool("")            # False
bool("text")        # True
```

### Множественное присваивание

```python
# Одно значение нескольким переменным
x = y = z = 0

# Распаковка кортежа
a, b, c = 1, 2, 3
print(a, b, c)      # 1 2 3

# С обменом значений
x, y = 10, 20
x, y = y, x         # Обменяли значения
print(x, y)         # 20 10
```

### Константы

```python
# Python соглашение - капс для констант
PI = 3.14159
MAX_SIZE = 100
TIMEOUT = 30

# Они не защищены, но это сигнал что менять нельзя
MAX_SIZE = 200  # ⚠️ Плохая идея, но работает
```

## 💻 Примеры

### Калькулятор

```python
# Входные данные
price = 100
quantity = 3
discount_percent = 10

# Вычисления
total = price * quantity
discount = total * (discount_percent / 100)
final_price = total - discount

# Вывод
print(f"Total: ${total}")
print(f"Discount: ${discount}")
print(f"Final: ${final_price}")
```

### Информация о человеке

```python
name = "Alice Johnson"
age = 28
height = 1.75
is_student = False

print(f"Name: {name}")
print(f"Age: {age} years")
print(f"Height: {height}m")
print(f"Student: {is_student}")
```

## ⚠️ Частые ошибки

```python
# ❌ Забыли присвоить значение
print(undefined_var)    # NameError: name 'undefined_var' is not defined

# ✅ Сначала присвойте
value = 42
print(value)            # 42

# ❌ Неправильное имя переменной
my-variable = 5         # SyntaxError: можно только буквы, цифры, _

# ✅ Используйте snake_case
my_variable = 5         # OK

# ❌ Забыли кавычки для строк
message = Hello         # NameError

# ✅ Строки в кавычках
message = "Hello"       # OK
```
