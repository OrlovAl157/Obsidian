# 📝 Ввод и вывод

## 📌 Коротко

> **print()** выводит текст, **input()** читает введённый пользователем текст.

---

## 🚀 Быстрая справка

```python
# Вывод
print("Hello")           # Hello
print(42)               # 42
print("A", "B", "C")    # A B C

# Ввод
name = input("Name: ")  # Прочитать строку
age = int(input("Age: ")) # Прочитать число

# Форматирование
x = 5
print(f"Value: {x}")    # Value: 5
print("Value: " + str(x))  # Value: 5
```

---

## 📖 Примеры

### Простой вывод

```python
print("Hello, World!")
print(42)
print(3.14)
print(True)
```

### Несколько значений

```python
print("Name:", "Alice", "Age:", 25)
# Вывод: Name: Alice Age: 25

print("A", "B", "C", sep="-")
# Вывод: A-B-C
```

### Ввод пользователя

```python
name = input("What is your name? ")
print("Hello, " + name)

# Ввод числа
age = int(input("How old are you? "))
print(f"You are {age} years old")
```

### Форматирование строк

```python
name = "Alice"
age = 25

# f-string (рекомендуется)
print(f"Name: {name}, Age: {age}")

# .format()
print("Name: {}, Age: {}".format(name, age))

# Старый способ
print("Name: %s, Age: %d" % (name, age))
```
