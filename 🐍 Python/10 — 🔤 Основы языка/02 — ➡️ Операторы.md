# ➡️ Операторы

## 📌 Коротко

> Операторы — символы которые выполняют операции с переменными и значениями (арифметика, сравнение, логика).

---

## 🚀 Быстрая справка

### Арифметические операторы

| Оператор | Описание | Пример |
|---|---|---|
| `+` | Сложение | `5 + 3` → `8` |
| `-` | Вычитание | `5 - 3` → `2` |
| `*` | Умножение | `5 * 3` → `15` |
| `/` | Деление | `6 / 3` → `2.0` |
| `//` | Целое деление | `7 // 3` → `2` |
| `%` | Остаток | `7 % 3` → `1` |
| `**` | Степень | `2 ** 3` → `8` |

### Операторы сравнения

| Оператор | Описание | Пример |
|---|---|---|
| `==` | Равно | `5 == 5` → `True` |
| `!=` | Не равно | `5 != 3` → `True` |
| `>` | Больше | `5 > 3` → `True` |
| `<` | Меньше | `5 < 3` → `False` |
| `>=` | Больше или равно | `5 >= 5` → `True` |
| `<=` | Меньше или равно | `5 <= 3` → `False` |

### Логические операторы

| Оператор | Описание | Пример |
|---|---|---|
| `and` | И | `True and False` → `False` |
| `or` | Или | `True or False` → `True` |
| `not` | Не | `not True` → `False` |

---

## 📖 Примеры

### Арифметика

```python
x = 10
y = 3

print(x + y)        # 13
print(x - y)        # 7
print(x * y)        # 30
print(x / y)        # 3.333...
print(x // y)       # 3
print(x % y)        # 1
print(x ** y)       # 1000
```

### Сравнение

```python
age = 25

print(age > 18)     # True
print(age == 25)    # True
print(age != 30)    # True
print(age <= 25)    # True
```

### Логика

```python
is_active = True
is_admin = False

print(is_active and is_admin)   # False
print(is_active or is_admin)    # True
print(not is_active)            # False
```
