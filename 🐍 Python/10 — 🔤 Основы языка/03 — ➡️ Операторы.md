# ➡️ Операторы в Python

## 📌 Коротко

> Операторы — символы и слова которые выполняют операции с переменными и значениями. Python имеет операторы для арифметики, сравнения, логики, присваивания, членства и идентичности.

---

## 💡 Полезные советы

- **Порядок операций важен** — как в математике (*, / перед +, -)
- **Используйте скобки для ясности** — `(a + b) * c` вместо `a + b * c`
- **Сравнивайте правильно** — `==` для значений, `is` для объектов
- **Логические операторы short-circuit** — вычисляют минимум нужного
- **Оператор `in` для членства** — проверка наличия элемента

---

## 🚀 Полная справка всех операторов

### 1️⃣ Арифметические операторы

| Оператор | Название | Описание | Пример | Результат |
|---|---|---|---|---|
| `+` | Сложение | Складывает два числа | `5 + 3` | `8` |
| `-` | Вычитание | Вычитает второе из первого | `5 - 3` | `2` |
| `*` | Умножение | Умножает числа | `5 * 3` | `15` |
| `/` | Деление | Делит (результат float) | `6 / 3` | `2.0` |
| `//` | Целое деление | Делит и округляет вниз | `7 // 3` | `2` |
| `%` | Остаток (модуль) | Остаток от деления | `7 % 3` | `1` |
| `**` | Степень (экспонента) | Возводит в степень | `2 ** 3` | `8` |
| `+x` | Унарный плюс | Не меняет число | `+5` | `5` |
| `-x` | Унарный минус | Меняет знак | `-5` | `-5` |

**Примеры:**
```python
# Арифметика
print(10 + 5)       # 15
print(10 - 5)       # 5
print(10 * 5)       # 50
print(10 / 3)       # 3.333...
print(10 // 3)      # 3
print(10 % 3)       # 1
print(2 ** 10)      # 1024

# Со строками
print("Hello" + " " + "World")  # Hello World
print("Hi" * 3)                 # HiHiHi
```

### 2️⃣ Операторы сравнения

| Оператор | Название | Описание | Пример | Результат |
|---|---|---|---|---|
| `==` | Равно | Проверяет равенство значений | `5 == 5` | `True` |
| `!=` | Не равно | Проверяет неравенство | `5 != 3` | `True` |
| `>` | Больше | Первое больше второго | `5 > 3` | `True` |
| `<` | Меньше | Первое меньше второго | `5 < 3` | `False` |
| `>=` | Больше или равно | Первое ≥ второго | `5 >= 5` | `True` |
| `<=` | Меньше или равно | Первое ≤ второго | `3 <= 5` | `True` |

**Примеры:**
```python
x = 10
print(x == 10)      # True
print(x != 5)       # True
print(x > 5)        # True
print(x < 5)        # False
print(x >= 10)      # True
print(x <= 10)      # True

# Цепочки сравнений
age = 25
print(18 <= age < 65)   # True (эквивалент: age >= 18 and age < 65)
```

### 3️⃣ Логические операторы

| Оператор | Описание | Пример | Результат |
|---|---|---|---|
| `and` | И (оба должны быть True) | `True and False` | `False` |
| `or` | Или (хотя бы один True) | `True or False` | `True` |
| `not` | Не (инвертирует) | `not True` | `False` |

**Таблица истинности AND:**
```
True  and True   = True
True  and False  = False
False and True   = False
False and False  = False
```

**Таблица истинности OR:**
```
True  or True   = True
True  or False  = True
False or True   = True
False or False  = False
```

**Примеры:**
```python
# AND - оба должны быть True
age = 25
has_license = True
if age >= 18 and has_license:
    print("Can drive")          # Выведется

# OR - хотя бы один True
score = 45
if score >= 90 or score <= 40:
    print("Exceptional")        # Выведется

# NOT - инвертирует
is_active = True
if not is_active:
    print("Inactive")           # НЕ выведется
```

**Short-circuit вычисление:**
```python
# AND - если первое False, второе не вычисляется
print(False and expensive_function())  # Функция не вызовется

# OR - если первое True, второе не вычисляется
print(True or expensive_function())    # Функция не вызовется
```

### 4️⃣ Операторы присваивания

| Оператор | Описание | Пример | Эквивалент |
|---|---|---|---|
| `=` | Присваивание | `x = 5` | `x = 5` |
| `+=` | Сложение и присваивание | `x += 3` | `x = x + 3` |
| `-=` | Вычитание и присваивание | `x -= 3` | `x = x - 3` |
| `*=` | Умножение и присваивание | `x *= 3` | `x = x * 3` |
| `/=` | Деление и присваивание | `x /= 3` | `x = x / 3` |
| `//=` | Целое деление и присваивание | `x //= 3` | `x = x // 3` |
| `%=` | Остаток и присваивание | `x %= 3` | `x = x % 3` |
| `**=` | Степень и присваивание | `x **= 3` | `x = x ** 3` |

**Примеры:**
```python
x = 10
x += 5      # x = 15
x -= 3      # x = 12
x *= 2      # x = 24
x /= 4      # x = 6.0
x //= 2     # x = 3.0
x %= 2      # x = 1.0
x **= 2     # x = 1.0

# Множественное присваивание
a = b = c = 0   # Все равны 0
```

### 5️⃣ Операторы членства (in/not in)

| Оператор | Описание | Пример | Результат |
|---|---|---|---|
| `in` | Элемент в последовательности | `5 in [1,2,5]` | `True` |
| `not in` | Элемента нет в последовательности | `5 not in [1,2,3]` | `True` |

**Примеры:**
```python
# Списки
fruits = ["apple", "banana", "cherry"]
print("apple" in fruits)        # True
print("grape" in fruits)        # False
print("grape" not in fruits)    # True

# Строки
text = "Hello, World!"
print("H" in text)              # True
print("xyz" in text)            # False

# Диапазоны
print(5 in range(10))           # True
print(15 in range(10))          # False

# Словари (проверяет ключи)
user = {"name": "Alice", "age": 25}
print("name" in user)           # True
print("Alice" in user)          # False (значение, не ключ)
```

### 6️⃣ Операторы идентичности (is/is not)

| Оператор | Описание | Пример | Результат |
|---|---|---|---|
| `is` | Один и тот же объект | `a is b` | `True/False` |
| `is not` | Разные объекты | `a is not b` | `True/False` |

**⚠️ Важно: `is` проверяет идентичность (адрес памяти), а не равенство!**

```python
# Равенство (==) vs Идентичность (is)
a = [1, 2, 3]
b = [1, 2, 3]

print(a == b)       # True (значения одинаковые)
print(a is b)       # False (разные объекты)

# None, True, False обычно используют is
x = None
if x is None:       # ✅ Правильно
    print("x is None")

if x == None:       # ⚠️ Работает, но не рекомендуется
    print("x equals None")

# Встроенные типы могут быть закэшированы
x = 5
y = 5
print(x is y)       # True (малые целые закэшированы)

x = 1000
y = 1000
print(x is y)       # False (большие целые не закэшированы)
```

### 7️⃣ Битовые операторы

| Оператор | Название | Описание | Пример | Результат |
|---|---|---|---|---|
| `&` | И (AND) | Битовое И | `5 & 3` | `1` |
| `\|` | Или (OR) | Битовое ИЛИ | `5 \| 3` | `7` |
| `^` | Исключающее или (XOR) | Битовое XOR | `5 ^ 3` | `6` |
| `~` | Не (NOT) | Битовое НЕ | `~5` | `-6` |
| `<<` | Левый сдвиг | Сдвиг влево | `5 << 1` | `10` |
| `>>` | Правый сдвиг | Сдвиг вправо | `5 >> 1` | `2` |

**Примеры:**
```python
# Битовые операции (для продвинутых)
print(5 & 3)    # 1  (101 & 011 = 001)
print(5 | 3)    # 7  (101 | 011 = 111)
print(5 ^ 3)    # 6  (101 ^ 011 = 110)
print(~5)       # -6
print(5 << 1)   # 10 (101 << 1 = 1010)
print(5 >> 1)   # 2  (101 >> 1 = 010)
```

### 8️⃣ Оператор условия (тернарный)

```python
# Синтаксис: value_if_true if condition else value_if_false

age = 25
status = "Adult" if age >= 18 else "Minor"
print(status)       # Adult

# С функциями
def check_grade(score):
    return "Pass" if score >= 60 else "Fail"

print(check_grade(75))  # Pass
print(check_grade(45))  # Fail

# Вложенные условия
x = 5
result = "Positive" if x > 0 else ("Negative" if x < 0 else "Zero")
print(result)       # Positive
```

## 📊 Приоритет операторов (порядок выполнения)

От высшего приоритета к низшему:

1. `()` — Скобки
2. `**` — Степень
3. `+x`, `-x`, `~x` — Унарные операторы
4. `*`, `/`, `//`, `%` — Умножение, деление
5. `+`, `-` — Сложение, вычитание
6. `<<`, `>>` — Битовые сдвиги
7. `&` — Битовое И
8. `^` — Битовое XOR
9. `|` — Битовое ИЛИ
10. `==`, `!=`, `>`, `<`, `>=`, `<=`, `is`, `in` — Сравнение
11. `not` — Логическое НЕ
12. `and` — Логическое И
13. `or` — Логическое ИЛИ
14. `if/else` — Условный оператор

**Примеры приоритета:**
```python
# Без скобок - порядок операций
print(2 + 3 * 4)        # 14 (умножение первое)
print((2 + 3) * 4)      # 20 (скобки первые)

# Степень важнее умножения
print(2 * 3 ** 2)       # 18 (3**2 = 9, затем 2*9)
print((2 * 3) ** 2)     # 36 (2*3 = 6, затем 6**2)
```

## 🎯 Практические примеры

### Валидация данных

```python
age = 25
is_citizen = True
has_record = False

# Может голосовать?
can_vote = age >= 18 and is_citizen and not has_record
print(can_vote)      # True

# Может взять кредит?
can_borrow = (age >= 21 and is_citizen) or age >= 25
print(can_borrow)    # True
```

### Расчёты со скидками

```python
price = 100
quantity = 5
discount_percent = 10

# Вычисления
total = price * quantity        # 500
discount = total * discount_percent / 100  # 50
final_price = total - discount  # 450

print(f"Total: ${total}")
print(f"Discount: ${discount}")
print(f"Final: ${final_price}")
```

### Проверка диапазонов

```python
age = 25
score = 85

# Возрастная категория
if age < 13:
    category = "Child"
elif 13 <= age < 18:
    category = "Teen"
elif 18 <= age < 65:
    category = "Adult"
else:
    category = "Senior"

print(f"Category: {category}")  # Adult

# Оценка
grade = "A" if score >= 90 else \
        "B" if score >= 80 else \
        "C" if score >= 70 else \
        "F"

print(f"Grade: {grade}")        # B
```

## ⚠️ Частые ошибки

```python
# ❌ Одиночный = вместо ==
if x = 5:           # SyntaxError!
    pass

# ✅ Двойной == для сравнения
if x == 5:
    pass

# ❌ 'and' вместо '&' для битовых операций
if x and y:         # Логическое И, не битовое

# ✅ & для битовых операций
result = x & y

# ❌ Забыли приоритет операций
result = 2 + 3 * 4  # 14 (не 20!)

# ✅ Используйте скобки для ясности
result = (2 + 3) * 4  # 20
```
