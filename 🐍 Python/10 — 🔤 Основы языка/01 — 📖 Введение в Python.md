# 📖 Введение в Python

## 📌 Коротко

> Python — интерпретируемый язык программирования высокого уровня, известный простотой синтаксиса и широким применением в веб, науке, автоматизации и ML.

---

## 💡 Ключевые особенности Python

- **Простой синтаксис** — читается почти как английский язык
- **Динамическая типизация** — типы определяются во время выполнения
- **Интерпретируемый язык** — код выполняется построчно
- **Объектно-ориентированный** — всё является объектом
- **Огромная экосистема** — множество библиотек и фреймворков
- **Кроссплатформенный** — работает везде

## 📖 Основные концепции

### Что такое программирование?

Программирование — это процесс написания инструкций для компьютера. Python выполняет эти инструкции слева направо, сверху вниз.

### Как работает Python?

1. **Вы пишете код** — текстовый файл (.py)
2. **Интерпретатор читает код** — строка за строкой
3. **Выполняет инструкции** — переменные, функции, циклы
4. **Выводит результат** — в консоль или файл

### Где используется Python?

- 🌐 **Веб** — Django, Flask
- 🔬 **Data Science** — Pandas, NumPy, Scikit-learn
- 🤖 **Machine Learning** — TensorFlow, PyTorch
- 🔧 **Автоматизация** — скрипты, боты
- 📊 **Анализ данных** — обработка больших объёмов
- 🎮 **Игры** — Pygame
- 📱 **Мобильные приложения** — Kivy

## 💻 Первая программа

### Hello, World!

```python
print("Hello, World!")
```

Вывод: `Hello, World!`

### С переменной

```python
name = "Alice"
print("Hello, " + name)
```

Вывод: `Hello, Alice`

### С функцией

```python
def greet(name):
    return f"Hello, {name}!"

message = greet("Bob")
print(message)
```

Вывод: `Hello, Bob!`

## 🔧 Установка и запуск

### Установка Python

```bash
# На Ubuntu/Debian
sudo apt-get install python3

# На macOS с Homebrew
brew install python3

# На Windows
# Скачать с python.org и установить
```

### Проверка установки

```bash
python3 --version
# Python 3.9.0
```

### Запуск программы

```bash
# Интерпретатор (для одной строки)
python3
>>> print("Hello")

# Запуск файла
python3 script.py

# Запуск модуля
python3 -m module_name
```

## ⚙️ Философия Python

Python следует "Дзен Python" — набору принципов:

Основные принципы:
- **Простота важнее сложности**
- **Читаемость имеет значение**
- **Явное лучше неявного**
- **Практичность важнее чистоты**
