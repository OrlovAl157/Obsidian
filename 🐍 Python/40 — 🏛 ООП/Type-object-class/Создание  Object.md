## 🧩 Что значит «`type` создаёт `object`»?

Когда мы говорим, что `type` _создаёт_ `object`,  
мы имеем в виду, что **`object` как класс** был **построен метаклассом `type`**  
(а не в том смысле, что `type` его "вылепил из воздуха").

---

## 🔹 Шаг 1. Как `type` обычно создаёт классы

Когда ты пишешь в коде:

`class MyClass:     pass`

Python под капотом делает примерно вот это:

`MyClass = type('MyClass', (), {})`

📘 То есть:

- `'MyClass'` — имя класса,
    
- `()` — кортеж родительских классов (от кого наследуется),
    
- `{}` — словарь атрибутов (методы, переменные класса).
    

👉 `type()` **создаёт новый класс в памяти** и возвращает его.  
Так Python делает с каждым твоим классом, даже встроенными.

---

## 🔹 Шаг 2. А `object` — это тоже класс

`object` — это **базовый класс**, с которого начинается вся иерархия.  
Он **тоже был создан** по этой схеме, только _в момент запуска интерпретатора Python_,  
а не твоим кодом.

Можно представить, что в “начале времён” Python сделал примерно так (упрощённо):

`object = type('object', (), {})`

Только это произошло **в C-коде** интерпретатора, когда Python стартовал.

---

## 🔹 Шаг 3. Почему `type` может это сделать

Потому что `type` — это **специальный встроенный класс**,  
который умеет **создавать другие классы**,  
в том числе сам `object`.

Он определён в исходниках CPython (на C) примерно как:

`PyType_Type  // это type PyBaseObject_Type  // это object`

И при инициализации Python делает примерно следующее:

1. Создаёт `object` как базовый тип.
    
2. Создаёт `type` как тип всех типов.
    
3. Связывает их между собой, чтобы получилась круговая логика:
    
    - `type(object) = type`
        
    - `type(type) = type`
        
    - `type` наследуется от `object`
        

---

## 🔁 Шаг 4. Ключевая идея

> “Создаёт” в данном контексте — это **не физическое создание руками**,  
> а **инициализация структуры класса** через метакласс `type`.

`type` знает, как в памяти оформить объект “класс”:

- имя,
    
- список атрибутов,
    
- базовые классы,
    
- методы.
    

Вот и всё. Так появился `object` — как первый класс.
