# 🎯 Декоратор `@contextmanager`

## 📌 Коротко

> `@contextmanager` из модуля `contextlib` позволяет создать контекстный менеджер через функцию-генератор. Всё **до** `yield` — это `__enter__`, всё **после** `yield` — это `__exit__`.

---

## 🔑 Шаблон / Формула

### Базовый шаблон

```python
from contextlib import contextmanager

@contextmanager
def my_context():
    # ШАГ 1 — до yield = __enter__
    # подготовка: открыть файл, сохранить состояние и т.д.
    print('Вход...')

    yield value   # ШАГ 2 — value попадает в переменную после as
                  # если as не нужен — просто yield без значения

    # ШАГ 3 — после yield = __exit__
    # завершение: закрыть файл, восстановить состояние и т.д.
    print('Выход...')


with my_context() as obj:
    # obj — это то что передали в yield
    pass
```

### Шаблон с обработкой исключений

```python
from contextlib import contextmanager

@contextmanager
def my_context():
    print('Вход...')
    try:
        yield value             # здесь выполняется тело блока with

    except SomeError as error:
        # обработать конкретное исключение
        # = return True в __exit__ (исключение подавляется)
        print(f'Ошибка: {error}')

    except:
        raise                   # все остальные исключения — пробросить дальше
                                # = return False в __exit__
    finally:
        # выполняется ВСЕГДА — была ошибка или нет
        print('Выход...')       # = завершающая логика в __exit__
```

### Соответствие классовому подходу

```
До yield          →   __enter__
yield value       →   return value из __enter__
После yield       →   __exit__
except SomeError  →   return True в __exit__ (подавить)
except: raise     →   return False в __exit__ (пробросить)
finally           →   гарантированное завершение
```

## ⚖️ Сравнение — класс vs `@contextmanager`

```python
# ── Классовый подход ──────────────────────────────
class WritableTextFile:
    def __init__(self, path):
        self.path = path

    def __enter__(self):
        self.file = open(self.path, mode='w', encoding='utf-8')
        return self.file

    def __exit__(self, exc_type, exc_value, traceback):
        self.file.close()


# ── @contextmanager ───────────────────────────────
from contextlib import contextmanager

@contextmanager
def writable_text_file(path):
    file = open(path, mode='w', encoding='utf-8')
    yield file        # ← всё до сюда = __enter__
    file.close()      # ← всё после = __exit__
```

||Класс|`@contextmanager`|
|---|---|---|
|Сложность|больше кода|компактнее|
|Многоразовый|✅ да|❌ одноразовый|
|Реентерабельный|✅ возможно|❌ нет|
|Обработка исключений|через `return True/False`|через `try/except`|
|Когда использовать|сложная логика|простые случаи|

## 💻 Примеры

### Trace — логирование

```python
@contextmanager
def trace():
    print('Начало выполнения блока with')
    try:
        yield
    except Exception as error:
        print(f'Было исключение: {error}')
    finally:
        print('Конец выполнения блока with')


with trace():
    print('Python generation!')
    print(1 / 0)

# Начало выполнения блока with
# Python generation!
# Было исключение: division by zero
# Конец выполнения блока with
```

### RedirectedStdout — временное изменение состояния

```python
import sys
from contextlib import contextmanager

@contextmanager
def redirected_stdout(new_output):
    standard_output = sys.stdout   # сохраняем
    sys.stdout = new_output        # меняем
    yield                          # работаем
    sys.stdout = standard_output   # восстанавливаем


with open('output.txt', mode='w', encoding='utf-8') as file:
    with redirected_stdout(file):
        print('Python generation!')   # → пишется в файл
    print('Возврат к экрану')         # → пишется на экран
```

### Timer — замер времени

```python
from time import perf_counter, sleep
from contextlib import contextmanager

@contextmanager
def timer():
    start = perf_counter()
    yield
    elapsed = perf_counter() - start
    print('Затраченное время:', elapsed)


with timer():
    sleep(0.7)
    sleep(1.5)
# Затраченное время: 2.208...
```

### Timer с лямбдой — отложенное вычисление

```python
@contextmanager
def timer():
    start = perf_counter()
    end = 0
    yield lambda: (start, end)   # возвращаем функцию, не значение!
    end = perf_counter()         # end заполняется после блока


with timer() as manager:
    sleep(1.4)
    print(manager())   # → (start, 0) — end ещё не заполнен

print(manager())       # → (start, end) — теперь всё заполнено
start, end = manager()
print('Затраченное время:', end - start)
```

### Несколько менеджеров в одном `with`

```python
@contextmanager
def greeter(name):
    print('Привет,', name)
    yield name
    print('Пока,', name)


with greeter('Илон') as m1, greeter('Гвидо') as m2:
    print(m1, m2)

# Привет, Илон
# Привет, Гвидо
# Илон Гвидо
# Пока, Гвидо   ← закрываются в обратном порядке!
# Пока, Илон
```

## ⚠️ Нюансы

### `yield` обязателен!

```python
# ❌ Нет yield — исключение при использовании
@contextmanager
def bad():
    print('Привет')
    return 'value'    # TypeError: 'str' object is not an iterator

# ❌ Нет yield вообще — тоже исключение
@contextmanager
def bad():
    print('Привет')
    print('Пока')     # TypeError: 'NoneType' object is not an iterator
```

### Одноразовый!

```python
# ❌ Повторное использование — AttributeError
ctx = my_context()

with ctx:
    pass

with ctx:   # AttributeError!
    pass

# ✅ Создавать заново каждый раз
with my_context():
    pass

with my_context():
    pass
```

### Порядок закрытия при нескольких менеджерах

```python
with A() as a, B() as b, C() as c:
    pass
# закрываются в обратном порядке: C → B → A
# если C подавил исключение — B и A его не увидят!
```
