# 🪄 Магические методы — шпаргалка

## 📌 Коротко

> Магические методы (dunder-методы) — методы с двойным подчёркиванием с обеих сторон (`__method__`). Python вызывает их автоматически в определённых ситуациях.

---

## 🏗️ Создание и удаление объекта

| Метод | Пример вызова | Описание |
|---|---|---|
| `__init__(self, ...)` | `Cat('Кемаль')` | Инициализация объекта после создания |
| `__new__(cls, ...)` | `Cat('Кемаль')` | Создание самого объекта (до `__init__`) |
| `__del__(self)` | `del cat` | Вызывается при удалении объекта |

---

## 🖨️ Строковое представление

| Метод | Пример вызова | Описание |
|---|---|---|
| `__str__(self)` | `str(cat)`, `print(cat)` | Неформальное представление — для человека |
| `__repr__(self)` | `repr(cat)`, в коллекции | Формальное представление — для интерпретатора |
| `__format__(self, spec)` | `f'{cat:.2f}'` | Вызывается при форматировании через f-строки / `format()` |
| `__bytes__(self)` | `bytes(cat)` | Возвращает байтовое представление объекта |

## ➕ Арифметические операторы

| Метод                       | Пример вызова | Описание              |
| --------------------------- | ------------- | --------------------- |
| `__add__(self, other)`      | `a + b`       | Сложение              |
| `__sub__(self, other)`      | `a - b`       | Вычитание             |
| `__mul__(self, other)`      | `a * b`       | Умножение             |
| `__truediv__(self, other)`  | `a / b`       | Деление               |
| `__floordiv__(self, other)` | `a // b`      | Целочисленное деление |
| `__mod__(self, other)`      | `a % b`       | Остаток от деления    |
| `__pow__(self, other)`      | `a ** b`      | Возведение в степень  |
| `__neg__(self)`             | `-a`          | Унарный минус         |
| `__pos__(self)`             | `+a`          | Унарный плюс          |
| `__abs__(self)`             | `abs(a)`      | Абсолютное значение   |

---

## 🔄 Арифметика справа (reflected)

Вызываются если левый операнд не поддерживает операцию:

| Метод | Пример вызова | Описание |
|---|---|---|
| `__radd__(self, other)` | `b + a` | Сложение (a справа) |
| `__rsub__(self, other)` | `b - a` | Вычитание (a справа) |
| `__rmul__(self, other)` | `b * a` | Умножение (a справа) |
| `__rtruediv__(self, other)` | `b / a` | Деление (a справа) |

## 🟰 Операторы сравнения

|Метод|EN|RU|Оператор|
|---|---|---|---|
|`__eq__(self, other)`|equal|равно|`a == b`|
|`__ne__(self, other)`|not equal|не равно|`a != b`|
|`__lt__(self, other)`|less than|меньше, чем|`a < b`|
|`__le__(self, other)`|less than or equal|меньше или равно|`a <= b`|
|`__gt__(self, other)`|greater than|больше, чем|`a > b`|
|`__ge__(self, other)`|greater than or equal|больше или равно|`a >= b`|

> 💡 Если определён `__eq__`, рекомендуется определить и `__hash__()`.

## 📦 Коллекции и контейнеры

| Метод | Пример вызова | Описание |
|---|---|---|
| `__len__(self)` | `len(obj)` | Длина / размер |
| `__getitem__(self, key)` | `obj[key]` | Получение элемента по ключу/индексу |
| `__setitem__(self, key, val)` | `obj[key] = val` | Установка элемента |
| `__delitem__(self, key)` | `del obj[key]` | Удаление элемента |
| `__contains__(self, item)` | `x in obj` | Проверка вхождения |
| `__iter__(self)` | `for x in obj` | Возвращает итератор |
| `__next__(self)` | `next(obj)` | Следующий элемент итератора |
| `__reversed__(self)` | `reversed(obj)` | Обратный итератор |

## 🎛️ Атрибуты объекта

| Метод | Пример вызова | Описание |
|---|---|---|
| `__getattr__(self, name)` | `obj.attr` (если не найден) | Вызывается когда атрибут не найден обычным способом |
| `__getattribute__(self, name)` | `obj.attr` (всегда) | Вызывается при любом обращении к атрибуту |
| `__setattr__(self, name, val)` | `obj.attr = val` | Установка атрибута |
| `__delattr__(self, name)` | `del obj.attr` | Удаление атрибута |
| `__dir__(self)` | `dir(obj)` | Список атрибутов и методов |

---

## 📞 Вызов объекта

| Метод | Пример вызова | Описание |
|---|---|---|
| `__call__(self, ...)` | `obj()` | Объект ведёт себя как функция |

```python
class Multiplier:
    def __init__(self, factor):
        self.factor = factor

    def __call__(self, x):
        return x * self.factor

double = Multiplier(2)
print(double(5))   # → 10
```
