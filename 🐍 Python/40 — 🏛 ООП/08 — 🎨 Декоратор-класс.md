# 🎨 Декоратор-класс

## 📌 Коротко

> Декоратор можно реализовать через класс — `__init__` принимает функцию или параметр, `__call__` добавляет логику.

---

## 📖 Теория

Декоратор-класс работает через два магических метода:

- `__init__` — вызывается при создании объекта
- `__call__` — вызывается когда объект используется как функция

---

## 💻 Примеры кода

### Вариант 1 — параметр при вызове

```python
class DecoratorFunc:
    def __init__(self, func):
        self.func = func          # принимаем функцию в __init__

    def __call__(self, name, *args, **kwargs):
        result = self.func(*args, **kwargs)
        return f'{result} {name}'  # name передаётся каждый раз при вызове


@DecoratorFunc
def hello(text):
    return text

print(hello('Яна', 'Привет'))   # → Привет Яна
# name меняется при каждом вызове
```

### Вариант 2 — параметр при создании (фиксированный)

```python
class DecoratorFunc:
    def __init__(self, name):
        self.name = name          # принимаем параметр в __init__

    def __call__(self, func):
        def wrapper(*args, **kwargs):
            result = func(*args, **kwargs)
            return f'{result} {self.name}'  # name фиксирован
        return wrapper


@DecoratorFunc('Яна')            # name фиксируется здесь
def hello(text):
    return text

print(hello('Привет'))           # → Привет Яна
# name всегда 'Яна'
```

### Как работает `@DecoratorFunc('Яна')` под капотом

```python
# @DecoratorFunc('Яна') — это два шага:

# Шаг 1 — создаётся объект с параметром
obj = DecoratorFunc('Яна')       # __init__('Яна') → self.name = 'Яна'

# Шаг 2 — объект вызывается с функцией как декоратор
hello = obj(hello)               # __call__(hello) → возвращает wrapper

# Теперь hello — это wrapper
print(hello('Привет'))           # → Привет Яна
```

---

## 🚀 Быстрая справка

||Вариант 1|Вариант 2|
|---|---|---|
|`name` передаётся|при каждом вызове|один раз при создании|
|`__init__` принимает|функцию|параметр|
|`__call__` принимает|аргументы + name|функцию|
|Когда использовать|name меняется|name фиксирован|
