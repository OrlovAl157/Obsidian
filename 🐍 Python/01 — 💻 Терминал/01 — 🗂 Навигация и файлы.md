# 🗂 Терминал — Навигация и работа с файлами

## 📌 Коротко

> Терминал — это текстовый интерфейс для управления компьютером. Mac/Linux используют bash/zsh, Windows — cmd или PowerShell.

---

## 📍 Навигация

| Что делает | Mac / Linux | Windows |
|---|---|---|
| Показать текущую папку | `pwd` | `cd` или `echo %cd%` |
| Список файлов | `ls` | `dir` |
| Список с деталями (скрытые) | `ls -la` | `dir /a` |
| Перейти в папку | `cd папка` | `cd папка` |
| Подняться на уровень выше | `cd ..` | `cd ..` |
| Перейти в домашнюю папку | `cd ~` | `cd %USERPROFILE%` |
| Открыть папку в Finder/Explorer | `open .` | `explorer .` |

---

## 📁 Файлы и папки

| Что делает | Mac / Linux | Windows |
|---|---|---|
| Создать папку | `mkdir имя` | `mkdir имя` |
| Создать пустой файл | `touch файл.txt` | `type nul > файл.txt` |
| Скопировать файл | `cp файл куда` | `copy файл куда` |
| Переместить / переименовать | `mv файл куда` | `move файл куда` |
| Удалить файл | `rm файл` | `del файл` |
| Удалить папку рекурсивно | `rm -rf папка` | `rmdir /s /q папка` |

---

## 👁 Просмотр содержимого

| Что делает | Mac / Linux | Windows |
|---|---|---|
| Показать файл целиком | `cat файл.txt` | `type файл.txt` |
| Показать постранично | `less файл.txt` | `more файл.txt` |
| Первые N строк | `head -n 10 файл` | — |
| Последние N строк | `tail -n 10 файл` | — |

---

## 🔍 Поиск

| Что делает | Mac / Linux | Windows |
|---|---|---|
| Найти файл по имени | `find . -name "*.py"` | `dir /s *.py` |
| Найти текст в файле | `grep "текст" файл` | `findstr "текст" файл` |
| Рекурсивный поиск текста | `grep -r "текст" .` | `findstr /s "текст" *.*` |
