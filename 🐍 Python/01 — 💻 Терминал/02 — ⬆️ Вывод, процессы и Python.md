# ⬆️ Терминал — Вывод, процессы и Python

## 📌 Коротко

> Вторая часть памятки по терминалу: перенаправление вывода, процессы, работа с Python и venv, шорткаты.

---

## 📤 Вывод и перенаправление

| Что делает | Mac / Linux | Windows |
|---|---|---|
| Вывести текст | `echo Привет` | `echo Привет` |
| Записать вывод в файл | `echo текст > файл` | `echo текст > файл` |
| Добавить в конец файла | `echo текст >> файл` | `echo текст >> файл` |
| Передать вывод другой команде | `cmd1 \| cmd2` | `cmd1 \| cmd2` |

---

## ⚙️ Процессы

| Что делает | Mac / Linux | Windows |
|---|---|---|
| Список процессов | `ps aux` | `tasklist` |
| Завершить процесс по PID | `kill PID` | `taskkill /PID номер /F` |
| Прервать текущую команду | `Ctrl + C` | `Ctrl + C` |

---

## 🐍 Python и pip

| Что делает | Mac / Linux | Windows |
|---|---|---|
| Запустить скрипт | `python3 файл.py` | `python файл.py` |
| Версия Python | `python3 --version` | `python --version` |
| Установить пакет | `pip3 install пакет` | `pip install пакет` |
| Список пакетов | `pip3 list` | `pip list` |

---

## 🌿 Виртуальное окружение (venv)

| Что делает | Mac / Linux | Windows |
|---|---|---|
| Создать venv | `python3 -m venv venv` | `python -m venv venv` |
| Активировать | `source venv/bin/activate` | `venv\Scripts\activate` |
| Деактивировать | `deactivate` | `deactivate` |

> 💡 После активации в начале строки появится `(venv)` — это значит окружение активно

---

## ⌨️ Шорткаты и полезное

| Что делает | Mac / Linux | Windows |
|---|---|---|
| Очистить экран | `clear` | `cls` |
| История команд | `history` | `doskey /history` |
| Автодополнение | `Tab` | `Tab` |
| Предыдущая команда | `↑` | `↑` |
| Поиск по истории | `Ctrl + R` | — |
| Справка команды | `man команда` | `команда /?` |
| Где находится программа | `which python3` | `where python` |
| Переменные окружения | `echo $PATH` | `echo %PATH%` |
