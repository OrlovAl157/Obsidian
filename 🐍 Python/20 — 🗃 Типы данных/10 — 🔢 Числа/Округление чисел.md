# ⚖️ Округление чисел в Python

## 📌 Коротко

> Способы округления чисел в Python: round(), math.floor(), math.ceil(), math.trunc(), форматирование строк и Decimal для точных расчётов. Важно: round() использует банковское округление.

---

## 📖 Теория

### Основы

В Python существует несколько способов округления чисел, каждый из которых подходит для разных задач:

- **round()** — округление к ближайшему целому (банковское округление)
- **math.floor()** — округление всегда вниз к -∞
- **math.ceil()** — округление всегда вверх к +∞
- **math.trunc()** — отбросить дробную часть (округление к 0)
- **f-строки / format()** — округление при форматировании (возвращает строку)
- **Decimal** — точное округление для финансовых расчётов

### Быстрая справка

```python
# Базовое округление
round(3.14159)          # 3 (до целого)
round(3.14159, 2)       # 3.14 (два знака после запятой)
round(1234.5678, -1)    # 1230.0 (до десятков)

# Всегда вверх/вниз
import math
math.ceil(3.1)          # 4 (всегда вверх)
math.floor(3.9)         # 3 (всегда вниз)
math.trunc(3.9)         # 3 (отбросить дробную часть)

# ⚠️ ВАЖНО: round() использует банковское округление
round(2.5)              # 2 (к ближайшему четному!)
round(3.5)              # 4 (к ближайшему четному!)
```

### Когда что использовать

```python
round()         # Обычные задачи
math.ceil()     # Всегда округлить ВВЕРХ
math.floor()    # Всегда округлить ВНИЗ
f"{x:.2f}"      # Нужна СТРОКА с округлением
Decimal         # Финансовые расчеты, точность критична
```

## 💻 Примеры кода

### Функция round()

```python
# Базовое округление
round(3.14159)       # → 3
round(3.7)           # → 4
round(-3.7)          # → -4

# С указанием количества знаков
round(3.14159, 2)    # → 3.14
round(3.14159, 3)    # → 3.142
round(123.456, 1)    # → 123.5

# Округление до десятков, сотен и т.д.
round(1234.5678, -1) # → 1230.0 (до десятков)
round(1234.5678, -2) # → 1200.0 (до сотен)
round(1234.5678, -3) # → 1000.0 (до тысяч)

# ⚠️ Банковское округление (round half to even)
# При округлении .5 выбирается ближайшее ЧЁТНОЕ число
round(0.5)           # → 0 (к чётному!)
round(1.5)           # → 2 (к чётному!)
round(2.5)           # → 2 (к чётному!)
round(3.5)           # → 4 (к чётному!)
round(4.5)           # → 4 (к чётному!)

# Почему банковское округление?
# Уменьшает систематическую ошибку накопления
# при множественных округлениях
```

### Модуль math

```python
import math

# math.floor() - округление вниз к -∞
math.floor(3.1)   # → 3
math.floor(3.9)   # → 3
math.floor(-3.1)  # → -4 (вниз к -∞!)
math.floor(-3.9)  # → -4

# math.ceil() - округление вверх к +∞
math.ceil(3.1)    # → 4
math.ceil(3.9)    # → 4
math.ceil(-3.1)   # → -3 (вверх к +∞!)
math.ceil(-3.9)   # → -3

# math.trunc() - отбросить дробную часть (к нулю)
math.trunc(3.1)   # → 3
math.trunc(3.9)   # → 3
math.trunc(-3.1)  # → -3 (к нулю!)
math.trunc(-3.9)  # → -3
```

### Форматирование строк

```python
pi = 3.14159

# f-строки
f"{pi:.2f}"         # → '3.14' (возвращает строку!)
f"{pi:.0f}"         # → '3'
f"{pi:.5f}"         # → '3.14159'

# format()
"{:.2f}".format(pi) # → '3.14'
"{:.0f}".format(pi) # → '3'

# Форматирование не меняет исходное число
x = 3.14159
formatted = f"{x:.2f}"  # → '3.14' (строка)
x                       # → 3.14159 (не изменилось)

# Для получения числа нужно преобразовать обратно
rounded = float(f"{x:.2f}")  # → 3.14 (число)
```

### Decimal для точных расчётов

```python
from decimal import Decimal, ROUND_HALF_UP, ROUND_DOWN, ROUND_UP

# ✅ Правильное создание Decimal (из строки)
x = Decimal("2.5")
y = Decimal("3.5")

# Банковское округление (по умолчанию)
x.quantize(Decimal('1'))    # → Decimal('2')
y.quantize(Decimal('1'))    # → Decimal('4')

# Математическое округление (всегда от нуля при .5)
Decimal('2.5').quantize(Decimal('1'), rounding=ROUND_HALF_UP)  # → Decimal('3')
Decimal('3.5').quantize(Decimal('1'), rounding=ROUND_HALF_UP)  # → Decimal('4')

# Округление вниз
Decimal('3.9').quantize(Decimal('1'), rounding=ROUND_DOWN)     # → Decimal('3')

# Округление вверх
Decimal('3.1').quantize(Decimal('1'), rounding=ROUND_UP)       # → Decimal('4')

# Округление до N знаков
price = Decimal("19.999")
price.quantize(Decimal("0.01"), rounding=ROUND_HALF_UP)  # → Decimal('20.00')

# Для денег
total = Decimal("59.974")
total.quantize(Decimal("0.01"), ROUND_HALF_UP)  # → Decimal('59.97')
```

### Практические примеры

#### Округление до кратного

```python
def round_to_multiple(number, multiple):
    """Округлить до ближайшего кратного"""
    return round(number / multiple) * multiple

round_to_multiple(23, 5)        # → 25
round_to_multiple(22, 5)        # → 20
round_to_multiple(127, 10)      # → 130
round_to_multiple(153, 25)      # → 150

def round_up_to_multiple(number, multiple):
    """Округлить вверх до кратного"""
    return ((number + multiple - 1) // multiple) * multiple

round_up_to_multiple(21, 5)     # → 25
round_up_to_multiple(25, 5)     # → 25
round_up_to_multiple(26, 5)     # → 30
```

#### Округление цены

```python
def round_price(price, precision=0.01):
    """Округлить цену до центов"""
    return round(price / precision) * precision

round_price(19.999)             # → 20.0
round_price(19.994)             # → 19.99
round_price(19.996)             # → 20.0

# С использованием Decimal (рекомендуется для денег)
from decimal import Decimal, ROUND_HALF_UP

def round_price_decimal(price_str):
    """Округлить цену с Decimal"""
    price = Decimal(price_str)
    return price.quantize(Decimal("0.01"), ROUND_HALF_UP)

round_price_decimal("19.999")   # → Decimal('20.00')
round_price_decimal("19.994")   # → Decimal('19.99')
```

#### Округление процентов

```python
def round_percentage(value, decimals=1):
    """Округлить процент"""
    return round(value, decimals)

round_percentage(15.678, 1)     # → 15.7
round_percentage(15.678, 0)     # → 16.0
round_percentage(15.678, 2)     # → 15.68

# С форматированием
def format_percentage(value):
    """Отформатировать как процент"""
    return f"{value:.1f}%"

format_percentage(15.678)       # → "15.7%"
```

#### Округление до значащих цифр

```python
def round_to_n_significant(number, n):
    """Округлить до N значащих цифр"""
    if number == 0:
        return 0
    from math import log10, floor
    return round(number, -int(floor(log10(abs(number)))) + (n - 1))

round_to_n_significant(1234.5678, 3)  # → 1230.0
round_to_n_significant(0.0012345, 3)  # → 0.00123
round_to_n_significant(1234567, 2)    # → 1200000.0
```

### Частые ошибки

```python
# ❌ Ожидание обычного округления от round()
round(2.5)              # → 2 (не 3! Банковское округление)
round(3.5)              # → 4

# ✅ Для обычного округления используйте Decimal
from decimal import Decimal, ROUND_HALF_UP
Decimal('2.5').quantize(Decimal('1'), ROUND_HALF_UP)  # → Decimal('3')

# ❌ Множественное округление накапливает ошибки
x = 2.5
x = round(x * 1.1, 2)   # → 2.75
x = round(x * 1.1, 2)   # → 3.03 (может накопиться ошибка)

# ✅ Округляйте только конечный результат
x = 2.5
x = x * 1.1 * 1.1       # → 3.025
x = round(x, 2)         # → 3.03

# ❌ Создание Decimal из float
from decimal import Decimal
Decimal(0.1)            # → Decimal('0.1000000000000000055...')

# ✅ Создавайте Decimal из строки
Decimal("0.1")          # → Decimal('0.1')

# ⚠️ Округление отрицательных чисел
math.floor(-3.5)        # → -4 (не -3!)
math.ceil(-3.5)         # → -3 (не -4!)
```

### Выбор метода округления

**Используйте round() когда:**
- Обычные задачи без особых требований
- Банковское округление подходит (уменьшает систематическую ошибку)
- Не критична точность при .5

**Используйте math.ceil() когда:**
- Нужно ВСЕГДА округлить вверх
- Расчёт необходимого количества (упаковок, страниц и т.д.)
- Запас прочности в расчётах

**Используйте math.floor() когда:**
- Нужно ВСЕГДА округлить вниз
- Расчёт максимального целого в пределах лимита
- Деление с остатком

**Используйте f-строки / format() когда:**
- Нужна строка, а не число
- Форматирование для вывода
- Округление только для отображения

**Используйте Decimal когда:**
- Финансовые расчёты
- Деньги, проценты, налоги
- Критична точность до копейки
- Нужно обычное математическое округление при .5

```python
# Пример выбора метода

# Количество упаковок (всегда округлить вверх)
import math
items = 23
per_pack = 10
packs_needed = math.ceil(items / per_pack)  # → 3

# Форматирование цены для вывода
price = 19.999
display_price = f"${price:.2f}"  # → "$20.00"

# Точные финансовые расчёты
from decimal import Decimal, ROUND_HALF_UP
subtotal = Decimal("19.99")
tax_rate = Decimal("0.15")
tax = (subtotal * tax_rate).quantize(Decimal("0.01"), ROUND_HALF_UP)
total = subtotal + tax
```
