# ✨ f-строки (Formatted String Literals)

## 📌 Коротко

> Самый современный, быстрый и читаемый способ форматирования строк в Python (с версии 3.6+). Позволяют встраивать выражения прямо в строку через `f"{переменная}"`.

---

## 💡 Полезные советы

- **f-строки быстрее** чем `.format()` и `%` форматирование
- **Читабельнее** — переменные видны прямо в строке
- **Выражения внутри** — можно использовать любой код Python: `f"{x + y}"`
- **Отладка через `=`** — `f"{x=}"` выводит `x=10` (Python 3.8+)
- **Форматирование** — `f"{value:спецификатор}"` для управления форматом
- **Python 3.6+** — требуется современная версия
- **Нельзя `\` внутри {}** — используйте переменную вместо `f"{'\n'}"`
- **Для статичных строк не нужна f** — `"Hello"` быстрее чем `f"Hello"`

## 🚀 Быстрая справка

| Задача | Синтаксис | Пример |
|--------|-----------|--------|
| Базовая подстановка | `f"{var}"` | `f"Hello {name}"` |
| Выражения | `f"{expr}"` | `f"{x + y}"` |
| Округление float | `f"{x:.2f}"` | `f"{3.14159:.2f}"` → `"3.14"` |
| Разделители тысяч | `f"{n:,}"` | `f"{1000000:,}"` → `"1,000,000"` |
| Процент | `f"{x:.1%}"` | `f"{0.756:.1%}"` → `"75.6%"` |
| Дополнение нулями | `f"{n:05d}"` | `f"{42:05d}"` → `"00042"` |
| Выравнивание влево | `f"{s:<10}"` | `f"{'Hi':<10}"` → `"Hi        "` |
| Выравнивание вправо | `f"{s:>10}"` | `f"{'Hi':>10}"` → `"        Hi"` |
| Выравнивание по центру | `f"{s:^10}"` | `f"{'Hi':^10}"` → `"    Hi    "` |
| Знак числа | `f"{n:+}"` | `f"{42:+}"` → `"+42"` |
| Двоичная система | `f"{n:b}"` | `f"{10:b}"` → `"1010"` |
| Hex система | `f"{n:x}"` | `f"{255:x}"` → `"ff"` |
| Отладка (3.8+) | `f"{x=}"` | `f"{x=}"` → `"x=10"` |

---

## 📖 Теория

### Основы

**f-строки** (formatted string literals) — строковые литералы с префиксом `f` или `F`, внутри которых можно использовать выражения Python в фигурных скобках `{}`.

**Ключевые особенности:**
- ✅ **Производительность** — быстрее других методов форматирования
- ✅ **Читабельность** — код понятнее, переменные видны сразу
- ✅ **Выражения** — любой валидный Python код внутри `{}`
- ✅ **Форматирование** — богатые возможности через спецификаторы
- ⚠️ **Python 3.6+** — не работает в старых версиях

### Синтаксис

```python
# Базовый синтаксис
f"текст {переменная} текст"
f'текст {переменная} текст'

# С форматированием
f"текст {переменная:спецификатор} текст"

# Многострочные
f"""
текст {переменная}
ещё текст
"""

# Примеры
name = "John"
age = 25
f"Меня зовут {name}, мне {age} лет"  # → "Меня зовут John, мне 25 лет"
f"{3.14159:.2f}"                     # → "3.14"
```

## 💻 Примеры кода

### Базовое использование

```python
# Простая подстановка
name = "John"
age = 25
f"Hello, {name}!"               # → "Hello, John!"
f"I am {age} years old"         # → "I am 25 years old"

# Выражения
x = 10
y = 5
f"{x} + {y} = {x + y}"          # → "10 + 5 = 15"
f"{x} * {y} = {x * y}"          # → "10 * 5 = 50"

# Вызов методов
text = "hello"
f"{text.upper()}"               # → "HELLO"
f"{text.capitalize()}"          # → "Hello"

# Вызов функций
numbers = [1, 2, 3, 4, 5]
f"Length: {len(numbers)}"       # → "Length: 5"
f"Sum: {sum(numbers)}"          # → "Sum: 15"

# Индексация
items = ["apple", "banana", "cherry"]
f"First: {items[0]}"            # → "First: apple"
f"Last: {items[-1]}"            # → "Last: cherry"

# Словари
user = {"name": "Bob", "age": 28}
f"{user['name']} is {user['age']}"  # → "Bob is 28"
```

### Форматирование чисел

```python
pi = 3.14159265359

# Округление float
f"{pi:.2f}"                     # → "3.14"
f"{pi:.4f}"                     # → "3.1416"
f"{pi:.0f}"                     # → "3"

# Научная нотация
large = 1234567.89
f"{large:.2e}"                  # → "1.23e+06"
f"{large:.3E}"                  # → "1.235E+06"

# Разделители тысяч
big_number = 1234567890
f"{big_number:,}"               # → "1,234,567,890"
f"{big_number:_}"               # → "1_234_567_890"

# Проценты
ratio = 0.7562
f"{ratio:.1%}"                  # → "75.6%"
f"{ratio:.2%}"                  # → "75.62%"

# Знак числа
f"{42:+}"                       # → "+42"
f"{-42:+}"                      # → "-42"
f"{0:+}"                        # → "+0"
```

### Выравнивание и заполнение

```python
text = "Hello"

# Выравнивание
f"{text:<10}"                   # → "Hello     " (влево)
f"{text:>10}"                   # → "     Hello" (вправо)
f"{text:^10}"                   # → "  Hello   " (центр)

# С заполнителем
f"{text:*<10}"                  # → "Hello*****"
f"{text:*>10}"                  # → "*****Hello"
f"{text:*^10}"                  # → "**Hello***"

# Дополнение нулями
number = 42
f"{number:05}"                  # → "00042"
f"{number:010}"                 # → "0000000042"

# Динамическая ширина
width = 10
f"{text:<{width}}"              # → "Hello     "

# Динамическая точность
precision = 3
f"{pi:.{precision}f}"           # → "3.142"
```

### Системы счисления

```python
num = 42

# Двоичная
f"{num:b}"                      # → "101010"
f"{num:#b}"                     # → "0b101010" (с префиксом)
f"{num:08b}"                    # → "00101010"

# Восьмеричная
f"{num:o}"                      # → "52"
f"{num:#o}"                     # → "0o52"

# Шестнадцатеричная
f"{num:x}"                      # → "2a" (строчные)
f"{num:X}"                      # → "2A" (заглавные)
f"{num:#x}"                     # → "0x2a"

# Пример
color = 0xFF5733
f"Color: #{color:06X}"          # → "Color: #FF5733"
```

### Отладка (Python 3.8+)

```python
# Оператор = для отладки
x = 10
y = 20
f"{x=}"                         # → "x=10"
f"{y=}"                         # → "y=20"
f"{x + y=}"                     # → "x + y=30"

# С вычислениями
numbers = [1, 2, 3, 4, 5]
f"{sum(numbers)=}"              # → "sum(numbers)=15"
f"{len(numbers)=}"              # → "len(numbers)=5"

# С форматированием
pi = 3.14159
f"{pi=:.2f}"                    # → "pi=3.14"

# Множественные переменные
name = "Alice"
age = 30
print(f"{name=}, {age=}")       # → "name='Alice', age=30"
```

### Практические примеры

#### Таблица

```python
products = [
    ("Apple", 150, 2.50),
    ("Banana", 200, 1.80),
    ("Cherry", 75, 5.20)
]

print(f"{'Product':<10} {'Qty':>5} {'Price':>8}")
print("-" * 25)
for name, qty, price in products:
    print(f"{name:<10} {qty:>5} {price:>8.2f}")

# Вывод:
# Product       Qty    Price
# -------------------------
# Apple         150     2.50
# Banana        200     1.80
# Cherry         75     5.20
```

#### Прогресс бар

```python
def progress_bar(current, total, width=50):
    percent = current / total
    filled = int(width * percent)
    bar = "█" * filled + "░" * (width - filled)
    return f"{bar} {percent:.1%} ({current}/{total})"

print(progress_bar(75, 100))
# → "█████████████████████████████████████░░░░░░░░░░░░░░░ 75.0% (75/100)"
```

#### Форматирование времени

```python
from datetime import datetime

now = datetime.now()
f"{now:%Y-%m-%d}"               # → "2024-10-03"
f"{now:%H:%M:%S}"               # → "14:30:45"
f"Current: {now:%Y-%m-%d %H:%M:%S}"
```

#### Условные выражения

```python
# Тернарный оператор
age = 18
f"You are {'adult' if age >= 18 else 'minor'}"  # → "You are adult"

score = 85
f"Grade: {'Pass' if score >= 60 else 'Fail'}"  # → "Grade: Pass"
```

### Частые ошибки

```python
# ❌ Обратный слэш внутри {}
# f"{'\n'}"                     # SyntaxError!

# ✅ Используйте переменную
newline = '\n'
f"{newline}"                    # ✅

# ❌ Пустые {}
# f"{}"                         # SyntaxError!

# ⚠️ f-строки вычисляются сразу
x = 10
s = f"{x}"                      # s = "10"
x = 20
print(s)                        # → "10" (не "20"!)

# ⚠️ Для статичных строк не нужна f
# Медленнее
f"Hello, World!"
# Быстрее
"Hello, World!"
```

### Сравнение со старыми методами

```python
name = "John"
age = 25

# ✅ f-строка (современный способ)
f"Hello, {name}! Age: {age}"

# ⏺ str.format() (старый)
"Hello, {}! Age: {}".format(name, age)

# ⏺ % форматирование (очень старый)
"Hello, %s! Age: %d" % (name, age)

# ❌ Конкатенация (плохой способ)
"Hello, " + name + "! Age: " + str(age)

# f-строки быстрее и читабельнее!
```
