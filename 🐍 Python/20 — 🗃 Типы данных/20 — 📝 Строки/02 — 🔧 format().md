# 🔄 Метод format()

## 📌 Коротко

> Универсальный метод форматирования строк через `"текст {позиция:формат}".format(значения)`. Для Python 3.6+ рекомендуется использовать f-строки.

---

## 💡 Полезные советы

- **Гибкость** — можно использовать позиции, имена, атрибуты
- **Повторное использование** — можно ссылаться на один аргумент много раз
- **Шаблоны** — удобно для многократного использования одного формата
- **f-строки быстрее** — для Python 3.6+ лучше использовать f-строки
- **Нельзя смешивать** пустые `{}` и индексы `{0}` в одной строке
- **Безопасность** — не использует eval, безопаснее чем `%`

## 🚀 Быстрая справка

| Задача | Синтаксис | Пример |
|--------|-----------|--------|
| Базовая подстановка | `"{}".format(value)` | `"Hello {}".format("World")` |
| По позиции | `"{0} {1}".format(a, b)` | `"{0} {1}".format("Hi", "Bob")` |
| По имени | `"{name}".format(name=val)` | `"{name}".format(name="Alice")` |
| Округление | `"{:.2f}".format(num)` | `"{:.2f}".format(3.14159)` → `"3.14"` |
| Ширина поля | `"{:10}".format(text)` | `"{:10}".format("Hi")` → `"Hi        "` |
| Дополнение нулями | `"{:05d}".format(num)` | `"{:05d}".format(42)` → `"00042"` |
| Выравнивание влево | `"{:<10}".format(text)` | `"{:<10}".format("Hi")` → `"Hi        "` |
| Выравнивание вправо | `"{:>10}".format(text)` | `"{:>10}".format("Hi")` → `"        Hi"` |
| По центру | `"{:^10}".format(text)` | `"{:^10}".format("Hi")` → `"    Hi    "` |
| Разделитель тысяч | `"{:,}".format(num)` | `"{:,}".format(1000000)` → `"1,000,000"` |
| Процент | `"{:.1%}".format(ratio)` | `"{:.1%}".format(0.756)` → `"75.6%"` |

## 📖 Теория

### Основы

**str.format()** — метод форматирования строк, который использует фигурные скобки `{}` как placeholder'ы для значений.

**💡 Ключевые особенности:**
- ✅ **Универсальность** — работает с любыми типами данных
- ✅ **Гибкость** — позиции, имена, атрибуты объектов
- ✅ **Совместимость** — Python 2.7+ и 3.x
- ⚠️ **Медленнее f-строк** — для Python 3.6+ лучше использовать f-строки

### Синтаксис

```python
# Базовый
"текст {позиция} текст".format(значение)

# С форматированием
"текст {позиция:формат} текст".format(значение)

# Полная спецификация
"{[index/name]:[fill][align][sign][#][0][width][,][.precision][type]}"

# Примеры
"Hello, {}!".format("World")                      # → "Hello, World!"
"{0} {1}".format("Hello", "World")                # → "Hello World"
"{name} is {age}".format(name="Alice", age=25)    # → "Alice is 25"
"{:.2f}".format(3.14159)                          # → "3.14"
```

## 💻 Примеры кода

### Базовое использование

```python
# Без параметров (по порядку)
"Hello, {}!".format("World")    # → "Hello, World!"
"{} + {} = {}".format(2, 3, 5)  # → "2 + 3 = 5"

# По позиции
"{0} {1}".format("Hello", "World")  # → "Hello World"
"{1} {0}".format("Hello", "World")  # → "World Hello"

# Повторное использование
"{0} {0} {0}".format("ha")      # → "ha ha ha"

# По имени
"{name} is {age}".format(name="Alice", age=25)
# → "Alice is 25"

# Из словаря
user = {"name": "Bob", "age": 30}
"{name} is {age}".format(**user)  # → "Bob is 30"
```

### Форматирование чисел

```python
# Целые числа
"{:d}".format(42)               # → "42"
"{:5d}".format(42)              # → "   42"
"{:05d}".format(42)             # → "00042"

# Знак
"{:+d}".format(42)              # → "+42"
"{: d}".format(42)              # → " 42"

# Системы счисления
"{:b}".format(10)               # → "1010"
"{:o}".format(10)               # → "12"
"{:x}".format(255)              # → "ff"
"{:#x}".format(255)             # → "0xff"

# Разделители
"{:,}".format(1000000)          # → "1,000,000"
"{:_}".format(1000000)          # → "1_000_000"

# Вещественные
"{:.2f}".format(3.14159)        # → "3.14"
"{:8.2f}".format(42.5678)       # → "   42.57"
"{:,.2f}".format(1234567.89)    # → "1,234,567.89"

# Проценты
"{:.1%}".format(0.756)          # → "75.6%"

# Научная нотация
"{:.2e}".format(1234.5)         # → "1.23e+03"
```

### Выравнивание и заполнение

```python
text = "Hello"

# Выравнивание
"{:<10}".format(text)           # → "Hello     "
"{:>10}".format(text)           # → "     Hello"
"{:^10}".format(text)           # → "  Hello   "

# С заполнителем
"{:*<10}".format(text)          # → "Hello*****"
"{:*>10}".format(text)          # → "*****Hello"
"{:*^10}".format(text)          # → "**Hello***"

# Для чисел
"{:05d}".format(42)             # → "00042"
"{:*>10d}".format(42)           # → "********42"
```

### Доступ к атрибутам и элементам

```python
# Атрибуты объекта
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

person = Person("Alice", 25)
"{0.name} is {0.age}".format(person)
# → "Alice is 25"

# Элементы списка
data = [10, 20, 30]
"{0[0]} {0[1]} {0[2]}".format(data)  # → "10 20 30"

# Элементы словаря
user = {"name": "Bob", "age": 30}
"{0[name]} is {0[age]}".format(user)  # → "Bob is 30"
```

### Динамическая ширина и точность

```python
# Динамическая ширина
width = 10
"{text:{width}}".format(text="Hello", width=width)
# → "Hello     "

# Динамическая точность
num = 3.14159
precision = 2
"{num:.{prec}f}".format(num=num, prec=precision)
# → "3.14"
```

### Практические примеры

#### Таблица

```python
products = [
    ("Apple", 150, 2.50),
    ("Banana", 200, 1.80),
    ("Cherry", 75, 5.20)
]

print("{:<10} {:>5} {:>8}".format("Product", "Qty", "Price"))
print("-" * 25)

for name, qty, price in products:
    print("{:<10} {:>5} {:>8.2f}".format(name, qty, price))

# Вывод:
# Product       Qty    Price
# -------------------------
# Apple         150     2.50
# Banana        200     1.80
# Cherry         75     5.20
```

#### Форматирование денег

```python
def format_money(amount, currency="₽"):
    return "{:,.2f} {}".format(amount, currency)

format_money(1234567.89)        # → "1,234,567.89 ₽"
format_money(999.5, "$")        # → "999.50 $"
```

#### Форматирование времени

```python
hours = 2
minutes = 35
seconds = 8

"{:02d}:{:02d}:{:02d}".format(hours, minutes, seconds)
# → "02:35:08"
```

### Частые ошибки

```python
# ❌ Смешивание пустых {} и индексов
# "{} {1}".format(1, 2)         # ValueError!

# ✅ Либо все пустые, либо все с индексами
"{} {}".format(1, 2)            # ✅
"{0} {1}".format(1, 2)          # ✅

# ❌ Неправильный порядок спецификаторов
# "{:5.2>f}".format(3.14)       # ValueError!

# ✅ Правильный порядок
"{:>5.2f}".format(3.14)         # ✅

# ❌ Несуществующий индекс
# "{0} {1}".format("one")       # IndexError!

# ✅ Все аргументы
"{0} {1}".format("one", "two")  # ✅
```

### Когда использовать

**✅ Используйте format() когда:**
- Поддерживаете Python 2.7 - 3.5
- Шаблон строки используется много раз
- Нужна совместимость со старым кодом

**✅ Используйте f-строки когда:**
- Python 3.6+ доступен (рекомендуется!)
- Нужна скорость
- Нужна читаемость

```python
name = "Alice"
age = 25

# format()
"Name: {}, Age: {}".format(name, age)

# f-строка (ЛУЧШЕ для Python 3.6+)
f"Name: {name}, Age: {age}"

# Но format() полезна для шаблонов:
template = "Hello, {name}! You are {age}."
print(template.format(name="Alice", age=25))
print(template.format(name="Bob", age=30))
```
