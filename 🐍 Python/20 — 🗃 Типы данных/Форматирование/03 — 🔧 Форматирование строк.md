## Справка по способам

| Способ | Синтаксис | Примеры | Плюсы |
| --- | --- | --- | --- |
| `%` (старый) | `"%(name)s" % {"name": "Alice"}` | `"%d %s %f" % (5, "test", 3.14)` | простой, быстрый, легко читается |
| `.format()` | `"{name}".format(name="Alice")` | `"{} {}".format("Hello", "World")` | гибкий, встроенные методы |
| f-строки | `f"{name}"` | `f"Hello {name}, age {age}"` | читаемый, быстрый, интерполяция |

## % Старое форматирование (%-style)

**Назначение:** встроить значения переменных в строку, используя символ `%` и спецификаторы типов.

**Спецификаторы типов:**

| Спец | Тип | Пример | Результат |
| --- | --- | --- | --- |
| `%s` | строка (любой объект) | `"Hello %s" % "World"` | `"Hello World"` |
| `%d` | целое число (int) | `"Count: %d" % 42` | `"Count: 42"` |
| `%i` | целое число (int, синоним `%d`) | `"Count: %i" % 42` | `"Count: 42"` |
| `%f` | число с плавающей точкой | `"Price: %f" % 3.14` | `"Price: 3.140000"` |
| `%x` | шестнадцатеричное число | `"Hex: %x" % 255` | `"Hex: ff"` |
| `%o` | восьмеричное число | `"Oct: %o" % 8` | `"Oct: 10"` |
| `%e` | экспоненциальная запись | `"%e" % 1000` | `"1.000000e+03"` |
| `%%` | литеральный `%` | `"100%%"` | `"100%"` |

**Примеры базового использования:**

```python
# Одно значение
print("Hello %s!" % "Alice")           # Hello Alice!
print("Age: %d years" % 25)            # Age: 25 years
print("Price: %.2f" % 19.9)            # Price: 19.90

# Несколько значений (кортеж)
print("%s is %d years old" % ("Bob", 30))  # Bob is 30 years old

# Проценты
print("Success: 95.5%%")               # Success: 95.5%
```

**Важное:** при множественных значениях справа от `%` должен быть кортеж. Если одно значение — скобки не обязательны, но при ошибке `TypeError` обычно помогает обернуть в `()`.

## 🆚 Формат % — именованные поля

**Назначение:** вставить значения через имена, а не порядковые номера. Правая сторона — словарь `{}`.

**Синтаксис:** `"%(имя)спец" % {"имя": значение}`

**Примеры:**

```python
# Простой пример
user = {"name": "Alice", "age": 25}
print("%(name)s is %(age)d years old" % user)
# Alice is 25 years old

# С несколькими вхождениями
template = "%(name)s works at %(company)s and %(name)s likes Python"
data = {"name": "Bob", "company": "TechCorp"}
print(template % data)
# Bob works at TechCorp and Bob likes Python
```

**С форматированием чисел:**

```python
data = {"price": 19.9, "quantity": 5}
print("Price: %(price).2f, Qty: %(quantity)03d" % data)
# Price: 19.90, Qty: 005
```

**Важное:** при использовании именованных полей справа **обязательно** словарь. Если случайно передашь кортеж — будет `TypeError: not enough arguments for format string`.

---

## 🆚 Формат % — позиционные поля

**Назначение:** встроить значения по порядку их появления. Правая сторона — кортеж или единственное значение.

**Синтаксис:** `"спец спец спец" % (знач1, знач2, знач3)`

**Примеры:**

```python
# Два значения
print("%s is %d years old" % ("Alice", 25))
# Alice is 25 years old

# Три значения
print("%d + %d = %d" % (2, 3, 5))
# 2 + 3 = 5

# Одно значение (скобки опциональны)
print("Hello %s" % "World")
# Hello World

# Одно значение из коллекции (скобки обязательны!)
data = ("Alice",)
print("Name: %s" % data)  # ✅ Name: Alice
# print("Name: %s" % data[0])  # ❌ TypeError
```

**С форматированием:**

```python
print("%.2f" % 3.14159)           # 3.14
print("%05d" % 42)                # 00042
print("%e" % 1000000)             # 1.000000e+06
```

**Важное:** при одном значении нужны скобки `(знач,)`, иначе Python может неправильно интерпретировать кортеж. При ошибке `TypeError: not all arguments converted during string formatting` — обычно забыли `%`.

## .format() Новое форматирование

**Назначение:** более гибкое форматирование, встроено с Python 2.6+. Явно передаёшь параметры методу `.format()`.

**Синтаксис:**

```python
"{} и {}".format(знач1, знач2)           # позиционные
"{name} и {age}".format(name="Alice", age=25)  # именованные
"{0} и {1}".format(знач1, знач2)        # явные индексы
```

**Примеры:**

```python
# Позиционные
print("Hello {} and {}".format("Alice", "Bob"))
# Hello Alice and Bob

# Именованные
print("{name} is {age} years old".format(name="Alice", age=25))
# Alice is 25 years old

# Явные индексы
print("{1}, {0}, {1}".format("A", "B"))
# B, A, B

# Форматирование
print("{:.2f}".format(3.14159))          # 3.14
print("{:05d}".format(42))               # 00042
print("{:>10}".format("right"))          # "     right"
```

**С методом (мини-язык форматирования):**

```python
# Тип выравнивания
print("{:<10}Left".format("text"))       # text      Left
print("{:^10}Center".format("text"))     # "  text   Center"
print("{:>10}Right".format("text"))      # "     text Right"

# Заполнение символом
print("{:*^10}".format("text"))          # "***text***"

# Число знаков после запятой
print("{:.3f}".format(3.14159))          # 3.142
```

**Важное:** `.format()` универсален, но требует больше кода. Старый `%` часто проще для простых случаев.

## f-строки Современный способ

**Назначение:** встроить переменные прямо в строку, используя `f"..."`. Самый быстрый и читаемый способ (Python 3.6+).

**Синтаксис:** `f"текст {переменная} текст"`

**Примеры:**

```python
name = "Alice"
age = 25

# Простая вставка
print(f"Hello {name}")
# Hello Alice

# Выражение в скобках
print(f"{name} is {age} years old")
# Alice is 25 years old

# Вычисления
x, y = 2, 3
print(f"{x} + {y} = {x + y}")
# 2 + 3 = 5

# Форматирование
price = 19.9
print(f"Price: {price:.2f}")
# Price: 19.90

# Выравнивание
print(f"{name:<15} | {name:>15}")
# Alice           |           Alice
```

**С вызовом методов:**

```python
name = "alice"
print(f"Name: {name.upper()}")
# Name: ALICE

words = ["hello", "world"]
print(f"Words: {', '.join(words)}")
# Words: hello, world
```

**Важное:** f-строки вычисляют выражения в скобках **во время создания строки**. Это быстрее всех других способов. Нельзя использовать сырые (raw) f-строки `fr"..."` и f-строки с backslash: `f"{'\n'}"` — ошибка.

## 📊 Сравнение способов

| Аспект | `%` | `.format()` | f-strings |
| --- | --- | --- | --- |
| **Читаемость** | 🟡 хороша | 🟡 хороша | 🟢 лучшая |
| **Скорость** | 🟡 средняя | 🟡 средняя | 🟢 быстрая |
| **Гибкость** | 🟡 базовая | 🟢 высокая | 🟢 высокая |
| **Вложенность** | 🟡 сложна | 🟡 сложна | 🟢 проста |
| **Версия Python** | 🟢 все | 🟢 2.6+ | 🟡 3.6+ |
| **Динамический ключ** | ❌ нет | 🟢 `"{x}".format(**vars)` | ❌ нет |

**Примеры сложных случаев:**

```python
# Динамический ключ (только .format() и %)
key = "name"
data = {"name": "Alice", "age": 25}
print("{%s}".format(**data))  # ❌ синтаксис ошибка
print(data[key])              # ✅ Alice

# Вложенность (f-строки удобнее)
items = [1, 2, 3]
# Старый %
print("Items: %s" % str(items))
# .format()
print("Items: {}".format(items))
# f-strings
print(f"Items: {items}")  # 🟢 самый простой
```

## ⚙️ Форматирование чисел

**Для всех способов работают одинаковые спецификаторы:**

| Спец | Значение | Пример | Результат |
| --- | --- | --- | --- |
| `:.2f` | 2 знака после запятой | `"%.2f" % 3.14159` | `"3.14"` |
| `:05d` | целое число, минимум 5 цифр, заполнено нулями | `"%05d" % 42` | `"00042"` |
| `:,` | разделитель тысяч (`,` или `.`) | `"{:,}".format(1000000)` | `"1,000,000"` |
| `:e` | экспоненциальная запись | `"{:e}".format(1000)` | `"1.000000e+03"` |
| `:x` | шестнадцатеричное | `"{:x}".format(255)` | `"ff"` |
| `:b` | двоичное | `"{:b}".format(5)` | `"101"` |
| `:%` | процент | `"{:.1%}".format(0.955)` | `"95.5%"` |

**Примеры:**

```python
# Плавающая точка
print("%.2f" % 19.9)              # 19.90
print("{:.2f}".format(19.9))      # 19.90
print(f"{19.9:.2f}")              # 19.90

# Целые числа
print("%05d" % 42)                # 00042
print("{:05d}".format(42))        # 00042
print(f"{42:05d}")                # 00042

# Тысячи
print("{:,}".format(1000000))     # 1,000,000

# Проценты
print("{:.1%}".format(0.955))     # 95.5%
```

## 🎯 Когда какой способ использовать

### ✅ Используй `%` когда:
- Нужна простота и легкость чтения
- Поддерживаешь старый код (Python 2)
- Один-два параметра в простой строке
- Работаешь с именованными полями (словарь)

```python
# Хорош для простых случаев
user = {"name": "Alice", "age": 25}
print("%(name)s is %(age)d" % user)
```

### ✅ Используй `.format()` когда:
- Нужна максимальная гибкость и вложенность
- Должна работать на Python 2.6+
- Нужны сложные спецификации выравнивания
- Передаёшь параметры из других функций

```python
# Хорош для гибкости
print("{name:>15} | {age:<5d}".format(name="Alice", age=25))
```

### ✅ Используй f-strings когда:
- Python 3.6+ (это почти везде сейчас)
- Нужна читаемость и скорость
- Вычисляешь выражения внутри строки
- Работаешь с переменными напрямую

```python
# Лучший выбор для новых проектов
name = "Alice"
age = 25
print(f"Hello {name}, you are {age} years old!")
print(f"Next year: {age + 1}")
```

---

## ❓ Примечания

- Python 3.12+ добавил синтаксис `f"{value=}"`, выводящий и значение и переменную: `f"{age=}"` → `"age=25"`
- При использовании `%` с кортежем одного значения обязательны скобки: `"%s" % ("text",)`, а не `"%s" % ("text")`
- F-строки **вычисляют выражение в момент создания строки**, `%` и `.format()` — при вызове метода
