# Памятка: Форматирование строк через .format() в Python

## 💡 Полезные советы

- **str.format()** - универсальный метод форматирования (Python 2.7+)
- **Гибкость**: можно использовать позиции, имена, атрибуты
- **Синтаксис**: `"текст {позиция:формат} текст".format(значения)`
- **Повторное использование**: можно ссылаться на один аргумент много раз
- **f-строки быстрее**: для Python 3.6+ лучше использовать f-строки
- **Спецификация формата**: `{[index]:[fill][align][sign][width][.precision][type]}`
- **Безопасность**: не использует eval, безопаснее чем %
- **Универсальность**: работает с любыми типами данных

## 🚀 Быстрая справка

## 📝 Базовое использование

```python
# ========== БЕЗ ПАРАМЕТРОВ ==========

# Один аргумент
"Hello, {}!".format("World")    # → "Hello, World!"

# Несколько аргументов (по порядку)
"{} + {} = {}".format(2, 3, 5)  # → "2 + 3 = 5"
"{} {} {}".format("a", "b", "c")  # → "a b c"

# ========== ПО ПОЗИЦИИ (индексу) ==========

# Явные индексы
"{0} {1}".format("Hello", "World")  # → "Hello World"
"{1} {0}".format("Hello", "World")  # → "World Hello"

# Повторное использование
"{0} {0} {0}".format("ha")      # → "ha ha ha"
"{0} and {1} and {0}".format("A", "B")  # → "A and B and A"

# Смешанные индексы
"{2} {0} {1}".format("a", "b", "c")  # → "c a b"

# ========== ПО ИМЕНИ ==========

# Именованные аргументы
"{name} is {age} years old".format(name="Alice", age=25)
# → "Alice is 25 years old"

# Можно комбинировать
"{0} is {age} years old, {0}!".format("Alice", age=25)
# → "Alice is 25 years old, Alice!"

# Из словаря (через распаковку)
user = {"name": "Bob", "age": 30}
"{name} is {age}".format(**user)  # → "Bob is 30"

# ========== АВТОМАТИЧЕСКАЯ НУМЕРАЦИЯ ==========

# Пустые скобки заполняются по порядку
"{} {} {}".format(1, 2, 3)      # → "1 2 3"

# ❌ Нельзя смешивать пустые и с индексами
# "{} {1}".format(1, 2)         # ValueError!
```

## 🔢 Форматирование чисел

### Целые числа

```python
num = 42

# ========== БАЗОВОЕ ==========
"{:d}".format(num)              # → "42" (десятичное)
"Number: {:d}".format(num)      # → "Number: 42"

# ========== ШИРИНА ПОЛЯ ==========
"{:5d}".format(num)             # → "   42" (минимум 5 символов)
"{:10d}".format(num)            # → "        42"

# ========== ДОПОЛНЕНИЕ НУЛЯМИ ==========
"{:05d}".format(num)            # → "00042"
"{:010d}".format(num)           # → "0000000042"

# ========== ЗНАК ==========
"{:+d}".format(42)              # → "+42" (всегда показывать)
"{:+d}".format(-42)             # → "-42"
"{: d}".format(42)              # → " 42" (пробел для положительных)
"{: d}".format(-42)             # → "-42"

# ========== ВЫРАВНИВАНИЕ ==========
"{:<5d}".format(42)             # → "42   " (влево)
"{:>5d}".format(42)             # → "   42" (вправо)
"{:^5d}".format(42)             # → " 42  " (центр)

# С заполнителем
"{:*<10d}".format(42)           # → "42********"
"{:0>10d}".format(42)           # → "0000000042"
"{:-^10d}".format(42)           # → "----42----"

# ========== СИСТЕМЫ СЧИСЛЕНИЯ ==========
"{:b}".format(10)               # → "1010" (двоичная)
"{:o}".format(10)               # → "12" (восьмеричная)
"{:x}".format(255)              # → "ff" (hex строчные)
"{:X}".format(255)              # → "FF" (hex заглавные)

# С префиксом
"{:#b}".format(10)              # → "0b1010"
"{:#o}".format(10)              # → "0o12"
"{:#x}".format(255)             # → "0xff"
"{:#X}".format(255)             # → "0XFF"

# ========== РАЗДЕЛИТЕЛИ ТЫСЯЧ ==========
"{:,}".format(1000000)          # → "1,000,000"
"{:_}".format(1000000)          # → "1_000_000"
"{:,d}".format(1234567)         # → "1,234,567"
```

### Вещественные числа

```python
num = 42.5678

# ========== БАЗОВОЕ ==========
"{:f}".format(num)              # → "42.567800" (6 знаков по умолчанию)

# ========== ТОЧНОСТЬ (знаков после запятой) ==========
"{:.2f}".format(num)            # → "42.57"
"{:.0f}".format(num)            # → "43" (округление)
"{:.4f}".format(num)            # → "42.5678"

# ========== ШИРИНА + ТОЧНОСТЬ ==========
"{:8.2f}".format(num)           # → "   42.57" (ширина 8, точность 2)
"{:10.3f}".format(num)          # → "    42.568"

# ========== ДОПОЛНЕНИЕ НУЛЯМИ ==========
"{:08.2f}".format(num)          # → "00042.57"
"{:010.3f}".format(num)         # → "000042.568"

# ========== ВЫРАВНИВАНИЕ ==========
"{:<10.2f}".format(num)         # → "42.57     "
"{:>10.2f}".format(num)         # → "     42.57"
"{:^10.2f}".format(num)         # → "  42.57   "

# С заполнителем
"{:*<10.2f}".format(num)        # → "42.57*****"
"{:*>10.2f}".format(num)        # → "*****42.57"
"{:*^10.2f}".format(num)        # → "**42.57***"

# ========== ЗНАК ==========
"{:+.2f}".format(42.57)         # → "+42.57"
"{:+.2f}".format(-42.57)        # → "-42.57"
"{: .2f}".format(42.57)         # → " 42.57"

# ========== РАЗДЕЛИТЕЛИ ТЫСЯЧ ==========
"{:,.2f}".format(1234567.89)    # → "1,234,567.89"
"{:_.2f}".format(1234567.89)    # → "1_234_567.89"

# ========== НАУЧНАЯ НОТАЦИЯ ==========
"{:e}".format(1234.5)           # → "1.234500e+03"
"{:.2e}".format(1234.5)         # → "1.23e+03"
"{:E}".format(1234.5)           # → "1.234500E+03" (заглавная E)

# ========== КОМПАКТНЫЙ ФОРМАТ ==========
"{:g}".format(0.0000123)        # → "1.23e-05"
"{:g}".format(123.456)          # → "123.456"

# ========== ПРОЦЕНТ ==========
"{:%}".format(0.756)            # → "75.600000%"
"{:.1%}".format(0.756)          # → "75.6%"
"{:.2%}".format(0.756)          # → "75.60%"
```

## 📏 Выравнивание и заполнение

```python
text = "Hello"

# ========== ВЫРАВНИВАНИЕ ==========

# Влево (по умолчанию для строк)
"{:<10}".format(text)           # → "Hello     "

# Вправо (по умолчанию для чисел)
"{:>10}".format(text)           # → "     Hello"

# По центру
"{:^10}".format(text)           # → "  Hello   "

# ========== ЗАПОЛНЕНИЕ СИМВОЛОМ ==========

# Звёздочки
"{:*<10}".format(text)          # → "Hello*****"
"{:*>10}".format(text)          # → "*****Hello"
"{:*^10}".format(text)          # → "**Hello***"

# Дефисы
"{:-<10}".format(text)          # → "Hello-----"

# Подчёркивания
"{:_^10}".format(text)          # → "__Hello___"

# Точки
"{:.<10}".format(text)          # → "Hello....."

# ========== ДЛЯ ЧИСЕЛ ==========

num = 42

# Дополнение нулями (специальный случай)
"{:05d}".format(num)            # → "00042"

# Другие символы
"{:0>10d}".format(num)          # → "0000000042"
"{:*>10d}".format(num)          # → "********42"

# С выравниванием после знака
"{:=+10d}".format(42)           # → "+       42"
"{:=+10d}".format(-42)          # → "-       42"
```

## 🎯 Доступ к атрибутам и элементам

```python
# ========== АТРИБУТЫ ОБЪЕКТА ==========

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

person = Person("Alice", 25)

# Доступ к атрибутам
"{0.name} is {0.age} years old".format(person)
# → "Alice is 25 years old"

"{p.name} works here".format(p=person)
# → "Alice works here"

# ========== ЭЛЕМЕНТЫ СПИСКА/КОРТЕЖА ==========

data = [10, 20, 30]

# По индексу
"{0[0]} {0[1]} {0[2]}".format(data)  # → "10 20 30"

"{lst[0]} + {lst[1]} = {lst[2]}".format(lst=[1, 2, 3])
# → "1 + 2 = 3"

# ========== ЭЛЕМЕНТЫ СЛОВАРЯ ==========

user = {"name": "Bob", "age": 30}

# По ключу
"{0[name]} is {0[age]}".format(user)  # → "Bob is 30"

"{data[city]}, {data[country]}".format(data={"city": "Moscow", "country": "Russia"})
# → "Moscow, Russia"

# ========== ВЛОЖЕННЫЕ СТРУКТУРЫ ==========

data = {
    "user": {"name": "Alice", "age": 25},
    "city": "Moscow"
}

# Не работает напрямую с вложенными структурами
# Нужно распаковывать или использовать отдельные аргументы
"{name} from {city}".format(name=data["user"]["name"], city=data["city"])
# → "Alice from Moscow"
```

## 🔧 Динамические параметры

```python
# ========== ДИНАМИЧЕСКАЯ ШИРИНА ==========

width = 10
text = "Hello"

# Ширина из переменной
"{text:{width}}".format(text=text, width=width)
# → "Hello     "

# Выравнивание с динамической шириной
"{text:>{width}}".format(text=text, width=width)
# → "     Hello"

# ========== ДИНАМИЧЕСКАЯ ТОЧНОСТЬ ==========

num = 3.14159
precision = 2

# Точность из переменной
"{num:.{prec}f}".format(num=num, prec=precision)
# → "3.14"

# ========== КОМБИНИРОВАНИЕ ==========

value = 42.5678
width = 10
precision = 2

"{val:{w}.{p}f}".format(val=value, w=width, p=precision)
# → "     42.57"

# С выравниванием
"{val:<{w}.{p}f}".format(val=value, w=width, p=precision)
# → "42.57     "
```

## 📋 Таблицы и отчёты

```python
# ========== ПРОСТАЯ ТАБЛИЦА ==========

products = [
    ("Apple", 150, 2.50),
    ("Banana", 200, 1.80),
    ("Cherry", 75, 5.20)
]

# Заголовок
print("{:<10} {:>5} {:>8}".format("Product", "Qty", "Price"))
print("-" * 25)

# Данные
for name, qty, price in products:
    print("{:<10} {:>5} {:>8.2f}".format(name, qty, price))

# Вывод:
# Product       Qty    Price
# -------------------------
# Apple         150     2.50
# Banana        200     1.80
# Cherry         75     5.20

# ========== С ИТОГАМИ ==========

total_qty = sum(qty for _, qty, _ in products)
total_price = sum(qty * price for _, qty, price in products)

print("-" * 25)
print("{:<10} {:>5} {:>8.2f}".format("Total", total_qty, total_price))

# ========== ВЫРАВНИВАНИЕ ПО ПРАВОМУ КРАЮ ==========

data = [
    {"name": "Alice", "score": 95},
    {"name": "Bob", "score": 87},
    {"name": "Charlie", "score": 92}
]

for item in data:
    print("{name:<10} {score:>5}".format(**item))

# Вывод:
# Alice         95
# Bob           87
# Charlie       92
```

    percent = current / total
    filled = int(width * percent)
    bar = "█" * filled + "░" * (width - filled)
    return "{} {:.1%} ({}/{})".format(bar, percent, current, total)

## 🔍 Спецификация формата

```python
# Полный синтаксис:
# {[index/name]:[fill][align][sign][#][0][width][,][.precision][type]}

# fill      - символ заполнения (любой символ)
# align     - < влево, > вправо, ^ центр, = после знака
# sign      - + всегда, - только отрицательные, пробел для положительных
# #         - альтернативная форма (0b, 0o, 0x)
# 0         - дополнение нулями
# width     - минимальная ширина
# ,         - разделитель тысяч (запятая)
# _         - разделитель тысяч (подчёркивание)
# .precision- точность (знаков после запятой)
# type      - d (int), f (float), e (научная), % (процент), s (строка)

# Примеры со всеми параметрами:
"{:*>+10,.2f}".format(1234.5)   # → "**+1,234.50"
# * - заполнитель
# > - выравнивание вправо
# + - всегда показывать знак
# 10 - минимальная ширина
# , - разделитель тысяч
# .2 - 2 знака после запятой
# f - float
```

## ⚠️ Частые ошибки

```python
# ❌ Смешивание пустых {} и индексов
# "{} {1}".format(1, 2)         # ValueError!

# ✅ Используйте либо все пустые, либо все с индексами
"{} {}".format(1, 2)            # ✅
"{0} {1}".format(1, 2)          # ✅

# ❌ Неправильный порядок спецификаторов
# "{:5.2>f}".format(3.14)       # ValueError!

# ✅ Правильный порядок: [fill][align][width][.precision][type]
"{:>5.2f}".format(3.14)         # ✅

# ❌ Забыли тип для чисел с форматом
# "{:05}".format(42)            # Может не сработать

# ✅ Указывайте тип явно
"{:05d}".format(42)             # ✅

# ❌ Попытка форматировать несуществующий индекс
# "{0} {1}".format("one")       # IndexError!

# ✅ Передайте все необходимые аргументы
"{0} {1}".format("one", "two")  # ✅

# ⚠️ Точность для целых чисел не работает
# "{:.2d}".format(42)           # ValueError!

# ✅ Точность только для float
"{:.2f}".format(42.0)           # ✅
```

## 🎯 Когда использовать .format()

**Используйте .format() когда:**

- Поддерживаете Python 2.7 - 3.5
- Шаблон строки используется много раз
- Нужна совместимость со старым кодом
- Работаете с внешними шаблонами

**Используйте f-строки когда:**

- Python 3.6+ доступен
- Нужна скорость (f-строки быстрее)
- Нужна читаемость
- Вставляете выражения

```python
name = "Alice"
age = 25

# .format()
"Name: {}, Age: {}".format(name, age)

# f-строка (ЛУЧШЕ для Python 3.6+)
f"Name: {name}, Age: {age}"

# Но .format() полезна для шаблонов:
template = "Hello, {name}! You are {age} years old."
# Можно использовать много раз
print(template.format(name="Alice", age=25))
print(template.format(name="Bob", age=30))
```
