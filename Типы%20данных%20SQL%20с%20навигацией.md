# Строковые типы данных

## CHAR

Фиксированная длина строки.

```sql
name CHAR(10)
```

## VARCHAR

Переменная длина строки.

```sql
name VARCHAR(100)
```

## TEXT

Большие текстовые данные.

| Тип | Размер |
|------|---------|
| TINYTEXT | 255 байт |
| TEXT | 65 КБ |
| MEDIUMTEXT | 16 МБ |
| LONGTEXT | 4 ГБ |

## Целые числа

| Тип | Размер |

| SMALLINT | 2 байта |
| INT | 4 байта |
| BIGINT | 8 байт |

## FLOAT

```sql
price FLOAT
```

## DECIMAL

```sql

```
