### **Памятка: slice в Python**

**1. Создание объекта среза**

`s = slice(start, stop, step)`

- `start` — индекс начала среза (включительно).
    
- `stop` — индекс конца среза (не включая).
    
- `step` — шаг (необязательный, по умолчанию 1).
    

**Пример:**

`s = slice(1, 5) text = "abcdef" print(text[s])  # вывод: 'bcde'`

**2. Использование с шагом:**

`s = slice(0, 6, 2) text = "abcdef" print(text[s])  # вывод: 'ace'`

**3. Эквивалентная запись через квадратные скобки:**

`text[1:5]        # то же, что text[slice(1, 5)] text[0:6:2]      # то же, что text[slice(0, 6, 2)]`

**4. Методы объекта slice:**

`s = slice(1, 5, 2) print(s.start)  # 1 print(s.stop)   # 5 print(s.step)   # 2`

**5. Особенности:**

- `stop` **не включается**.
    
- Можно использовать с любыми последовательностями: строки, списки, кортежи.
    
- Очень удобно, если нужно **повторно использовать один и тот же диапазон среза**.
