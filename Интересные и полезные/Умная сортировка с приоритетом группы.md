```
def sort_priority(values, group):
    values.sort(key=lambda x: (x in group, x))
```


```
До:       [5, 2, 8, 1, 3]
group = {2, 5}

Ключи:
5 → (True, 5)   # в группе
2 → (True, 2)   # в группе  
8 → (False, 8)  # не в группе
1 → (False, 1)  # не в группе
3 → (False, 3)  # не в группе

Сортировка кортежей:
(True, 2)  # группа, малое значение
(True, 5)  # группа, большое значение
(False, 1) # не группа, малое значение
(False, 3) # не группа, среднее значение  
(False, 8) # не группа, большое значение

После:     [2, 5, 1, 3, 8]
```
