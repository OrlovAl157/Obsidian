## 🔀 Условные выражения `(?(n)yes|no)`

Если группа n нашла совпадение - применяется паттерн `yes`, иначе - `no`.

### 📋 Синтаксис

```
(?(n)yes-pattern|no-pattern)
(?(name)yes-pattern|no-pattern)  # для именованных групп
```

`no-pattern` можно опустить: `(?(n)yes-pattern|)`

-----

### 💻 Примеры

**Пример 1: Простое условие**

```python
import re

regex = r"(a)?(?(1)b|c)"

# Логика:
# - Группа (a)? ищет 'a' (может не быть)
# - Если группа 1 нашла 'a' → ищет 'b'
# - Если группа 1 пустая → ищет 'c'

text = "ab ac c cb"
re.findall(regex, text)  # ['a', '', '']  (найдёт: ab, c)
```

**Пример 2: Скобки**

```python
import re

# Цифра в скобках или без
regex = r"^(\[)?\d(?(1)\]|)$"

# Логика:
# - (\[)? - может быть '['
# - \d - обязательная цифра
# - (?(1)\]|) - если была '[', то ищем ']', иначе ничего

re.match(regex, "1")     # ✅ Match
re.match(regex, "[2]")   # ✅ Match
re.match(regex, "3]")    # ❌ None (нет '[')
re.match(regex, "[4")    # ❌ None (нет ']')
```
