## Основное различие

||`str()`|`repr()`|
|:-:|:-:|:-:|
|**Для кого**|Для пользователей|Для разработчиков|
|**Цель**|Читаемость|Однозначность|
|**Вызов**|`str(obj)` или `print(obj)`|`repr(obj)` или в консоли|
|**Магический метод**|`__str__()`|`__repr__()`|

---

## Основные принципы

### `str()` - для людей

- Красивый, понятный вывод
- Может опускать технические детали
- Используется в `print()`, `format()`, `f-strings`

### `repr()` - для отладки

- Максимально точное представление
- Должен содержать всю важную информацию
- **Идеал:** `eval(repr(obj)) == obj` (можно воссоздать объект)
- Используется в консоли, отладчиках, логах

---
### Дата и время

```python
from datetime import datetime

dt = datetime(2025, 10, 16, 14, 30, 45)

print(str(dt))   # 2025-10-16 14:30:45
print(repr(dt))  # datetime.datetime(2025, 10, 16, 14, 30, 45)
```
