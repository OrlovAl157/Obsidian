### Пример кода с отслеживанием выполнения

python

```python
import functools

def stop_num(stop):
    print('1️⃣ Выводит при определении (уровень параметров)')
    
    def decorator(func):
        print('2️⃣ Выводит при определении (уровень декоратора)')
        
        @functools.wraps(func)
        def wrapper(*args, **kwargs):
            print('3️⃣ Выводит при ВЫЗОВЕ функции')
            count = 0
            while True:
                try:
                    return func(*args, **kwargs)
                except:
                    print(f'   Попытка №{count}')
                    count += 1
                    if count == stop:
                        raise
        return wrapper
    return decorator


@stop_num(6)
def my_func(x, y):
    return x ** y


print('4️⃣ После определения, перед вызовом')
result = my_func(2, 3)
print('5️⃣ После вызова функции')
```

### 🔄 Что происходит при повторных вызовах

python

```python
@stop_num(6)
def my_func(x, y):
    return x ** y

# Вывод при определении:
# 1️⃣ Выводит при определении (уровень параметров)
# 2️⃣ Выводит при определении (уровень декоратора)

print("Первый вызов:")
my_func(2, 3)
# 3️⃣ Выводит при ВЫЗОВЕ функции

print("Второй вызов:")
my_func(3, 2)
# 3️⃣ Выводит при ВЫЗОВЕ функции

print("Третий вызов:")
my_func(5, 2)
# 3️⃣ Выводит при ВЫЗОВЕ функции
```

**Вывод:**

```
1️⃣ Выводит при определении (уровень параметров)
2️⃣ Выводит при определении (уровень декоратора)
Первый вызов:
3️⃣ Выводит при ВЫЗОВЕ функции
Второй вызов:
3️⃣ Выводит при ВЫЗОВЕ функции
Третий вызов:
3️⃣ Выводит при ВЫЗОВЕ функции
```

**Заметьте:** 1️⃣ и 2️⃣ выводятся ТОЛЬКО ОДИН РАЗ при определении!

### 📌 Ключевое правило

|Уровень|Когда выполняется|Сколько раз|Пример вывода|
|---|---|---|---|
|1️⃣ `stop_num(stop)`|При определении функции|1 раз|"Выводит при определении 1"|
|2️⃣ `decorator(func)`|При определении функции|1 раз|"Выводит при определении 2"|
|3️⃣ `wrapper(*args)`|При каждом вызове|Каждый раз|"3"|
