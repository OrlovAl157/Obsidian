
## 💡 Полезные советы

- **Всегда используйте `with`**: автоматически закрывает файл
- **Режимы**: `'r'` (чтение), `'w'` (запись), `'a'` (добавление)
- **Кодировка**: явно указывайте `encoding='utf-8'` для кириллицы
- **Построчное чтение**: экономит память для больших файлов
- **`\n` vs `\r\n`**: Windows использует `\r\n`, Unix/Mac - `\n`
- **Путь**: используйте `pathlib.Path` для кроссплатформенности
- **Проверка существования**: `Path.exists()` перед чтением
- **Ошибки**: обрабатывайте `FileNotFoundError` и `PermissionError`

## 🚀 Быстрая справка

|Задача|Код|Пример|
|---|---|---|
|Открыть файл|`open(file, mode)`|`open('data.txt', 'r')`|
|Открыть с кодировкой|`open(file, encoding='utf-8')`|`open('text.txt', 'r', encoding='utf-8')`|
|Прочитать всё|`.read()`|`f.read()`|
|Прочитать строки|`.readlines()`|`f.readlines()`|
|Прочитать строку|`.readline()`|`f.readline()`|
|Записать текст|`.write()`|`f.write('Hello')`|
|Записать строки|`.writelines()`|`f.writelines(lines)`|
|Перебор строк|`for line in f:`|`for line in f:`|
|Закрыть файл|`.close()`|`f.close()`|
|Проверить существование|`Path.exists()`|`Path('file.txt').exists()`|

## 📖 Чтение файлов

### Чтение всего файла

```python
# Простой способ (read всё в память)
with open('file.txt', 'r', encoding='utf-8') as f:
    content = f.read()
    print(content)

# Пример
# Содержимое file.txt:
# Hello World
# Python is awesome
# 
# content = "Hello World\nPython is awesome"

# Чтение с обработкой ошибок
try:
    with open('file.txt', 'r', encoding='utf-8') as f:
        content = f.read()
except FileNotFoundError:
    print("Файл не найден")
except PermissionError:
    print("Нет доступа к файлу")
```

### Чтение построчно

```python
# Читать строки в список
with open('file.txt', 'r', encoding='utf-8') as f:
    lines = f.readlines()
    # lines = ['Hello World\n', 'Python is awesome\n']

# Убрать \n с концов строк
lines = [line.rstrip('\n') for line in lines]
# lines = ['Hello World', 'Python is awesome']

# Читать по одной строке (экономит память!)
with open('file.txt', 'r', encoding='utf-8') as f:
    line1 = f.readline()        # → "Hello World\n"
    line2 = f.readline()        # → "Python is awesome\n"
    line3 = f.readline()        # → "" (конец файла)

# Перебор строк (РЕКОМЕНДУЕТСЯ для больших файлов)
with open('file.txt', 'r', encoding='utf-8') as f:
    for line in f:
        print(line.strip())     # strip() убирает \n
```

### Чтение с различными кодировками

```python
# UTF-8 (рекомендуется для кириллицы)
with open('file.txt', 'r', encoding='utf-8') as f:
    content = f.read()

# Windows-1251 (старые файлы)
with open('file.txt', 'r', encoding='cp1251') as f:
    content = f.read()

# ASCII (только английские символы)
with open('file.txt', 'r', encoding='ascii') as f:

# Автоопределение кодировки (требует chardet)

with open('file.txt', 'rb') as f:
    raw_data = f.read()
    result = chardet.detect(raw_data)
    encoding = result['encoding']

with open('file.txt', 'r', encoding=encoding) as f:
    content = f.read()
```

## 🔄 Режимы открытия файлов

```python
# Основные режимы
'r'     # Чтение (файл должен существовать)
'w'     # Запись (создаёт новый или перезаписывает)
'a'     # Добавление в конец
'x'     # Создание (ошибка если существует)

# Комбинированные режимы
'r+'    # Чтение и запись (файл должен существовать)
'w+'    # Чтение и запись (создаёт новый или перезаписывает)
'a+'    # Чтение и добавление

# Бинарные режимы (для не-текстовых файлов)
'rb'    # Чтение в бинарном режиме
'wb'    # Запись в бинарном режиме
'ab'    # Добавление в бинарном режиме

# Примеры
# Только чтение
with open('file.txt', 'r') as f:
    content = f.read()

# Только запись (перезапись)
with open('file.txt', 'w') as f:
    f.write('New content')

# Добавление
with open('file.txt', 'a') as f:
    f.write('Added line\n')

# Создание (ошибка если существует)
try:
    with open('new.txt', 'x') as f:
        f.write('Content')
except FileExistsError:
    print("Файл уже существует")
```

```python
def search_in_file(filename, query):
    """Найти строки, содержащие query"""
    results = []
    with open(filename, 'r', encoding='utf-8') as f:
        for line_num, line in enumerate(f, 1):
            if query in line:
                results.append((line_num, line.strip()))
    return results
```

### Замена текста в файле

```python
def replace_in_file(filename, old_text, new_text):
    """Заменить текст в файле"""
    # Прочитать файл
    with open(filename, 'r', encoding='utf-8') as f:
        content = f.read()
    
    # Заменить текст
    content = content.replace(old_text, new_text)
    
    # Записать обратно
    with open(filename, 'w', encoding='utf-8') as f:
        f.write(content)

replace_in_file('config.txt', 'old_value', 'new_value')

# Безопасная версия (с бэкапом)
def safe_replace(filename, old_text, new_text):
    import shutil
    
    # Создать резервную копию
    backup = f"{filename}.bak"
    shutil.copy2(filename, backup)
    
    try:
        replace_in_file(filename, old_text, new_text)
    except Exception as e:
        # Восстановить из бэкапа при ошибке
        shutil.copy2(backup, filename)
        raise e
```

```python
def read_csv_manual(filename):
    """Прочитать CSV файл"""
    data = []
    with open(filename, 'r', encoding='utf-8') as f:
        header = f.readline().strip().split(',')
        for line in f:
            values = line.strip().split(',')
```

### Удаление дубликатов строк

```python
def remove_duplicates(input_file, output_file):
    """Удалить дубликаты строк"""
    seen = set()
    with open(input_file, 'r', encoding='utf-8') as infile:
        with open(output_file, 'w', encoding='utf-8') as outfile:
            for line in infile:
                if line not in seen:
                    outfile.write(line)
                    seen.add(line)

remove_duplicates('data.txt', 'unique_data.txt')

# С сохранением порядка
def remove_duplicates_ordered(input_file, output_file):
    seen = set()
    with open(input_file, 'r', encoding='utf-8') as infile:
        lines = []
        for line in infile:
            clean_line = line.strip()
            if clean_line not in seen:
                lines.append(line)
                seen.add(clean_line)
    
    with open(output_file, 'w', encoding='utf-8') as outfile:
        outfile.writelines(lines)
```

```python
def split_file(filename, lines_per_file):
    """Разделить файл на части"""
    with open(filename, 'r', encoding='utf-8') as f:
        file_num = 1
        lines_written = 0
        outfile = None
        
        for line in f:
            if lines_written == 0:
                if outfile:
                    outfile.close()
                outfile = open(f'part_{file_num}.txt', 'w', encoding='utf-8')
                file_num += 1
            
            outfile.write(line)
            lines_written += 1
            
            if lines_written >= lines_per_file:
                lines_written = 0
        
        if outfile:
            outfile.close()

split_file('large_file.txt', 1000)  # По 1000 строк в каждом файле
```

### Статистика файла

```python
def file_stats(filename):
    """Получить статистику файла"""
    lines = 0
    words = 0
    chars = 0
    
    with open(filename, 'r', encoding='utf-8') as f:
        for line in f:
            lines += 1
            words += len(line.split())
            chars += len(line)
    
    return {
        'lines': lines,
        'words': words,
        'characters': chars
    }

stats = file_stats('document.txt')
print(f"Строк: {stats['lines']}")
print(f"Слов: {stats['words']}")
print(f"Символов: {stats['characters']}")
```

```python
def filter_lines(input_file, output_file, condition):
    """Отфильтровать строки по условию"""
    with open(input_file, 'r', encoding='utf-8') as infile:
        with open(output_file, 'w', encoding='utf-8') as outfile:
```

```python
with open('huge_file.txt', 'r') as f:

with open('huge_file.txt', 'r', encoding='utf-8') as f:
    for line in f:              # Читает по одной строке
        process(line)           # Обрабатывает

# Чтение порциями (chunks)
def process_large_file(filename, chunk_size=1024*1024):  # 1MB
    """Обработка файла кусками"""
    with open(filename, 'r', encoding='utf-8') as f:
        while True:
            chunk = f.read(chunk_size)
            if not chunk:
                break
            process(chunk)

# Подсчёт строк эффективно
def count_lines_efficient(filename):
    """Быстрый подсчёт строк"""
    with open(filename, 'r', encoding='utf-8') as f:
        return sum(1 for _ in f)
```

## 🎯 Когда использовать что

**Используйте `.read()` когда:**

- Файл маленький (< 10 MB)
- Нужно работать со всем содержимым сразу
- Производите замены или парсинг всего текста

**Используйте построчное чтение когда:**

- Файл большой (> 10 MB)
- Обрабатываете данные построчно
- Не нужно всё содержимое в памяти
- Логи, CSV, данные для анализа

**Используйте pathlib когда:**

- Работаете с путями файлов
- Нужна кроссплатформенность
- Проверяете существование файлов
- Получаете метаинформацию о файлах

**Используйте временные файлы когда:**

- Нужно хранить данные временно
- Создаёте промежуточные файлы
- Не хотите засорять систему
- Безопасность (данные удаляются автоматически)
