## 💡 Полезные советы

- **Всегда используйте `with`**: автоматически закрывает файл
- **Режимы**: `'r'` (чтение), `'w'` (запись), `'a'` (добавление)
- **Кодировка**: явно указывайте `encoding='utf-8'` для кириллицы
- **Построчное чтение**: экономит память для больших файлов
- **`\n` vs `\r\n`**: Windows использует `\r\n`, Unix/Mac - `\n`
- **Путь**: используйте `pathlib.Path` для кроссплатформенности
- **Проверка существования**: `Path.exists()` перед чтением
- **Ошибки**: обрабатывайте `FileNotFoundError` и `PermissionError`

## 🚀 Быстрая справка

|Задача|Код|Пример|
|---|---|---|
|Открыть файл|`open(file, mode)`|`open('data.txt', 'r')`|
|Открыть с кодировкой|`open(file, encoding='utf-8')`|`open('text.txt', 'r', encoding='utf-8')`|
|Прочитать всё|`.read()`|`f.read()`|
|Прочитать строки|`.readlines()`|`f.readlines()`|
|Прочитать строку|`.readline()`|`f.readline()`|
|Записать текст|`.write()`|`f.write('Hello')`|
|Записать строки|`.writelines()`|`f.writelines(lines)`|
|Перебор строк|`for line in f:`|`for line in f:`|
|Закрыть файл|`.close()`|`f.close()`|
|Проверить существование|`Path.exists()`|`Path('file.txt').exists()`|

## 🔄 Режимы открытия файлов

```python
# ========== ТЕКСТОВЫЕ РЕЖИМЫ ==========

# Только чтение
'r'     # Чтение (файл должен существовать, ошибка если нет)
        # Курсор в начале файла

# Только запись
'w'     # Запись (создаёт новый файл или ПОЛНОСТЬЮ перезаписывает существующий!)
        # Курсор в начале файла
        # ОСТОРОЖНО: стирает всё содержимое!

# Добавление
'a'     # Добавление в конец (создаёт файл если не существует)
        # Курсор в конце файла
        # НЕ стирает существующее содержимое

# Эксклюзивное создание
'x'     # Создание нового файла (ошибка если уже существует)
        # Защита от случайной перезаписи

# ========== КОМБИНИРОВАННЫЕ РЕЖИМЫ ==========

# Чтение + Запись
'r+'    # Чтение и запись (файл должен существовать)
        # Курсор в начале файла
        # НЕ стирает содержимое
        # Можно читать и писать в любом месте

'w+'    # Чтение и запись (создаёт новый или СТИРАЕТ существующий!)
        # Курсор в начале файла
        # ОСТОРОЖНО: стирает содержимое при открытии!

'a+'    # Чтение и добавление
        # Курсор в конце для записи
        # НЕ стирает содержимое
        # Можно читать с любого места, но запись только в конец

# ========== БИНАРНЫЕ РЕЖИМЫ ==========
# Для работы с не-текстовыми файлами (изображения, видео, архивы)

'rb'    # Чтение в бинарном режиме
'wb'    # Запись в бинарном режиме
'ab'    # Добавление в бинарном режиме
'rb+'   # Чтение и запись в бинарном режиме
'wb+'   # Запись и чтение в бинарном режиме
'ab+'   # Добавление и чтение в бинарном режиме

# ========== ПРИМЕРЫ ==========

# Только чтение
with open('file.txt', 'r') as f:
    content = f.read()

# Только запись (ОСТОРОЖНО: стирает файл!)
with open('file.txt', 'w') as f:
    f.write('New content')

# Добавление в конец
with open('file.txt', 'a') as f:
    f.write('Added line\n')

# Создание нового файла (защита от перезаписи)
try:
    with open('new.txt', 'x') as f:
        f.write('Content')
except FileExistsError:
    print("Файл уже существует")

# Чтение и запись (без стирания)
with open('file.txt', 'r+') as f:
    content = f.read()          # Прочитать
    f.write('\nNew line')       # Дописать

# ========== СРАВНЕНИЕ РЕЖИМОВ ==========

# 'w' vs 'a' vs 'r+'
# 'w'  - СТИРАЕТ файл при открытии
# 'a'  - НЕ стирает, добавляет в конец
# 'r+' - НЕ стирает, можно писать в любое место

# Пример разницы:
# Исходный файл содержит: "Hello"

# С 'w':
with open('test.txt', 'w') as f:
    f.write('World')
# Результат: "World" (Hello стёрто!)

# С 'a':
with open('test.txt', 'a') as f:
    f.write('World')
# Результат: "HelloWorld" (добавлено в конец)

# С 'r+':
with open('test.txt', 'r+') as f:
    f.write('World')
# Результат: "Worldo" (перезаписано с начала, остальное осталось)
```

## 🎯 Управление позицией в файле (seek и tell)

```python
# tell() - узнать текущую позицию курсора
# seek() - переместить курсор в нужную позицию

# ========== tell() - получить позицию ==========
with open('file.txt', 'r') as f:
    print(f.tell())             # → 0 (курсор в начале)
    f.read(5)                   # Прочитать 5 символов
    print(f.tell())             # → 5 (курсор на 5-й позиции)
    f.readline()                # Прочитать строку
    print(f.tell())             # → позиция после строки

# ========== seek() - переместить курсор ==========

# seek(offset, whence)
# offset - смещение в байтах
# whence - откуда считать:
#   0 - от начала файла (по умолчанию)
#   1 - от текущей позиции (только в бинарном режиме!)
#   2 - от конца файла (только в бинарном режиме!)

# Переместить в начало файла
with open('file.txt', 'r') as f:
    f.read()                    # Прочитать весь файл
    f.seek(0)                   # Вернуться в начало
    content = f.read()          # Прочитать снова

# Переместить на конкретную позицию
with open('file.txt', 'r') as f:
    f.seek(10)                  # Перейти на 10-й байт
    char = f.read(1)            # Прочитать 1 символ

# Прочитать файл с середины
with open('file.txt', 'r') as f:
    # Перейти в середину файла
    f.seek(0, 2)                # Перейти в конец (только в rb!)
    file_size = f.tell()        # Узнать размер
    f.seek(file_size // 2)      # Перейти в середину
    content = f.read()          # Прочитать от середины

# ========== ПРАКТИЧЕСКИЕ ПРИМЕРЫ ==========

# Чтение последних N строк файла
def read_last_lines(filename, n=10):
    """Прочитать последние N строк"""
    with open(filename, 'rb') as f:
        # Перейти в конец
        f.seek(0, 2)
        file_size = f.tell()
        
        # Читать с конца, пока не найдём N строк
        lines_found = 0
        position = file_size
        
        while position > 0 and lines_found <= n:
            position -= 1
            f.seek(position)
            if f.read(1) == b'\n':
                lines_found += 1
        
        # Вернуться к началу последних N строк
        if position > 0:
            f.seek(position + 1)
        else:
            f.seek(0)
        
        return f.read().decode('utf-8').splitlines()[-n:]

# Чтение файла кусками с возможностью вернуться
def read_chunks_with_rewind(filename, chunk_size=100):
    """Читать куски с возможностью вернуться"""
    with open('file.txt', 'r') as f:
        # Запомнить позицию
        start_pos = f.tell()
        
        # Прочитать кусок
        chunk1 = f.read(chunk_size)
        print(f"Позиция после чтения: {f.tell()}")
        
        # Вернуться назад
        f.seek(start_pos)
        
        # Прочитать снова
        chunk2 = f.read(chunk_size)
        assert chunk1 == chunk2     # Одинаковые куски

# Пропустить первые N байт
def skip_bytes(filename, skip_n):
    """Пропустить начало файла"""
    with open(filename, 'r') as f:
        f.seek(skip_n)              # Пропустить N байт
        content = f.read()
        return content

# Чтение и перезапись в одном файле (r+)
def insert_at_position(filename, position, text):
    """Вставить текст на позицию"""
    with open(filename, 'r+') as f:
        # Прочитать всё содержимое
        content = f.read()
        
        # Переместить курсор на нужную позицию
        f.seek(position)
        
        # Записать новый текст + старое содержимое
        f.write(text + content[position:])
        
        # Обрезать лишнее
        f.truncate()

# Чтение файла с позиции до позиции
def read_range(filename, start, end):
    """Прочитать диапазон байт"""
    with open(filename, 'r') as f:
        f.seek(start)
        length = end - start
        return f.read(length)

content = read_range('file.txt', 10, 50)  # Байты с 10 по 50

# ========== ВАЖНЫЕ ЗАМЕЧАНИЯ ==========

# ⚠️ seek() работает с БАЙТАМИ, не символами!
# В UTF-8 кириллица занимает 2 байта на символ

with open('russian.txt', 'r', encoding='utf-8') as f:
    f.seek(1)                   # Может разорвать UTF-8 символ!
    # Может вызвать UnicodeDecodeError

# ✅ Для текстовых файлов безопаснее:
# - seek(0) для начала файла
# - tell() для запоминания позиции
# - Избегать произвольных seek() в середине UTF-8

# ⚠️ В текстовом режиме whence может быть только 0
with open('file.txt', 'r') as f:
    f.seek(10, 0)               # ✅ OK
    # f.seek(10, 1)             # ❌ Ошибка в текстовом режиме!
    # f.seek(-10, 2)            # ❌ Ошибка в текстовом режиме!

# ✅ Используйте бинарный режим для полного контроля
with open('file.txt', 'rb') as f:
    f.seek(10, 0)               # От начала
    f.seek(10, 1)               # От текущей позиции
    f.seek(-10, 2)              # От конца (назад на 10 байт)

# Метод truncate() - обрезать файл
with open('file.txt', 'r+') as f:
    f.truncate(100)             # Обрезать до 100 байт
    # или
    f.seek(100)
    f.truncate()                # Обрезать от текущей позиции
```

```python
def replace_in_file(filename, old_text, new_text):
    """Заменить текст в файле"""
    # Прочитать файл
    with open(filename, 'r', encoding='utf-8') as f:
        content = f.read()
    
    # Заменить текст
    content = content.replace(old_text, new_text)
    
    # Записать обратно
    with open(filename, 'w', encoding='utf-8') as f:
        f.write(content)

replace_in_file('config.txt', 'old_value', 'new_value')

# Безопасная версия (с бэкапом)
def safe_replace(filename, old_text, new_text):
    import shutil
    
    # Создать резервную копию
    backup = f"{filename}.bak"
    shutil.copy2(filename, backup)
    
    try:
        replace_in_file(filename, old_text, new_text)
    except Exception as e:
        # Восстановить из бэкапа при ошибке
        shutil.copy2(backup, filename)
        raise e
```

```python
def remove_duplicates(input_file, output_file):
    """Удалить дубликаты строк"""
    seen = set()
    with open(input_file, 'r', encoding='utf-8') as infile:
        with open(output_file, 'w', encoding='utf-8') as outfile:
            for line in infile:
                if line not in seen:
                    outfile.write(line)
                    seen.add(line)

remove_duplicates('data.txt', 'unique_data.txt')

# С сохранением порядка
def remove_duplicates_ordered(input_file, output_file):
    seen = set()
    with open(input_file, 'r', encoding='utf-8') as infile:
        lines = []
        for line in infile:
            clean_line = line.strip()
            if clean_line not in seen:
                lines.append(line)
                seen.add(clean_line)
    
    with open(output_file, 'w', encoding='utf-8') as outfile:
        outfile.writelines(lines)
```

```python
def split_file(filename, lines_per_file):
    """Разделить файл на части"""
    with open(filename, 'r', encoding='utf-8') as f:
        file_num = 1
        lines_written = 0
        outfile = None
        
        for line in f:
            if lines_written == 0:
                if outfile:
                    outfile.close()
                outfile = open(f'part_{file_num}.txt', 'w', encoding='utf-8')
                file_num += 1
            
            outfile.write(line)
            lines_written += 1
            
            if lines_written >= lines_per_file:
                lines_written = 0
        
        if outfile:
            outfile.close()

split_file('large_file.txt', 1000)  # По 1000 строк в каждом файле
```

```python
def file_stats(filename):
    """Получить статистику файла"""
    lines = 0
    words = 0
    chars = 0
    
    with open(filename, 'r', encoding='utf-8') as f:
        for line in f:
            lines += 1
            words += len(line.split())
            chars += len(line)
    
    return {
        'lines': lines,
        'words': words,
        'characters': chars
    }

stats = file_stats('document.txt')
print(f"Строк: {stats['lines']}")
print(f"Слов: {stats['words']}")
print(f"Символов: {stats['characters']}")
```

```python
def filter_lines(input_file, output_file, condition):
    """Отфильтровать строки по условию"""
    with open(input_file, 'r', encoding='utf-8') as infile:
        with open(output_file, 'w', encoding='utf-8') as outfile:
            for line in infile:
                if condition(line):
                    outfile.write(line)

# Оставить только строки длиннее 50 символов
filter_lines('input.txt', 'output.txt', lambda line: len(line) > 50)

# Оставить строки, содержащие слово "Python"
filter_lines('log.txt', 'python_logs.txt', lambda line: 'Python' in line)

# Удалить пустые строки
filter_lines('text.txt', 'clean.txt', lambda line: line.strip() != '')
```

```python
def add_line_numbers(input_file, output_file):
    """Добавить номера строк"""
    with open(input_file, 'r', encoding='utf-8') as infile:
        with open(output_file, 'w', encoding='utf-8') as outfile:
```

```python
    for line in f:              # Читает по одной строке
        process(line)           # Обрабатывает

# Чтение порциями (chunks)
def process_large_file(filename, chunk_size=1024*1024):  # 1MB
    """Обработка файла кусками"""
    with open(filename, 'r', encoding='utf-8') as f:
        while True:
            chunk = f.read(chunk_size)
            if not chunk:
                break
            process(chunk)

# Подсчёт строк эффективно
def count_lines_efficient(filename):
    """Быстрый подсчёт строк"""
    with open(filename, 'r', encoding='utf-8') as f:
        return sum(1 for _ in f)
```

## ⚠️ Частые ошибки

```python
# ❌ ПЛОХО: файл не закрывается при ошибке
f = open('file.txt', 'r')
content = f.read()
# Если произойдёт ошибка, файл останется открытым!
f.close()

# ✅ ХОРОШО: with автоматически закрывает
with open('file.txt', 'r', encoding='utf-8') as f:
    content = f.read()
# Файл закроется автоматически

# ❌ Забыли указать кодировку (проблемы с кириллицей)
with open('russian.txt', 'r') as f:  # Может быть ошибка!
    content = f.read()

# ✅ Всегда указывайте кодировку
with open('russian.txt', 'r', encoding='utf-8') as f:
    content = f.read()

# ❌ Не проверили существование файла
with open('missing.txt', 'r') as f:  # FileNotFoundError!
    content = f.read()

# ✅ Проверка или обработка ошибок
from pathlib import Path

if Path('file.txt').exists():
    with open('file.txt', 'r') as f:
        content = f.read()
else:
    print("Файл не найден")

# Или try-except
try:
    with open('file.txt', 'r') as f:

except FileNotFoundError:
    print("Файл не найден")

# ❌ Перепутали режимы 'w' и 'a'
with open('log.txt', 'w') as f:  # СТИРАЕТ весь файл!

with open('log.txt', 'a') as f:  # Добавляет в конец
```

## 🎯 Когда использовать что

**Используйте `.read()` когда:**

- Файл маленький (< 10 MB)
- Нужно работать со всем содержимым сразу
- Производите замены или парсинг всего текста

**Используйте построчное чтение когда:**

- Файл большой (> 10 MB)
- Обрабатываете данные построчно
- Не нужно всё содержимое в памяти
- Логи, CSV, данные для анализа

**Используйте pathlib когда:**

- Работаете с путями файлов
- Нужна кроссплатформенность
- Проверяете существование файлов
- Получаете метаинформацию о файлах

**Используйте временные файлы когда:**

- Нужно хранить данные временно
- Создаёте промежуточные файлы
- Не хотите засорять систему
- Безопасность (данные удаляются автоматически)
