

## Импорт модуля

```python
from zipfile import ZipFile  # импорт модуля zipfile объекта ZipFile
```

---

## Открытие архива

```python
with ZipFile('файл.zip') as архив:
    # обращение к архиву подобно open
    pass
```

---

## Просмотр содержимого архива

### Вывод таблицы с информацией

```python
архив.printdir()
# выводит таблицу с информацией о содержимом архива:
# полные названия файлов с указанием даты изменения и размера в байтах
```

---

## Получение информации о файлах

### Список объектов ZipInfo

```python
инфо = архив.infolist()  # список специальных объектов (тип ZipInfo)
# с дополнительной информацией о каждом файле

инфо[индекс].file_size      # размер начального файла в байтах
инфо[индекс].compress_size  # размер сжатого файла в байтах
инфо[индекс].filename       # имя файла
инфо[индекс].date_time      # дата изменения файла
инфо[индекс].is_dir()       # проверяет является ли объект папкой
```

### Список названий файлов

```python
инфо = архив.namelist()  # список названий файлов и директорий
```

### Информация об отдельном файле

```python
инфо = архив.namelist()  # получаем названия всех файлов архива
инфо_файл = архив.getinfo(инфо[-4])  # получаем информацию об отдельном файле по имени

инфо_файл.file_size      # размер начального файла в байтах
инфо_файл.compress_size  # размер сжатого файла в байтах
инфо_файл.filename       # имя файла
инфо_файл.date_time      # дата изменения файла
```

## Чтение файлов из архива

### Чтение в бинарном формате

```python
with ZipFile('test.zip') as архив:
    with архив.open('test/Разные файлы/astros.json') as файл:
        print(файл.read())  # выводим содержимое файла из архива в виде бинарной строки
```

### Чтение с декодированием

```python
with ZipFile('test.zip') as архив:
    with архив.open('test/Разные файлы/astros.json') as файл:
        print(файл.read().decode('utf-8'))  # выводим содержимое файла из архива,
                                            # декодируя содержимое (указана кодировка)
```

## Создание и добавление файлов в архив

```python
# mode='w' - создание нового архива
# mode='a' - добавление в существующий архив
with ZipFile('archive.zip', mode='w') as архив:
    архив.write('файл', 'директория/новое_имя_файла')
    # запись файла в архив под новым именем
    # Возможно указать директорию, куда помещается файл
```

---

## Извлечение файлов из архива

### Извлечение отдельного файла

```python
with ZipFile('test.zip') as архив:
    архив.extract('директории/avatar.png', 'путь_куда_извлекаем')
    # извлечение указанного файла из архива с указанием пути (куда извлечь)
    # Если путь не указываем, извлекает в папку с архивом
```

### Извлечение всего содержимого

```python
with ZipFile('test.zip') as zip_file:
    zip_file.extractall('путь_куда_извлекаем')
    # извлечение содержимого архива с указанием пути (куда извлечь)
    # Если путь не указываем, извлекает в папку с архивом
```

## Проверка файла

```python
from zipfile import is_zipfile

is_zipfile('путь/файл')  # проверка, является ли указанный файл zip архивом
                         # Результат True/False
```

---

## Основные режимы открытия архива

- `'r'` - чтение (по умолчанию)
- `'w'` - создание нового архива (перезапись существующего)
- `'a'` - добавление в существующий архив
- `'x'` - создание нового архива (ошибка, если файл существует)
