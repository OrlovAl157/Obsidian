## Умная сортировка с приоритетом группы

Сортировка, при которой элементы из заданной группы идут первыми.

```python
def sort_priority(values, group):
    values.sort(key=lambda x: (x in group, x))
```

**Как работает:**

```
До:       [5, 2, 8, 1, 3]
group = {2, 5}

Ключи сортировки:
5 → (True, 5)   # в группе
2 → (True, 2)   # в группе  
8 → (False, 8)  # не в группе
1 → (False, 1)  # не в группе
3 → (False, 3)  # не в группе

После:     [2, 5, 1, 3, 8]
```

Python сортирует кортежи лексикографически: сначала по первому элементу (False < True), потом по второму.
