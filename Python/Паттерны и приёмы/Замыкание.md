## Замыкание (Closure)

Функция, которая запоминает значения из внешней области видимости.

```python
def make_power(n):
    return lambda x: x ** n

square = make_power(2)
cube   = make_power(3)

print(square(5))  # 25
print(cube(5))    # 125
```

**Как это работает:** `make_power(2)` возвращает функцию, которая "помнит" что `n = 2` — это и есть замыкание.
