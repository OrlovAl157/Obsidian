### Чтение CSV по столбцам

```python
import csv

def csv_columns(filename):
    with open(filename, encoding="utf-8") as file_in:
        rows = list(csv.reader(file_in))
        return {key: value for key, *value in zip(*rows)}
```

**Что делает:** транспонирует CSV — превращает строки в столбцы. Первый элемент каждого столбца становится ключом словаря, остальные — значением (списком).
