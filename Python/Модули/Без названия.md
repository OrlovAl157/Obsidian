
## Быстрая справка: самые используемые

|Константа/Класс|Значение|Применение|
|---|---|---|
|`string.ascii_letters`|`'abcd...xyzABCD...XYZ'`|Все латинские буквы|
|`string.ascii_lowercase`|`'abcdefghijklmnopqrstuvwxyz'`|Строчные буквы|
|`string.ascii_uppercase`|`'ABCDEFGHIJKLMNOPQRSTUVWXYZ'`|Заглавные буквы|
|`string.digits`|`'0123456789'`|Цифры|
|`string.punctuation`|`'!"#$%&\'()*+,-./:;<=>?@[\\]^_`{|}~'`|
|`string.whitespace`|`' \t\n\r\x0b\x0c'`|Пробельные символы|
|`Template('$var')`|Шаблон строки|Безопасная подстановка переменных|

**Примеры использования:**

```python

# Шаблоны
from string import Template
t = Template('Привет, $name!')

```

## Класс Template (шаблоны строк)

### Базовое использование

```python
from string import Template

# Создание шаблона с $переменная
t = Template('Привет, $name! Тебе $age лет.')

# Подстановка значений
result = t.substitute(name='Иван', age=25)
# 'Привет, Иван! Тебе 25 лет.'

# Альтернативный синтаксис ${переменная}
t2 = Template('${greeting}, ${name}!')
result2 = t2.substitute(greeting='Здравствуй', name='Мария')
# 'Здравствуй, Мария!'
```

### Методы Template

```python
from string import Template

t = Template('$who любит $what')

# substitute() - строгая подстановка (ошибка если ключ отсутствует)
try:
    t.substitute(who='Я')  # KeyError: 'what'
except KeyError as e:
    print(f"Ошибка: {e}")

# safe_substitute() - безопасная подстановка
result = t.safe_substitute(who='Я')
# 'Я любит $what' - неподставленные переменные остаются

# Использование словаря
data = {'who': 'Кот', 'what': 'рыбу'}
result = t.substitute(data)
# 'Кот любит рыбу'

# Использование **kwargs
result = t.substitute(who='Она', what='музыку')
# 'Она любит музыку'
```

### Экранирование символа $

```python
from string import Template

# Двойной $$ для вывода одного $
t = Template('Цена: $$${price}')
result = t.substitute(price='100')
# 'Цена: $100'
```

### Пользовательские шаблоны

```python
from string import Template

# Изменение разделителя (вместо $)
class MyTemplate(Template):
    delimiter = '%'  # Используем % вместо $

t = MyTemplate('Привет, %name!')
result = t.substitute(name='Алекс')
# 'Привет, Алекс!'

# Изменение паттерна идентификатора
class BraceTemplate(Template):
    delimiter = '#'
    idpattern = r'[a-z][_a-z0-9]*'  # только строчные буквы

t = BraceTemplate('#user говорит: #message')
result = t.substitute(user='Боб', message='Привет!')
# 'Боб говорит: Привет!'
```

## Класс Formatter (расширенное форматирование)

```python
from string import Formatter

f = Formatter()

# Базовое форматирование
result = f.format('{0}, {1}!', 'Привет', 'мир')
# 'Привет, мир!'

# С именованными аргументами
result = f.format('{greeting}, {name}!', greeting='Здравствуй', name='Анна')
# 'Здравствуй, Анна!'

# parse() - разбор строки формата
for parsed in f.parse('Привет, {name}! Тебе {age} лет.'):
    print(parsed)
# ('Привет, ', 'name', '', None)
# ('! Тебе ', 'age', '', None)
# (' лет.', None, None, None)

# vformat() - форматирование с args и kwargs
result = f.vformat('{0} + {1} = {sum}', ('2', '3'), {'sum': 5})
# '2 + 3 = 5'
```

## Практические примеры

### Валидация ввода

```python
import string

def is_valid_password(password):
    """Проверка пароля: мин 8 символов, буквы и цифры"""
    if len(password) < 8:
        return False
    
    has_letter = any(c in string.ascii_letters for c in password)
    has_digit = any(c in string.digits for c in password)
    
    return has_letter and has_digit

# Тесты
print(is_valid_password("abc123"))      # False (короткий)
print(is_valid_password("abcdefgh"))    # False (нет цифр)
print(is_valid_password("abc12345"))    # True
```

### Генератор случайных строк

```python
import string
import random

def generate_random_string(length=10, include_digits=True, include_special=False):
    """Генерация случайной строки"""
    chars = string.ascii_letters
    
    if include_digits:
        chars += string.digits
    
    if include_special:
        chars += string.punctuation
    
    return ''.join(random.choice(chars) for _ in range(length))

# Примеры
print(generate_random_string(8))                        # 'aBcD1234'
print(generate_random_string(12, include_special=True)) # 'aB3$dE!f9#gH'
```

### Очистка текста

```python
import string

def clean_text(text, remove_punctuation=True, remove_digits=False):
    """Очистка текста от ненужных символов"""
    result = text
    
    if remove_punctuation:
        translator = str.maketrans('', '', string.punctuation)
        result = result.translate(translator)
    
    if remove_digits:
        translator = str.maketrans('', '', string.digits)
        result = result.translate(translator)
    
    return result.strip()

# Примеры
text = "Привет, мир! Это тест #123."
print(clean_text(text))                           # "Привет мир Это тест 123"
print(clean_text(text, remove_digits=True))       # "Привет мир Это тест "
```

### Создание slug для URL

```python
import string

def create_slug(text):
    """Создание URL-friendly строки"""
    # Оставляем только буквы, цифры и пробелы
    allowed = string.ascii_letters + string.digits + ' '
    cleaned = ''.join(c if c in allowed else ' ' for c in text.lower())
    
    # Заменяем пробелы на дефисы
    slug = '-'.join(cleaned.split())
    
    return slug

# Примеры
print(create_slug("Привет, мир!"))           # "privyet-mir"
print(create_slug("Python 3.11 Released"))   # "python-311-released"
```

## Сравнение с f-строками и format()

```python
from string import Template

name = "Мария"
age = 30

# Template
t = Template("$name, $age лет")
result1 = t.substitute(name=name, age=age)

# f-строки (современный способ, рекомендуется)
result2 = f"{name}, {age} лет"

# str.format()
result3 = "{}, {} лет".format(name, age)

# Все дают одинаковый результат: "Мария, 30 лет"

# Template полезен для пользовательских шаблонов
# и когда строка формата приходит извне
```

## Когда использовать модуль string

✅ **Используйте константы** (`ascii_letters`, `digits` и т.д.) для:

- Валидации ввода
- Генерации случайных строк
- Фильтрации символов

✅ **Используйте Template** когда:

- Шаблон задается пользователем
- Нужна простая и безопасная подстановка
- Работаете с конфигурационными файлами

❌ **Не используйте** для обычного форматирования:

- Вместо Template используйте f-строки (более читаемо)
- Вместо Formatter используйте встроенный format()
