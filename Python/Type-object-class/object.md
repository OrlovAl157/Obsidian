`object` — это действительно **ещё один фундаментальный участник** этой иерархии.  
Он стоит **рядом с `type` на вершине**, и вот как устроена их взаимосвязь 👇

---

## 🧭 Большая картина (иерархия основ всего Python)

        ┌──────────────┐
        │   type       │  ← метакласс (создаёт классы)
        └─────▲────────┘
              │
              │ создает
              ▼
        ┌──────────────┐
        │   object     │  ← базовый класс для всех объектов
        └─────▲────────┘
              │
              │ наследуются от него
              ▼
   ┌────────────┬──────────────┬──────────────┐
   │   int                         │    str                            │   MyClass                     │  ← классы
   └────▲───────┴──────▲───────┴──────▲───────┘
        │                                      │                                     │
        │ создают                       │ создают                       │ создают
        ▼                                     ▼                                     ▼
        5                                   "hello"                             my_obj       ← экземпляры

## 🔍 А теперь подробно:

### 1️⃣ `object` — это **базовый класс для всех объектов**

- Все классы Python **наследуются от `object`**, даже если ты это явно не пишешь.
    

`class MyClass:     pass  print(issubclass(MyClass, object))  # True`

- Он задаёт минимальный набор поведения: `__init__`, `__repr__`, `__eq__`, и т.д.
    

---

### 2️⃣ `type` — это **метакласс**, который создаёт классы (в том числе `object`)

`print(type(object))  # <class 'type'>`

→ `object` был создан `type`.

---

### 3️⃣ И наоборот — `type` сам наследуется от `object`

`print(isinstance(type, object))  # True`

---

### 💫 Получается “замкнутый круг”:

- `type(object) → type`
    
- `type(type) → type`
    
- `isinstance(type, object)` → `True`
    
- `isinstance(object, type)` → `True`
    

🔁  
То есть:

> `object` — прародитель **всех экземпляров**,  
> `type` — прародитель **всех классов**,  
> а **они оба ссылаются друг на друга** и замыкают систему.
