## ⚡ БЫСТРАЯ ШПАРГАЛКА

```python
# Базовое округление
round(3.14159)          # 3 (до целого)
round(3.14159, 2)       # 3.14 (два знака после запятой)
round(1234.5678, -1)    # 1230.0 (до десятков)

# Всегда вверх/вниз
import math
math.ceil(3.1)          # 4 (всегда вверх)
math.floor(3.9)         # 3 (всегда вниз)
math.trunc(3.9)         # 3 (отбросить дробную часть)

# ⚠️ ВАЖНО: round() использует банковское округление
round(2.5)              # 2 (к ближайшему четному!)
round(3.5)              # 4 (к ближайшему четному!)
```

### Когда что использовать

```python
round()         # Обычные задачи
math.ceil()     # Всегда округлить ВВЕРХ
math.floor()    # Всегда округлить ВНИЗ
f"{x:.2f}"      # Нужна СТРОКА с округлением
Decimal         # Финансовые расчеты, точность критична
```

## 1. Функция round()

```python
round(3.14159)       # 3
round(3.14159, 2)    # 3.14
round(1234.5678, -1) # 1230.0
```

⚠️ Python использует **банковское округление** (round half to even):
```python
round(2.5)  # 2
round(3.5)  # 4
```

## 2. Модуль math

```python
import math
math.ceil(3.1)   # 4 — вверх
math.floor(3.9)  # 3 — вниз
math.trunc(3.9)  # 3 — отбросить дробь
```

## 3. Форматирование

```python
f"{3.14159:.2f}"    # '3.14'
f"{3.14159:.0f}"    # '3'
"{:.2f}".format(x)  # то же через format()
```

## 4. Decimal для точных расчётов

```python
from decimal import Decimal, ROUND_HALF_UP
Decimal('2.5').quantize(Decimal('1'), rounding=ROUND_HALF_UP)  # 3
```
