**Пошагово про хэш-таблицы в Python:**

## 1. **Что такое хэш:**
```python
hash("hello")  # возвращает число (например, 123456789)
```

## 2. **Процесс добавления в словарь:**
```python
d = {}
d["name"] = "Alice"

# Шаги:
# 1. Вычисляем хэш от "name" → допустим, 12345
# 2. Берем остаток от деления на размер таблицы: 12345 % 8 = 1
# 3. Помещаем запись в ячейку 1: (key, value, hash)
```

## 3. **Структура ячейки:**
```
[hash, key, value]  # триплет в памяти
```

## 4. **Разрешение коллизий:**
- **Открытая адресация** - ищем следующую свободную ячейку
- **Пример:** ячейка 1 занята → пробуем 2, 3, 4...

## 5. **Поиск:**
```python
d["name"]
# 1. hash("name") → 12345
# 2. 12345 % 8 = 1
# 3. Проверяем ячейку 1: ключ совпал? → возвращаем значение
# 4. Если нет → проверяем следующую ячейку
```

## 6. **Расширение таблицы:**
- Когда заполнено ~2/3 → создаем таблицу в 2 раза больше
- **Рехеширование** всех элементов в новую таблицу

## 7. **Особенности:**
- **Размер таблицы** всегда степень двойки (8, 16, 32...)
- **Load factor** ~0.66 (оптимальный баланс)
- **Псевдослучайность** для защиты от DoS-атак

Вот так работает магия O(1)! ⚡
