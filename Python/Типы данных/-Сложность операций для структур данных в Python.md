## 🧠 Таблица сложности операций

| Операция / Структура     | `list`        | `tuple`      | `set`           | `dict`          | `deque`      |
| ------------------------ | ------------- | ------------ | --------------- | --------------- | ------------ |
| Доступ по индексу `[i]`  | 🟢 O(1)       | 🟢 O(1)      | 🔴 N/A          | 🔴 N/A          | 🟡 O(n)      |
| Поиск `x in s`           | 🔴 O(n)       | 🔴 O(n)      | 🟢 O(1)         | 🟢 O(1)         | 🔴 O(n)      |
| Добавление в конец       | 🟢 O(1)*      | —            | 🟢 O(1)         | 🟢 O(1)         | 🟢 O(1)      |
| Добавление в начало      | 🔴 O(n)       | —            | —               | —               | 🟢 O(1)      |
| Удаление по значению     | 🔴 O(n)       | —            | 🟢 O(1)         | 🟢 O(1)         | 🔴 O(n)      |
| `len()`                  | 🟢 O(1)       | 🟢 O(1)      | 🟢 O(1)         | 🟢 O(1)         | 🟢 O(1)      |
| Сортировка               | 🟡 O(n log n) | —            | —               | —               | —            |

*Амортизированное O(1)

## 💬 Когда использовать

| Цель | Структура |
|------|-----------|
| Быстрый доступ по индексу | `list`, `tuple` |
| Проверка наличия элемента | `set`, `dict` |
| Уникальные значения | `set` |
| Ключ–значение | `dict` |
| Добавление/удаление с обоих концов | `deque` |
| Неизменяемая последовательность | `tuple` |
| Подсчёт элементов | `Counter` |
